# RT_PROEZD: Структурированные билеты для подготовки к экзамену в Obsidian.

## 🚀 Установка

Для корректного отображения и работы со всеми функциями выполните следующие шаги:

1. **Клонирование репозитория:**
   ```bash
   git clone https://github.com/evanvd/RT_PROEZD.git
   ```
Или загрузите zip архив на этой страничке
![[Pasted image 20260524201337.png]]

2. **Открытие в Obsidian:**
   - Откройте Obsidian.
   - Выберите "Открыть папку как хранилище".

![[cache/Pasted image 20260524201536.png]]
   - Укажите путь к склонированной папке `RT_PROEZD` или распакованному архиву.

Если у вас открыта другая папка как хранилище, выбырете пункт внизу и в выпадающем списке выберете `Управление хранилищами...` и проделайте те же операции, что и в пункте 2

## 🏗 Устройство и возможности проекта

### Навигация и перекрестные ссылки
- **Главный хаб:** Файл `RT_PROEZD.md` содержит полный список билетов.
- **Интерактивность:** В доказательствах используются прямые ссылки на определения и теоремы внутри билетов. Это позволяет мгновенно перемещаться к нужной формулировке.
- **Якоря:** Каждое важное утверждение имеет уникальный ID (например, `^th17d1`), на который можно ссылаться из любой части базы знаний.

###  Отслеживание прогресса (Tasks)
- **Статусы:** В файле `RT_PROEZD.md` каждый билет оформлен как задача.
- **Fullscreen Mode:** Используйте горячую клавишу **Alt + F**, чтобы развернуть текущий билет на весь экран (плагин `Obsidian Fullscreen`).

### Экспорт и печать
- Используйте плагин **Better Export PDF** для превращения билетов в печатные материалы. Он сохраняет форматирование callout-блоков и корректно обрабатывает формулы LaTeX.

## 📂 Структура проекта

- `Программа/` — Основная рабочая папка. Содержит готовые билеты, оформленные по стандарту.
- `Петрович md/` — Исходные тексты глав учебника в формате Markdown в том числе первая глава из 3 части. (Удобно использовать для notebooklm)
- `Материалы/` — Центральный узел ссылок на все ресурсы.
- `RT_PROEZD.md` — Главный навигационный файл со списком всех тем и статусами их готовности.
- `.obsidian/` — Конфигурация рабочего пространства (темы, плагины, сниппеты).
