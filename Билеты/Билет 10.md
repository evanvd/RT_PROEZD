# Билет 10. Критерий измеримости по Жордану

### Определение измеримости по Жордану

> [!info] Определение 11.4 (Параллелепипед).
> Параллелепипедом в $\mathbb{R}^n$ называется множество $\Pi = \{x = (x_1, \dots, x_n) : a_i < x_i < b_i, i = 1, 2, \dots, n \}$, где $a_i, b_i$ — такие числа, что $a_i < b_i, i = 1, 2, \dots, n$.
> ^def11d4

> [!info] Определение 11.5 (Клетка и ее мера).
> Клеткой ранга $k$ ($k = 0, 1, 2, \dots$) в $\mathbb{R}^n$ называется параллелепипед
> $$Q^{(k)} = \left\{ x = (x_1, \dots, x_n) : \frac{p_i}{2^k} < x_i < \frac{p_i + 1}{2^k}, p_i \in \mathbb{Z}, i = 1, \dots, n \right\}.$$
> Мерой клетки $Q^{(k)}$ называется число $m Q^{(k)} = \frac{1}{2^{nk}}$.
> ^def11d5

> [!info] Определение 11.7 (Клеточное множество и его мера).
> Множество $S \subset \mathbb{R}^n$ называется **клеточным множеством** ранга $k$ ($k = 0, 1, 2, \dots$), если
> $$S = \left( \bigcup_{j=1}^N Q_j^{(k)} \right) \cup \Gamma_k, \quad N = 0, 1, 2, \dots,$$
> где $Q_j^{(k)}, j = 1, \dots, N$ — различные клетки ранга $k$; $\Gamma_k$ — ограниченное $\Gamma$-множество ранга $k$. Мерой клеточного множества $S$ называется число $m S = \frac{N}{2^{nk}}$.
> ^def11d7

> [!info] Определение 11.8 (Мера Жордана).
> Внешней мерой ограниченного множества $X \subset \mathbb{R}^n$ называется число $\mu^* X = \inf_{S \supset X} m S$, где $S$ — всевозможные клеточные множества, включающие в себя множество $X$. Внутренней мерой ограниченного множества $X \subset \mathbb{R}^n$ называется число $\mu_* X = \sup_{S \subset X} m S$, где $S$ — всевозможные клеточные множества, содержащиеся в $X$. Ограниченное множество $X \subset \mathbb{R}^n$ называется измеримым (по Жордану), если $\mu^* X = \mu_* X$; общее значение внешней и внутренней мер измеримого множества называется мерой множества $X$ и обозначается $\mu X$.
> ^def11d8

> [!note] Лемма 11.7 (Неравенства для внутренней и внешней меры).
> Для ограниченного множества $X \subset \mathbb{R}^n$ выполняются неравенства
> $$0 \le \mu_* X \le \mu^* X < +\infty. \tag{11.1}$$
> ^l11d7

**Доказательство:**
$\square$ Левое из трёх неравенств (11.1) очевидно, так как для всех клеточных множеств $S \subset X$ их мера $m S \ge 0$. Правое из трёх неравенств (11.1) означает, что $\mu^* X$ конечна; это следует из того, что $\mu^* X \le m S$, где $S$ — некоторая клетка такая, что $S \supset X$. Наконец, если $S_1$ и $S_2$ — такие клеточные множества, что $S_1 \subset X \subset S_2$, то мера $m S_1 \le m S_2$. Переходя к точной верхней грани по $S_1 \subset X$, получим $\mu_* X \le m S_2$ для любого клеточного множества $S_2 \supset X$. Переходя к точной нижней грани по $S_2 \supset X$, имеем неравенство $\mu_* X \le \mu^* X$. $\blacksquare$

> [!note] Лемма 11.8 (Измеримость клеточного множества).
> Клеточное множество $S$ измеримо в смысле [[#^def11d8|Определения 11.8 (Мера Жордана)]], и $\mu S = m S$.
> ^l11d8

**Доказательство:**
$\square$ Так как $S \subset S$, то $m S \le \mu_* S$; $\mu^* S \le m S$. Но $\mu_* S \le \mu^* S$, и из полученной цепочки неравенств следует, что $\mu_* S = \mu^* S = m S$, значит, $S$ измеримо и $\mu S = m S$. $\blacksquare$

> [!note] Лемма 11.12 (Связь меры с внутренними точками).
> $\mu_* X = 0 \iff$ множество $X$ не имеет внутренних точек.
> ^l11d12

**Доказательство:**
$\square$ ($\implies$) Пусть $X$ имеет внутреннюю точку $x^0 = (\alpha_1^0, \dots, \alpha_n^0)$. Тогда $\exists \delta > 0 : U_\delta(x^0) \subset X$. Рассмотрим $n$-мерный куб:
$$\Pi = \{x = (x_1, \dots, x_n) : \alpha_i^0 < x_i < \alpha_i^0 + \frac{\delta}{\sqrt{n}}, i = 1, 2, \dots, n \}.$$
![[Pasted image 20260519221240.png]]
Для каждой точки $x$ этого куба
$$\rho(x, x^0) = \sqrt{\sum_{i=1}^n (x_i - \alpha_i^0)^2} < \sqrt{\sum_{i=1}^n \left( \frac{\delta}{\sqrt{n}} \right)^2} = \delta,$$
поэтому $\Pi \subset U_\delta(x^0)$.
Куб $\Pi$, вообще говоря, не является клеткой. Покажем, что найдётся клетка $Q \subset \Pi$.
Рассмотрим два произвольных действительных числа $\alpha$ и $\beta$ такие, что $\alpha < \beta$. Пусть $k$ — такое натуральное число, что $2^k(\beta - \alpha) \ge 2$. Тогда длина интервала $(2^k\alpha; 2^k\beta)$ не меньше, чем 2, и на этом интервале найдутся два последовательных целых числа $p$ и $p + 1$:
$$2^k\alpha < p < p + 1 < 2^k\beta \implies \alpha < \frac{p}{2^k} < \frac{p + 1}{2^k} < \beta.$$
Вставим такую пару чисел $\frac{p_i}{2^k} < \frac{p_i + 1}{2^k}$ на каждом из интервалов $\left( \alpha_i^0; \alpha_i^0 + \frac{\delta}{\sqrt{n}} \right), i = 1, \dots, n$. Тогда клетка
$$Q = \{x = (x_1, \dots, x_n) : \frac{p_i}{2^k} < x_i < \frac{p_i + 1}{2^k}, i = 1, \dots, n \}$$
целиком принадлежит кубу $\Pi$. Поэтому $Q \subset \Pi \subset U_\delta(x^0) \subset X$, и $\mu_* X \ge m Q > 0$.
($\impliedby$) Если $\mu_* X > 0$, то найдётся клеточное множество $S \subset X$ такое, что $m S > 0$. Значит, $S$ содержит хотя бы одну клетку, а так как клетка — открытое множество, то она имеет внутреннюю точку, следовательно, $X$ имеет внутреннюю точку. $\blacksquare$

### Критерий измеримости

> [!error] Теорема 11.6 (Критерий измеримости через аппроксимацию).
> Множество $X \subset \mathbb{R}^n$ измеримо $\iff$ для любого числа $\varepsilon > 0$ найдутся измеримые множества $X_1$ и $X_2$ такие, что $X_1 \subset X \subset X_2$ и $\mu X_2 - \mu X_1 < \varepsilon$.
> ^th11d6

**Доказательство:**
$\square$ ($\implies$) Так как множество $X$ измеримо, то по [[#^def11d8|Определению 11.8 (Мера Жордана)]] для любого $\varepsilon > 0$ найдутся клеточные множества $S_1$ и $S_2$ такие, что $S_1 \subset X \subset S_2$, $m S_2 < \mu X + \frac{\varepsilon}{2}$, $m S_1 > \mu X - \frac{\varepsilon}{2}$. Тогда $m S_2 - m S_1 < \varepsilon$, и множества $S_1$ и $S_2$ можно взять в качестве измеримых множеств $X_1$ и $X_2$ (клеточные множества измеримы по [[#^l11d8|Лемме 11.8]]).
($\impliedby$) Пусть $X_1$ и $X_2$ — измеримые множества такие, что $X_1 \subset X \subset X_2$, $\mu X_2 - \mu X_1 < \varepsilon$. Возьмём клеточные множества $S_1$ и $S_2$ такие, что $S_1 \subset X_1 \subset X \subset X_2 \subset S_2$, $m S_1 > \mu X_1 - \varepsilon$, $m S_2 < \mu X_2 + \varepsilon$. Тогда $m S_2 - m S_1 < \mu X_2 - \mu X_1 + 2\varepsilon < 3\varepsilon$. Так как $m S_1 \le \mu_* X \le \mu^* X \le m S_2$, то $0 \le \mu^* X - \mu_* X < 3\varepsilon$. В силу произвольности $\varepsilon$ имеем $\mu^* X = \mu_* X. \blacksquare$

> [!error] Теорема 11.7 (Критерий измеримости по границе).
> Множество $X \subset \mathbb{R}^n$ измеримо $\iff X$ — ограничено и $\mu \partial X = 0$.
> ^th11d7

**Доказательство:**
$\square$ ($\implies$) Ограниченность следует из определения. Для любого $\varepsilon > 0$ найдутся клеточные множества $S_1$ и $S_2$ такие, что $S_1 \subset \text{int } X \subset X \subset \bar{X} \subset S_2$ и $m S_2 - m S_1 < \varepsilon$. Тогда $\partial X = (\bar{X} \setminus \text{int } X) \subset (S_2 \setminus S_1)$. Поэтому $\mu^* \partial X \le m(S_2 \setminus S_1) < \varepsilon$. Значит, $\mu^* \partial X = 0$, и $\partial X$ измерима с мерой 0 по [[#^l11d9|Лемме 11.9]].
($\impliedby$) Если $\mu \partial X = 0$, то для любого $\varepsilon > 0$ найдётся замкнутое клеточное множество $S \supset \partial X$ с мерой $m S < \varepsilon$. Множество $\hat{S}$ состоит из клеток, попавших либо в $\text{int } X$, либо в $\text{int } (\mathbb{R}^n \setminus X)$. Пусть $S_1 = \hat{S} \cap \text{int } X$ — клеточное множество. Тогда $S_1 \subset X \subset S_2 = S_1 \cup S$, причём $\mu S_2 - \mu S_1 = m S < \varepsilon$. По [[#^th11d6|Теореме 11.6 (Критерий измеримости через аппроксимацию)]] множество $X$ измеримо. $\blacksquare$

> [!note] Лемма 11.9 (Измеримость множества нулевой внешней меры).
> Если $\mu^* X = 0$, то множество $X$ измеримо и $\mu X = 0$.
> ^l11d9
