# Билет 13. Критерий интегрируемости Римана

### Определенный интеграл Римана

> [!info] Определение 12.1 (Определенный интеграл Римана).
> Верхним и нижним интегралами Дарбу функции $f$, ограниченной на отрезке $[a; b]$, называются соответственно числа $I^* = \inf_R S^*_R$ и $I_* = \sup_R S_{*R}$ (точные нижняя и верхняя грани берутся по всем разбиениям отрезка $[a; b]$). Если $I^* = I_*$, то ограниченная функция $f$ называется интегрируемой по Риману на отрезке $[a; b]$, а общее значение $I^* = I_*$ называется определённым интегралом Римана функции $f$ по отрезку $[a; b]$ (обозначается $\int_a^b f(x) \, dx$).
> ^def12d1

> [!note] Лемма 12.3 (Неравенства для интегралов Дарбу).
> Для ограниченной функции $f$ на отрезке $[a; b]$ выполняются неравенства
> $$-\infty < I_* \le I^* < +\infty. \tag{12.1}$$
> ^l12d3

**Доказательство:**
$\square$ Из [[Билет 12#^l12d2|Леммы 12.2 (О сравнении верхней и нижней сумм Дарбу)]] следует, что для любых разбиений $R_1$ и $R_2$ отрезка $[a; b]$ выполняется неравенство $S_{*R_2} \le S^*_{R_1}$. Переходя к точной верхней грани по $R_2$, мы видим, что $I_* \le S^*_{R_1}$ для любого разбиения $R_1$, а переходя затем к точной нижней грани по $R_1$, получим неравенство $I_* \le I^*$. Правое из неравенств (12.1) означает конечность $I^*$, которая следует из того, что $I^* \le S^*_{R_1}$ для некоторого разбиения $R_1$. Аналогично получается левое из неравенств (12.1). $\blacksquare$

### Критерий интегрируемости Римана

> [!error] Теорема 12.9 (Критерий интегрируемости Римана).
> Функция $f$ интегрируема на отрезке $[a; b]$ и $\int_a^b f(x) \, dx = I \iff$ для любой последовательности разбиений $R_n$ отрезка $[a; b]$ такой, что $\lim_{n \to \infty} |R_n| = 0$, при любом выборе точек $\xi_i^{(n)} \in [x_{i-1}^{(n)}; x_i^{(n)}]$ выполняется равенство $\lim_{n \to \infty} \sigma_{R_n} = I$.
> ^th12d9

**Доказательство:**

$\square$ ($\implies$) Рассмотрим соответствующие последовательности сумм Дарбу $S^*_{R_n}$ и $S_{*R_n}$. Так как $m_i \le f(\xi_i) \le M_i$ при $i = 1, \dots, N$, то
$$S_{*R_n} \le \sigma_{R_n} \le S^*_{R_n}. \tag{12.2}$$

![[Материалы/Петрович md/cache/Pasted image 20260519224420.png]]
Так как $f$ интегрируема на $[a; b]$, то по [[Билет 12#^th12d1|пункту 3° критерия Дарбу (Теорема 12.1)]]
$$\forall \varepsilon > 0 \to \exists \delta > 0 : \forall R, |R| < \delta \to S^*_R - S_{*R} < \varepsilon. \tag{12.3}$$
Пусть $\lim_{n \to \infty} |R_n| = 0$. Тогда для указанного в (12.3) числа $\delta(\varepsilon)$
$$\exists n_0 : \forall n \ge n_0 \to |R_n| < \delta.$$
Итак, $\forall \varepsilon > 0 \to \exists n_0 \forall n \ge n_0 \to S^*_{R_n} - S_{*R_n} < \varepsilon$. Так как $S_{*R_n} \le I \le S^*_{R_n}$, то, учитывая (12.2), имеем
$$\forall \varepsilon > 0 \to \exists n_0 : \forall n \ge n_0 \to |\sigma_{R_n} - I| \le S^*_{R_n} - S_{*R_n} < \varepsilon,$$
а это значит, что $\lim_{n \to \infty} \sigma_{R_n} = I$ при любом выборе точек $\xi_i^{(n)}$.
($\impliedby$) Докажем сначала, что функция $f$ ограничена на $[a; b]$. Пусть это не так. Рассмотрим некоторую последовательность разбиений $R_n$ такую, что $\lim_{n \to \infty} |R_n| = 0$. При фиксированном $n$ функция $f$ неограничена хотя бы на одном из отрезков разбиения $[x_{k-1}^{(n)}; x_k^{(n)}]$. Пусть $\sigma'_{R_n}$ — часть римановой суммы без $k$-го слагаемого с фиксированными точками $\xi_i^{(n)}$ на остальных отрезках ($i \ne k$). Тогда, в силу неограниченности $f$ на $k$-м отрезке, $\exists \xi_k^{(n)} \in [x_{k-1}^{(n)}; x_k^{(n)}]$ такая, что $|f(\xi_k^{(n)})| > \frac{n + |\sigma'_{R_n}|}{\Delta x_k^{(n)}}$. Значит, $|\sigma_{R_n}| = |\sigma'_{R_n} + f(\xi_k^{(n)}) \cdot \Delta x_k^{(n)}| \ge |f(\xi_k^{(n)})| \cdot \Delta x_k^{(n)} - |\sigma'_{R_n}| > n + |\sigma'_{R_n}| - |\sigma'_{R_n}| = n$.

Это противоречит существованию конечного $\lim_{n \to \infty} \sigma_{R_n}$. Итак, $f$ ограничена на $[a; b]$. Докажем теперь, что она интегрируема. Если это не так, то, в силу [[Билет 12#^th12d1|пункту 3° критерия Дарбу (Теорема 12.1)]],
$$\exists \varepsilon > 0 : \forall \delta > 0 \to \exists R, |R| < \delta : S^*_R - S_{*R} \ge \varepsilon.$$
Пусть $\delta = \frac{1}{n}, n = 1, 2, \dots$ Тогда
$$\exists \varepsilon > 0 : \forall n \to \exists R_n, |R_n| < \frac{1}{n} : S^*_{R_n} - S_{*R_n} \ge \varepsilon.$$
Так как $M_i^{(n)} = \sup_{[x_{i-1}^{(n)};x_i^{(n)}]} f(x)$, то при фиксированном разбиении $R_n$ можно подобрать точки $\xi_i^{(n)}, i = 1, 2, \dots, N_n$, так, что $|f(\xi_i^{(n)}) - M_i^{(n)}| < \frac{\varepsilon}{4(b - a)}$. Тогда
$$|S^*_{R_n} - \sigma_{R_n}| = \left| \sum_{i=1}^{N_n} (M_i^{(n)} - f(\xi_i^{(n)})) \Delta x_i^{(n)} \right| < \frac{\varepsilon}{4(b - a)} \cdot \sum_{i=1}^{N_n} \Delta x_i^{(n)} = \frac{\varepsilon}{4}.$$
Аналогично можно подобрать точки $\eta_i^{(n)}, i = 1, 2, \dots, N_n$, так, что
$$|f(\eta_i^{(n)}) - m_i^{(n)}| < \frac{\varepsilon}{4(b - a)}, \quad \text{и } |\sigma'_{R_n} - S_{*R_n}| < \frac{\varepsilon}{4}$$
(суммы Римана $\sigma_{R_n}$ и $\sigma'_{R_n}$ рассматриваются для одного разбиения $R_n$, но выбираются разные промежуточные точки). Для построенных сумм Римана $|\sigma_{R_n} - \sigma'_{R_n}| \ge \frac{\varepsilon}{2}, n = 1, 2, \dots$ (иначе $|S^*_{R_n} - S_{*R_n}| \le |S^*_{R_n} - \sigma_{R_n}| + |\sigma_{R_n} - \sigma'_{R_n}| + |\sigma'_{R_n} - S_{*R_n}| < \frac{\varepsilon}{4} + \frac{\varepsilon}{2} + \frac{\varepsilon}{4} = \varepsilon$, а это не так). Но $|R_n| < \frac{1}{n}$, поэтому $\lim_{n \to \infty} |R_n| = 0$, и $\lim_{n \to \infty} \sigma_{R_n} = \lim_{n \to \infty} \sigma'_{R_n} = I$. Значит, $\lim_{n \to \infty}(\sigma_{R_n} - \sigma'_{R_n}) = 0$, что противоречит неравенству $|\sigma_{R_n} - \sigma'_{R_n}| \ge \frac{\varepsilon}{2}, n = 1, 2, \dots$ Полученное противоречие показывает, что $f$ интегрируема на $[a; b]. \blacksquare$
