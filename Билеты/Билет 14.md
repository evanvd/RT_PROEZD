# Билет 14. Интегрируемость непрерывной функции и монотонной функции

### Интегрируемость непрерывной функции.
>[!error] Теорема 12.2 (Интегрируемость непрерывной функции).
> Если функция $f$ непрерывна на отрезке $[a; b]$, то она интегрируема на $[a; b]$.
> ^th14d1

**Доказательство:**
$\square$ По теореме Кантора $f$ равномерно непрерывна на $[a; b]$, значит, $\forall \varepsilon > 0 \to \exists \delta > 0 : \forall x', x'' \in [a; b], |x' - x''| < \delta \to |f(x') - f(x'')| < \frac{\varepsilon}{b - a}$. Рассмотрим разбиение $R$ отрезка $[a; b]$ такое, что $|R| < \delta$. Тогда на каждом отрезке $[x_{i-1}; x_i]$ колебание $\omega_i = M_i - m_i = f(x') - f(x'') < \frac{\varepsilon}{b - a}$, и $\omega_R = \sum_{i=1}^N \omega_i \Delta x_i < \frac{\varepsilon}{b - a} \cdot (b - a) = \varepsilon$. По критерию Дарбу (пункт 3°) функция $f$ интегрируема. $\blacksquare$

### Интегрируемость монотонной функции.
>[!error] Теорема 12.3 (Интегрируемость монотонной функции).
> Если функция $f$ монотонна на отрезке $[a; b]$, то она интегрируема на $[a; b]$.
> ^th14d2

**Доказательство:**
$\square$ Пусть $f$ возрастает на $[a; b]$. Тогда $f(a) \le f(x) \le f(b)$, и функция $f$ ограничена. Для $\varepsilon > 0$ рассмотрим $\delta = \frac{\varepsilon}{f(b) - f(a) + 1}$. Если $R$ — разбиение с $|R| < \delta$, то $\omega_i = f(x_i) - f(x_{i-1})$ и
$\omega_R = \sum (f(x_i) - f(x_{i-1})) \Delta x_i < \delta \sum (f(x_i) - f(x_{i-1})) = \delta(f(b) - f(a)) < \varepsilon$.
По критерию Дарбу (пункт 3°) функция $f$ интегрируема. $\blacksquare$
