# Билет 15. Интегрируемость ограниченной функции с конечным числом точек разрыва

### Интегрируемость функции с конечным числом точек разрыва.
>[!error] Теорема 12.7 (Интегрируемость функции с конечным числом точек разрыва).
> Если функция $f$ ограничена на отрезке $[a; b]$ и имеет конечное число точек разрыва на этом отрезке, то она интегрируема на $[a; b]$.
> ^th15d1

**Доказательство:**
$\square$ Достаточно доказать утверждение для функции, имеющей единственную точку разрыва $\xi$ в одном из концов отрезка, например, на $[\xi; b]$. Для любого $\xi' \in (\xi; b)$ функция $f$ непрерывна (следовательно, интегрируема) на $[\xi'; b]$. 
Пусть $|f(x)| \le M$. Для любого $\varepsilon > 0$ выберем $\xi' = \xi + \frac{\varepsilon}{4M}$. Так как $f$ интегрируема на $[\xi'; b]$, найдётся разбиение $R'$ этого отрезка с $\omega_{R'} < \varepsilon/2$. Добавим к нему отрезок $[\xi; \xi']$. Тогда для полученного разбиения $R$ отрезка $[\xi; b]$ имеем $\omega_R = \omega_0(\xi' - \xi) + \omega_{R'} < 2M \cdot \frac{\varepsilon}{4M} + \frac{\varepsilon}{2} = \varepsilon$. Значит, $f$ интегрируема на $[\xi; b]$. 
Отрезок $[a; b]$ с конечным числом точек разрыва можно разбить на конечное число таких отрезков. Из аддитивности интегрируемости следует интегрируемость на всём $[a; b]$. $\blacksquare$
