# 2) Внутренность, замыкание и граница множества, непустота границы нетривиального множества в $\mathbb{R}^{n}$
## Определение внутренности и замыкания.
>[!info] **Определение.** (Внутренность множества)
>Множество всех внутренних точек множества $X \subset \mathbb{R}^n$ называется **внутренностью** множества $X$ и обозначается $\text{int } X$.
>^def02d1

>[!info] **Определение.** (Замыкание множества)
>Для множества $X \subset \mathbb{R}^n$ его **замыканием** $\bar{X}$ называется множество точек прикосновения $X$ (предельных и изолированных точек).
>^def02d2

>[!error] **Теорема.** (Эквивалентное определение замыкания)
>Для любого множества $X \subset \mathbb{R}^n$ его замыкание $\bar{X}$ совпадает с множеством $\widehat{\text{int } \hat{X}}$ (дополнение к внутренности дополнения).
>
>$\square$ $x_0$ — точка прикосновения $X \iff \forall \delta > 0 \implies U_\delta(x_0) \cap X \ne \varnothing \iff \forall \delta > 0 \implies \exists x \in U_\delta(x_0) : x \notin \hat{X} \iff x_0$ не является внутренней точкой $\hat{X} \iff x_0 \notin \text{int } \hat{X} \iff x_0 \in \bar{X}$. $\blacksquare$
>^th02d1

## Свойство замыкания: $\overline{X} = \hat{Int\hat{X}}$.
>[!info] **Определение.** (Формальное определение замыкания)
>Для множества $X \subset \mathbb{R}^n$ его **замыканием** $\bar{X}$ называется множество $\widehat{\text{int } \hat{X}}$.
>^def02d3

>[!info] **Свойства замыкания:**
>1) $\bar{X}$ — замкнутое множество (как дополнение к открытому множеству $\text{int } \hat{X}$).
>2) $\bar{X} \supset X$.
>3) $\bar{X} = X \iff X$ замкнуто.
>^info02d1

## Свойство границы: $\partial X = \overline{X} \setminus IntX$.
>[!info] **Определение.** (Граница множества)
>Для множества $X \subset \mathbb{R}^n$ его **границей** $\partial X$ называется множество $\bar{X} \setminus \text{int } X$.
>^def02d4

>[!error] **Лемма.** (Эквивалентное представление границы)
>Для любого множества $X \subset \mathbb{R}^n$:
>$$\partial X = \widehat{\text{int } X \cup \text{int } \hat{X}}.$$
>
>$\square$ $\partial X = \bar{X} \setminus \text{int } X = \bar{X} \cap \widehat{\text{int } X} = \widehat{\text{int } \hat{X}} \cap \widehat{\text{int } X} = \widehat{\text{int } X \cup \text{int } \hat{X}}$. $\blacksquare$
>^th02d2

## $\mathbb{R}^{n} = IntX \cup \partial X \cup Int\hat{X}$.
>[!error] **Теорема.** (Разбиение пространства)
>Для любого множества $X \subset \mathbb{R}^n$ всё пространство $\mathbb{R}^n$ является объединением попарно непересекающихся множеств:
>$$\mathbb{R}^n = \text{int } X \cup \text{int } \hat{X} \cup \partial X.$$
>
>$\square$ Утверждение следует из определения границы и свойств внутренности и замыкания. Граница $\partial X$ — это множество точек, не являющихся внутренними ни для $X$, ни для $\hat{X}$, т.е. таких точек, в любой окрестности которых есть как точки $X$, так и точки $\hat{X}$. $\blacksquare$
>^th02d3

## Непустота границы нетривиального множества:
## $x_1 \in X, x_2 \in \hat{X} \implies$ на непрерывной кривой, соединяющей $x_1$ и $x_2$, найдётся $x_0 \in \partial X$.
>[!error] **Лемма.** (О существовании граничной точки на кривой)
>Если $X$ — множество в $\mathbb{R}^n$, $x' \in X, x'' \in \hat{X}$, то на любой непрерывной кривой (в частности, на отрезке), соединяющей точки $x'$ и $x''$, найдётся точка $\tilde{x} \in \partial X$.
>
>$\square$ Рассмотрим отрезок $x(t) = x' + t(x'' - x')$, $t \in [0; 1]$. При этом $x(0) = x', x(1) = x''$. Нужно доказать, что найдётся значение $t_0 \in [0; 1]$, для которого $\tilde{x} = x(t_0) \in \partial X$.
>Из двух половинок отрезка $[0; 1]$ выберем отрезок $\Delta_1$, для которого в одном из концов $x(t) \in X$, в другом $x(t) \in \hat{X}$, и т.д. Получим последовательность вложенных отрезков $\Delta_1 \supset \Delta_2 \supset \dots \supset \Delta_k \supset \dots$, в одном из концов каждого из которых $x(t) \in X$, в другом $x(t) \in \hat{X}$. Длина отрезка $\Delta_k$ стремится к нулю. По теореме Кантора о вложенных отрезках существует единственная точка $t_0$, принадлежащая всем $\Delta_k$.
>В любой окрестности точки $\tilde{x} = x(t_0)$ найдутся концы отрезков $\Delta_k$ с достаточно большими номерами. В одном из концов такого отрезка точка $x(t) \in X$, в другом $x(t) \in \hat{X}$. Значит, в любой $U_\delta(\tilde{x})$ есть как точки из $X$, так и точки из $\hat{X}$. Следовательно, $\tilde{x} \in \partial X$. $\blacksquare$
>^th02d4
