# Билет 29. Равномерная сходимость функциональных последовательностей и рядов, критерий Коши, признак Вейерштрасса

### Определение равномерной сходимости.
>[!info] **Определение 16.1.** (Равномерная сходимость функциональной последовательности)
> Функциональная последовательность $f_n$ называется **равномерно сходящейся** к функции $f$ на множестве $E$, если
> $$\forall \varepsilon > 0 \to \exists n_0 : \forall n \ge n_0, \forall x \in E \to |f_n(x) - f(x)| < \varepsilon.$$
> ^def29d1

>[!info] **Определение 16.2.** (Равномерная сходимость функционального ряда)
> Функциональный ряд $\sum_{n=1}^{\infty} u_n(x)$ называется равномерно сходящимся на множестве $E$, если последовательность его частичных сумм $S_n(x) = \sum_{k=1}^n u_k(x)$ равномерно сходится на $E$.
> ^def29d2

### Критерий Коши.
>[!error] **Теорема 16.1.** (Критерий Коши равномерной сходимости)
> Функциональная последовательность $f_n$ равномерно сходитcя на множестве $E \iff \forall \varepsilon > 0 \to \exists n_0 : \forall n, m \ge n_0, \forall x \in E \to |f_n(x) - f_m(x)| < \varepsilon$.
> ^th29d1

**Доказательство:**
$\square$ ($\implies$) Из равномерной сходимости к $f$: $|f_n - f_m| \le |f_n - f| + |f - f_m| < \varepsilon/2 + \varepsilon/2 = \varepsilon$.
($\impliedby$) Из критерия Коши для числовых последовательностей в каждой точке $x \in E$ существует предел $f(x)$. Переходя к пределу при $m \to \infty$ в неравенстве $|f_n(x) - f_m(x)| \le \varepsilon$, получаем $|f_n(x) - f(x)| \le \varepsilon$. $\blacksquare$

### Признак Вейерштрасса.
>[!error] **Теорема 16.6.** (Признак Вейерштрасса)
> Пусть для всех $x \in E$ выполняется $|u_n(x)| \le \alpha_n$, где числовой ряд $\sum \alpha_n$ сходится. Тогда ряд $\sum u_n(x)$ сходится равномерно на $E$.
> ^th29d2

**Доказательство:**
$\square$ По критерию Коши для числового ряда $\sum \alpha_k$: $\sum_{k=n+1}^{n+p} \alpha_k < \varepsilon$. 
Тогда для функционального ряда $\left| \sum_{k=n+1}^{n+p} u_k(x) \right| \le \sum_{k=n+1}^{n+p} |u_k(x)| \le \sum_{k=n+1}^{n+p} \alpha_k < \varepsilon$. 
По критерию Коши функциональный ряд сходится равномерно. $\blacksquare$
