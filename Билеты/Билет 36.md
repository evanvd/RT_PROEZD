# Билет 36. Бесконечная дифференцируемость суммы степенного ряда на интервале сходимости. Единственность разложения функции в степенной ряд

### Бесконечная дифференцируемость суммы степенного ряда

> [!error] Теорема 17.6.
> Пусть 
> $$f(x) = \sum_{n=0}^{\infty} a_n (x - x_0)^n$$
>  — сумма степенного ряда с радиусом сходимости $R \in (0; +\infty]$. Тогда функция $f$ бесконечно дифференцируема (т.е. имеет производные всех порядков $k = 1, 2, \dots$) на интервале $(x_0 - R; x_0 + R)$; при этом при всех $k = 1, 2, \dots$
> $$f^{(k)}(x) = \sum_{n=k}^{\infty} a_n \cdot n(n - 1) \dots (n - k + 1)(x - x_0)^{n-k}, \quad x \in (x_0 - R; x_0 + R),$$
> and каждый такой ряд имеет тот же радиус сходимости $R$.
> ^th17d6

**Доказательство:**
$\square$ Теорему достаточно доказать для $k = 1$, далее применять её последовательно. Формально продифференцированный ряд 
$$\sum_{n=1}^{\infty} n a_n (x - x_0)^{n-1}$$
 имеет тот же радиус сходимости, что и исходный, по **Лемме 17.2 (Сохранение радиуса сходимости)**. Так как при всех $r \in (0; R)$ продифференцированный ряд равномерно сходится на отрезке $[x_0 - r; x_0 + r]$, а сам ряд сходится в точке $x = x_0$, то по теореме о почленном дифференцировании функционального ряда равенство 
$$f'(x) = \sum_{n=1}^{\infty} n a_n (x - x_0)^{n-1}$$
 имеет место на отрезке $[x_0 - r; x_0 + r]$, а в силу произвольности $r \in (0; R)$ — и на всём интервале $(x_0 - R; x_0 + R). \blacksquare$

### Единственность разложения функции в степенной ряд

> [!error] Теорема 17.7. (Единственность разложения)
> Если функция $f$ — аналитическая в точке $x_0$, то она имеет в точке $x_0$ производные всех порядков, причём коэффициенты степенного ряда, являющегося разложением функции $f$, вычисляются по формулам $a_k = \frac{f^{(k)}(x_0)}{k!}$, $k = 1, 2, \dots$, т.е. разложение аналитической функции в степенной ряд в окрестности точки единственно:
> $$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(x_0)}{n!} (x - x_0)^n, \quad x_0 - \delta < x < x_0 + \delta.$$
> ^th17d7

**Доказательство:**
$\square$ Бесконечная дифференцируемость аналитической функции на всём интервале $(x_0 - \delta; x_0 + \delta)$ следует из **Теоремы 17.6**; при этом $f(x_0) = a_0$. Так как при $k = 1, 2, \dots$
$$f^{(k)}(x) = \sum_{n=k}^{\infty} a_n \cdot n(n - 1) \dots (n - k + 1)(x - x_0)^{n-k},$$
то $f^{(k)}(x_0) = a_k \cdot k!$ (все слагаемые при $n > k$ обращаются в нуль), и $a_k = \frac{f^{(k)}(x_0)}{k!}, k = 1, 2, \dots. \blacksquare$

### Пример бесконечно дифференцируемой функции, не разлагаемой в ряд

> [!example] **Пример 17.11.**
> Рассмотрим функцию
> $$f(x) = \begin{cases} e^{-1/x^2}, & \text{если } x \ne 0; \\ 0, & \text{если } x = 0. \end{cases}$$

Докажем, что для этой функции $f^{(k)}(0) = 0$ при всех $k = 0, 1, 2, \dots$. Отсюда будет следовать, что ряд Тейлора этой функции в точке $x = 0$ — нулевой. Сумма этого ряда равна 0 при всех $x$, и ряд Тейлора функции $f$ не сходится к $f(x)$ ни в одной окрестности точки 0 (т.е. эта функция бесконечно дифференцируема в любой точке, но не является аналитической в точке $x = 0$).

$\square$ Так как $\lim_{x \to 0} f(x) = 0$, то функция $f$ непрерывна в точке $x = 0$. Далее,
$$f'(0) = \lim_{t \to 0} \frac{e^{- \frac{1}{t^2}}}{t} = \lim_{y \to +\infty} \frac{\sqrt{y}}{e^y} = 0$$
(сделана замена $y = \frac{1}{t^2}$). Тогда $f'(x) = \begin{cases} \frac{2}{x^3} \cdot e^{- \frac{1}{x^2}}, & \text{если } x \ne 0; \\ 0, & \text{если } x = 0. \end{cases}$ Покажем по индукции, что при всех $k = 1, 2, \dots$
$$f^{(k)}(x) = \begin{cases} \frac{P_k(x)}{x^{3k}} \cdot e^{- \frac{1}{x^2}}, & \text{если } x \ne 0; \\ 0, & \text{если } x = 0, \end{cases}$$
где $P_k(x)$ — многочлен. При $k = 1$ утверждение уже доказано. Пусть оно верно для некоторого $k \in \mathbb{N}$, докажем его справедливость для $k + 1$. При $x \ne 0$ имеем
$$f^{(k+1)}(x) = (f^{(k)}(x))' = e^{- \frac{1}{x^2}} \cdot \frac{2}{x^3} \cdot \frac{P_k(x)}{x^{3k}} + e^{- \frac{1}{x^2}} \frac{x^{3k}P'_k(x) - 3kx^{3k-1}P_k(x)}{x^{6k}} = e^{- \frac{1}{x^2}} \cdot \frac{P_{k+1}(x)}{x^{3k+3}}.$$
Далее, $f^{(k+1)}(0) = \lim_{t \to 0} \frac{f^{(k)}(t) - f^{(k)}(0)}{t} = \lim_{t \to 0} \frac{P_k(t)}{t^{3k+1}} e^{- \frac{1}{t^2}} = 0. \blacksquare$
