# Билет 39. Разложение в степенной ряд $(1+x)^\alpha$

### Биномиальное разложение.
>[!error] Теорема 17.9.
> Для любого действительного числа $\alpha \ne 0, 1, 2, \dots$ ряд Маклорена функции $(1 + x)^\alpha$ сходится к этой функции в любой точке $x \in (-1; 1)$:
> $$(1 + x)^\alpha = 1 + \sum_{n=1}^{\infty} \frac{\alpha(\alpha - 1) \dots (\alpha - n + 1)}{n!} x^n.$$
> Радиус сходимости такого ряда равен 1.
> ^th39d1

**Доказательство:**
$\square$ 1) По признаку Даламбера: $\left| \frac{u_{n+1}}{u_n} \right| = \frac{|\alpha - n|}{n+1} |x| \to |x|$. Ряд абсолютно сходится при $|x| < 1$, значит $R = 1$.
2) Пусть $f(x)$ — сумма ряда. Рассмотрим $\varphi(x) = f(x)(1+x)^{-\alpha}$. 
Дифференцируя, можно показать, что $\varphi'(x) = 0$ для всех $x \in (-1, 1)$. 
Действительно, $\varphi'(x) = (1+x)^{-\alpha-1} [(1+x)f'(x) - \alpha f(x)]$. 
Подставляя ряд и его производную, выражение в скобках обращается в нуль. 
Так как $\varphi(0) = f(0) = 1$, то $\varphi(x) \equiv 1$, т.е. $f(x) = (1+x)^\alpha$. $\blacksquare$
