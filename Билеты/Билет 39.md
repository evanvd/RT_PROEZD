# Билет 39. Разложение в степенной ряд (1+x)^alpha

### Биномиальное разложение

> [!error] Теорема 17.9.
> Для любого действительного числа $\alpha \ne 0, 1, 2, \dots$ ряд Маклорена функции $(1 + x)^\alpha$ сходится к этой функции в любой точке $x \in (-1; 1)$, т.е. при всех $x \in (-1; 1)$:
> $$(1 + x)^\alpha = \sum_{n=0}^{\infty} C_\alpha^n x^n,$$
> where 
> $$C_\alpha^0 = 1, C_\alpha^n = \frac{\alpha(\alpha - 1) \dots (\alpha - n + 1)}{n!}$$
>  при $n \in \mathbb{N}$; радиус сходимости такого ряда равен 1.
> ^th17d9

**Доказательство:**
$\square$ Найдём радиус сходимости ряда при $\alpha \ne 0, 1, 2, 3, \dots$ Если $x \ne 0$, то
$$\frac{|u_{n+1}(x)|}{|u_n(x)|} = \left| \frac{\alpha(\alpha - 1) \dots (\alpha - n)x^{n+1} \cdot n!}{(n + 1)!\alpha(\alpha - 1) \dots (\alpha - n + 1)x^n} \right| = \frac{|\alpha - n|}{n + 1} |x| \to |x| \text{ при } n \to \infty.$$
По признаку Даламбера ряд абсолютно сходится при 
$$0 < |x| < 1$$
 and абсолютно расходится при $|x| > 1$; так как в точке $x = 0$ ряд абсолютно сходится, то по **Лемме 17.1** радиус сходимости ряда равен 1.
Сумму ряда 
$$\sum_{n=0}^{\infty} C_\alpha^n x^n, |x| < 1,$$
 обозначим $f(x)$. Докажем, что $f(x) = (1 + x)^\alpha$. Рассмотрим функцию 
$$\varphi(x) = f(x)(1 + x)^{-\alpha}, |x| < 1.$$
 Так как внутри интервала сходимости ряд можно почленно дифференцировать, то при всех $x \in (-1; 1)$
$$\varphi'(x) = f'(x)(1 + x)^{-\alpha} - \alpha f(x)(1 + x)^{-\alpha-1} = (1 + x)^{-\alpha-1} \left[ (1 + x)f'(x) - \alpha f(x) \right] =$$
$$= (1 + x)^{-\alpha-1} \left[ (1 + x) \sum_{n=1}^{\infty} C_\alpha^n n x^{n-1} - \alpha \sum_{n=0}^{\infty} C_\alpha^n x^n \right] =$$
$$= (1 + x)^{-\alpha-1} \left( \sum_{n=1}^{\infty} C_\alpha^n n x^{n-1} + \sum_{n=1}^{\infty} C_\alpha^n n x^n - \alpha \sum_{n=0}^{\infty} C_\alpha^n x^n \right) = (1 + x)^{-\alpha-1} \sum_{n=0}^{\infty} ((n + 1)C_\alpha^{n+1} + (n - \alpha)C_\alpha^n) x^n.$$
Но при всех $n = 0, 1, 2, \dots$
$$(n + 1)C_\alpha^{n+1} + (n - \alpha)C_\alpha^n = (n + 1) \cdot \frac{\alpha(\alpha - 1) \dots (\alpha - n + 1)(\alpha - n)}{(n + 1)!} + (\alpha - n) \cdot \frac{\alpha(\alpha - 1) \dots (\alpha - n + 1)}{n!} = 0;$$
поэтому $\varphi'(x) = 0$ при всех $x \in (-1; 1)$, т.е. функция $\varphi(x)$ постоянна. Так как 
$$\varphi(0) = f(0) = 1,$$
 то $\varphi(x) \equiv 1$, т.е. $f(x) = (1 + x)^\alpha$ на $(-1; 1). \blacksquare$
