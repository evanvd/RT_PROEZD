# Билет 40. Разложение в степенной ряд  $e^x$

### Разложение экспоненты, тригонометрических и гиперболических функций

> [!error] Теорема 17.8.
> Ряды Маклорена (т.е. ряды Тейлора в точке $x_0 = 0$) для функций $e^x, \text{ch } x, \text{sh } x, \cos x, \sin x$ сходятся к этим функциям в любой точке $x \in \mathbb{R}$, т.е. при всех $x$
> $$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}; \quad \text{ch } x = \sum_{n=0}^{\infty} \frac{x^{2n}}{(2n)!}; \quad \text{sh } x = \sum_{n=0}^{\infty} \frac{x^{2n+1}}{(2n + 1)!};$$
> $$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!}; \quad \sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n + 1)!};$$
> радиусы сходимости всех этих рядов равны $+\infty$.
> ^th17d8

**Доказательство:**
$\square$ Так как при всех $n = 1, 2, \dots$
$$(e^x)^{(n)} = e^x; \quad (\text{ch } x)^{(n)} = \frac{e^x + (-1)^n e^{-x}}{2}; \quad (\text{sh } x)^{(n)} = \frac{e^x - (-1)^n e^{-x}}{2};$$
$$(\cos x)^{(n)} = \cos \left( x + \frac{\pi}{2} n \right); \quad (\sin x)^{(n)} = \sin \left( x + \frac{\pi}{2} n \right),$$
то при всех $x \in (-\delta; \delta), \delta > 0$, выполняются неравенства
$$|(e^x)^{(n)}| \le e^\delta; \quad |(\text{ch } x)^{(n)}| \le e^\delta; \quad |(\text{sh } x)^{(n)}| \le e^\delta; \quad |(\cos x)^{(n)}| \le 1; \quad |(\sin x)^{(n)}| \le 1.$$
Поэтому по **Лемме 17.5** эти функции являются суммами своих рядов Тейлора на любом интервале $(-\delta; \delta), \delta > 0$, а значит, и при всех $x \in \mathbb{R}$. Так как ряды сходятся в каждой точке, то их радиусы сходимости равны $+\infty. \blacksquare$

### Разложение логарифма и арктангенса

> [!error] Теорема 17.10.
> Ряды Маклорена функций $\ln(1 + x)$ и $\text{arctg } x$ сходятся к этим функциям в любой точке $x \in (-1; 1)$, т.е. при всех $x \in (-1; 1)$
> $$\ln(1 + x) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}x^n}{n}; \quad \text{arctg } x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{2n + 1};$$
> радиусы сходимости этих рядов равны 1.
> ^th17d10

**Доказательство:**
$\square$ Интегрируя разложение 
$$\frac{1}{1+x},$$
 имеем
$$\sum_{n=0}^{\infty} \frac{(-1)^n x^{n+1}}{n + 1} = \int_0^x \frac{dt}{1 + t} = \ln(1 + x), \quad -1 < x < 1.$$
После сдвига индекса в сумме получим нужное равенство:
$$\ln(1 + x) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}x^n}{n}, \quad -1 < x < 1. \tag{17.2}$$
Так как при интегрировании радиус сходимости степенного ряда не меняется, то у ряда в правой части (17.2) тот же радиус сходимости, что и у разложения 
$$\frac{1}{1+x},$$
 т.е. $R = 1$.
Интегрируя разложение 
$$\frac{1}{1+x^2} = \sum_{n=0}^{\infty} (-1)^n x^{2n},$$
 имеем
$$\sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{2n + 1} = \int_0^x \frac{dt}{1 + t^2} = \text{arctg } x, \quad -1 < x < 1. \tag{17.4}$$
Радиус сходимости этого ряда также равен 1. $\blacksquare$
