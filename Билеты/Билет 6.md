# 6) Дифференцируемость сложной функции

>[!error] **Теорема.** (Дифференцируемость суперпозиции)
>Пусть функции $x_1(t_1, \dots, t_k), \dots, x_n(t_1, \dots, t_k)$ дифференцируемы в точке $\bar{t^0} = (t_1^0, \dots, t_k^0)$, а функция $f(x_1, \dots, x_n)$ дифференцируема в точке $\bar{x^0} = (x_1^0, \dots, x_n^0)$, где $x_j^0 = x_j(\bar{t^0})$.
>Тогда сложная функция $F(\bar{t}) = f(x_1(\bar{t}), \dots, x_n(\bar{t}))$ дифференцируема в точке $\bar{t^0}$, и её частные производные вычисляются по формулам:
>$$\frac{\partial F}{\partial t_i} = \sum_{j=1}^n \frac{\partial f}{\partial x_j} \cdot \frac{\partial x_j}{\partial t_i}, \quad i = 1, \dots, k.$$
>^th06d1

**Доказательство:**
$\square$ Т.к. $f$ дифференцируема в $\bar{x^0}$, её приращение:
$$\Delta f = \sum_{j=1}^n A_j \Delta x_j + \alpha(\Delta \bar{x}) \cdot |\Delta \bar{x}|, \quad \text{где } A_j = \frac{\partial f}{\partial x_j}, \alpha \to 0 \text{ при } \Delta \bar{x} \to 0.$$
Т.к. $x_j$ дифференцируемы в $\bar{t^0}$, их приращения:
$$\Delta x_j = \sum_{i=1}^k B_{ij} \Delta t_i + \alpha_j(\Delta \bar{t}) \cdot |\Delta \bar{t}|, \quad \text{где } B_{ij} = \frac{\partial x_j}{\partial t_i}, \alpha_j \to 0 \text{ при } \Delta \bar{t} \to 0.$$
Подставим $\Delta x_j$ в приращение $\Delta f$:
$$\Delta f = \sum_{j=1}^n A_j \left( \sum_{i=1}^k B_{ij} \Delta t_i + \alpha_j \rho \right) + \alpha \cdot |\Delta \bar{x}|, \quad \text{где } \rho = |\Delta \bar{t}|.$$
Меняя порядок суммирования:
$$\Delta f = \sum_{i=1}^k \left( \sum_{j=1}^n A_j B_{ij} \right) \Delta t_i + \left( \sum_{j=1}^n A_j \alpha_j \right) \rho + \alpha \cdot |\Delta \bar{x}|.$$
Т.к. $x_j$ дифференцируемы, они непрерывны $\implies |\Delta \bar{x}| \le C \rho$. Тогда последние два слагаемых суть $o(\rho)$. Следовательно, функция $F$ дифференцируема, и коэффициенты при $\Delta t_i$ есть её частные производные $\frac{\partial F}{\partial t_i}$. $\blacksquare$

>[!info] **Свойство.** (Инвариантность формы первого дифференциала)
>Форма первого дифференциала $df = \sum_{j=1}^n \frac{\partial f}{\partial x_j} dx_j$ сохраняется независимо от того, являются ли $x_j$ независимыми переменными или дифференцируемыми функциями других переменных.
>^info06d1
