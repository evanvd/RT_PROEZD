# 8) Формула Тейлора с остаточным членом в форме Лагранжа

>[!error] **Теорема.** (Формула Тейлора с остатком в форме Лагранжа)
>Пусть $f \in C^{k+1}(U_\delta(\bar{x^0}))$. Тогда для любой точки $\bar{x} \in U_\delta(\bar{x^0})$ найдётся число $\xi \in (0, 1)$ такое, что:
>$$f(\bar{x}) = f(\bar{x^0}) + \sum_{j=1}^k \frac{d^j f(\bar{x^0})}{j!} + \frac{d^{k+1} f(\bar{x^0} + \xi \Delta \bar{x})}{(k + 1)!},$$
>где $d^j f$ — дифференциал $j$-го порядка, вычисленный при приращениях $\Delta x_i = x_i - x_i^0$.
>^th08d1

**Доказательство:**
$\square$ Проведём доказательство для $n=2$. Рассмотрим вспомогательную функцию одной переменной $t \in [0, 1]$:
$$F(t) = f(x_0 + t \Delta x, y_0 + t \Delta y).$$
По правилу дифференцирования сложной функции:
$F'(t) = f'_x \Delta x + f'_y \Delta y = df(x_0 + t \Delta x, y_0 + t \Delta y)$.
Используя индукцию, можно показать, что $F^{(j)}(t) = d^j f(x_0 + t \Delta x, y_0 + t \Delta y)$.
Применим к функции $F(t)$ одномерную формулу Тейлора на отрезке $[0, 1]$ с остаточным членом в форме Лагранжа:
$$F(1) = F(0) + \sum_{j=1}^k \frac{F^{(j)}(0)}{j!} + \frac{F^{(k+1)}(\xi)}{(k + 1)!}, \quad \text{где } \xi \in (0, 1).$$
Так как $F(1) = f(x, y)$ и $F(0) = f(x_0, y_0)$, подстановка производных $F^{(j)}$ даёт искомую формулу. $\blacksquare$
