# Билет 8. Формула Тейлора с остаточным членом в форме Лагранжа

>[!error] **Теорема 10.8.** (Формула Тейлора с остатком в форме Лагранжа)
> Пусть при некотором $\delta > 0$ функция $n$ переменных $f \in C^{k+1}(U_\delta(x_1^0, \dots, x_n^0))$, где $k = 0, 1, 2, \dots$ Тогда для любой точки $(x_1, \dots, x_n) \in U_\delta(x_1^0, \dots, x_n^0)$ найдётся число $\xi \in (0; 1)$ такое, что
> $$f(x_1, \dots, x_n) = f(x_1^0, \dots, x_n^0) + \sum_{j=1}^k \frac{d^j f(x_1^0, \dots, x_n^0)}{j!} + \frac{d^{k+1} f(x_1^0 + \xi \Delta x_1, \dots, x_n^0 + \xi \Delta x_n)}{(k + 1)!}, \tag{10.16}$$
> где $d^j f = \left( \Delta x_1 \frac{\partial}{\partial x_1} + \dots + \Delta x_n \frac{\partial}{\partial x_n} \right)^j f$ — дифференциал $j$-го порядка функции $f$.
> ^th08d1

**Доказательство:**
$\square$ Проведём доказательство для $n = 2$. Пусть $f \in C^{k+1}(U_\delta(x_0, y_0))$. Рассмотрим функцию одной переменной $F(t) = f(x_0 + t\Delta x, y_0 + t\Delta y)$, где $t \in [0; 1]$.
По теореме о дифференцируемости сложной функции:
$F'(t) = f'_x \Delta x + f'_y \Delta y = df(x_0 + t\Delta x, y_0 + t\Delta y)$.
Индукцией по $j \le k + 1$: $F^{(j)}(t) = d^j f(x_0 + t\Delta x, y_0 + t\Delta y)$.
Применим к $F(t)$ одномерную формулу Тейлора с остатком в форме Лагранжа на $[0; 1]$:
$$F(1) = F(0) + \sum_{j=1}^k \frac{F^{(j)}(0)}{j!} + \frac{F^{(k+1)}(\xi)}{(k + 1)!}, \quad \xi \in (0; 1).$$
Подставляя $F(1) = f(x_0 + \Delta x, y_0 + \Delta y)$ и $F^{(j)}(0) = d^j f(x_0, y_0)$, получаем требуемую формулу. $\blacksquare$
