# Билет 8. Формула Тейлора с остаточным членом в форме Лагранжа

### Дифференциалы высших порядков

>[!info] **Определение 10.7.** (Дифференциалы высших порядков)
> Дифференциалом второго порядка функции $f$ называется дифференциал от её первого дифференциала: $d^2 f = d(df)$. Аналогично $d^k f = d(d^{k-1} f)$. 
> ^def10d7

Для независимых переменных $dx_i$ считаются постоянными ($d(dx_i) = 0$), поэтому
$$d^2 f = \sum_{i,j=1}^n \frac{\partial^2 f}{\partial x_i \partial x_j} dx_i dx_j = \left( \sum \frac{\partial}{\partial x_i} dx_i \right)^2 f.$$

**Отсутствие инвариантности:** Если $x_i$ — зависимые переменные, то $d(dx_i) = d^2 x_i \ne 0$. В этом случае
$$d^2 f = \sum \sum \frac{\partial^2 f}{\partial x_i \partial x_j} dx_i dx_j + \sum \frac{\partial f}{\partial x_i} d^2 x_i.$$
Вторая сумма нарушает инвариантность формы дифференциала (начиная со второго порядка).

### Формула Тейлора для функций нескольких переменных

>[!error] **Теорема 10.8.** (Формула Тейлора с остатком в форме Лагранжа)
> Пусть при некотором $\delta > 0$ функция $n$ переменных $f \in C^{k+1}(U_\delta(x_1^0, \dots, x_n^0))$, где $k = 0, 1, 2, \dots$ Тогда для любой точки $(x_1, \dots, x_n) \in U_\delta(x_1^0, \dots, x_n^0)$ найдётся число $\xi \in (0; 1)$ такое, что
> $$f(x_1, \dots, x_n) = f(x_1^0, \dots, x_n^0) + \sum_{j=1}^k \frac{d^j f(x_1^0, \dots, x_n^0)}{j!} + \frac{d^{k+1} f(x_1^0 + \xi \Delta x_1, \dots, x_n^0 + \xi \Delta x_n)}{(k + 1)!}, \tag{10.16}$$
> где 
> $$d^j f = \left( \Delta x_1 \frac{\partial}{\partial x_1} + \dots + \Delta x_n \frac{\partial}{\partial x_n} \right)^j f, j = 1, \dots, k + 1$$
>  — дифференциал $j$-го порядка функции $f$ в соответствующей точке; 
> $$\Delta x_i = x_i - x_i^0, i = 1, \dots, n; \sqrt{(\Delta x_1)^2 + \dots + (\Delta x_n)^2} < \delta.$$
>  
> ^th10d8

**Доказательство:**
$\square$ Доказательство проведём для случая $n = 2$; для больших $n$ доказательство аналогично.
Пусть функция двух переменных $f \in C^{k+1}(U_\delta(x_0, y_0))$. Рассмотрим функцию одной переменной $F(t) = f(x_0 + t\Delta x, y_0 + t\Delta y)$, где $\Delta x$ и $\Delta y$ такие, что 
$$\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2} < \delta;$$
 $t$ — такое, что $(x_0 + t\Delta x, y_0 + t\Delta y) \in U_\delta(x_0, y_0)$, т.е. $t \in (-1 - \varepsilon; 1 + \varepsilon)$, где $\varepsilon$ — некоторое положительное число (точнее, 
$$\varepsilon = \frac{\delta}{\rho} - 1$$
).
По **Теореме 10.3 (Дифференцируемость сложной функции)** при всех $t \in (-1 - \varepsilon; 1 + \varepsilon)$ существует
$$F'(t) = f'_x(x_0 + t\Delta x, y_0 + t\Delta y) \Delta x + f'_y(x_0 + t\Delta x, y_0 + t\Delta y) \Delta y = df(x_0 + t\Delta x, y_0 + t\Delta y).$$
Покажем по индукции, что при 
$$1 \le j \le k + 1$$
 при всех $t \in (-1 - \varepsilon; 1 + \varepsilon)$ существует $F^{(j)}(t) = d^j f(x_0 + t\Delta x, y_0 + t\Delta y)$. В самом деле, при $j = 1$ это верно. Если утверждение верно для некотого $j \le k$, то 
$$F^{(j+1)}(t) = \frac{d}{dt} (d^j f(x_0 + t\Delta x, y_0 + t\Delta y)).$$
 Но все частные производные функции $f$ порядка вплоть до $k + 1$ непрерывны, поэтому $d^j f(x, y)$ — непрерывно дифференцируемая функция двух переменных в $U_\delta(x_0, y_0)$, и по **Теореме 10.3 (Дифференцируемость сложной функции)** при всех $t \in (-1 - \varepsilon; 1 + \varepsilon)$ существует
$$F^{(j+1)}(t) = (d^j f(x_0 + t\Delta x, y_0 + t\Delta y))'_x \cdot \Delta x + (d^j f(x_0 + t\Delta x, y_0 + t\Delta y))'_y \cdot \Delta y = d(d^j f(x_0 + t\Delta x, y_0 + t\Delta y)).$$

Внешний дифференциал подразумевает дифференцирование по $x, y$ при постоянных $\Delta x, \Delta y$; по [[#^def10d7|Определению 10.7]] он равен $d^{j+1} f(x_0 + t\Delta x, y_0 + t\Delta y)$. Индукция завершена.
Так как функция одной переменной $F$ имеет $k + 1$ производную на $(-1 - \varepsilon; 1 + \varepsilon)$, то по одномерной формуле Тейлора с остаточным членом в форме Лагранжа (теорема 4.19) при $t \in (-1 - \varepsilon; 1 + \varepsilon)$ существует $\xi \in (0; t)$ такое, что
$$F(t) = F(0) + \sum_{j=1}^k \frac{F^{(j)}(0)}{j!} t^j + \frac{F^{(k+1)}(\xi)}{(k + 1)!} t^{k+1}.$$
При $t = 1$ получим
$$f(x_0 + \Delta x, y_0 + \Delta y) = f(x_0, y_0) + \sum_{j=1}^k \frac{d^j f(x_0, y_0)}{j!} + \frac{d^{k+1} f(x_0 + \xi \Delta x, y_0 + \xi \Delta y)}{(k + 1)!},$$
где $\xi \in (0; 1)$. $\blacksquare$
