## 1 Ранг матрицы

**Определение 1.1.** Набор матриц назовем *линейно зависимым*, если существует нетривиальная линейная комбинация, равная нулевой матрице.

Все теоремы о линейной зависимости для векторов верны и для матриц.

**Теорема 1.1.** *Если некоторая матрица раскладывается на линейную комбинацию некоторого набора матриц, то это разложение единственно тогда и только тогда, когда этот набор линейно независимый.*

*Доказательство.*  
Доказательство аналогично доказательству соответствующей теоремы для векторов.  
$\square$

Пусть набор строк произвольной матрицы $A$ линейно зависим, т.е. существует ненулевая строка $\bar{c}$ такая, что $\bar{c}A = \bar{0}$.

**Теорема 1.2.** *Пусть $A, B$ — квадратные матрицы, а строки $A$ линейно зависимы. Тогда строки $AB$ тоже линейно зависимы.*

*Доказательство.*  
Строки $A$ - линейно зависимые $\Longleftrightarrow \exists \bar{c} \neq \bar{0} : \bar{c}A = \bar{0}$. Тогда:
$$\bar{c}(AB) = (\bar{c}A)B = \bar{0} \cdot B = \bar{0}$$
Значит, набор строк $AB$ линейно зависим.  
$\square$

**Теорема 1.3.** *Элементарные преобразования строк (столбцов) переводят линейно независимый набор в линейно независимый, а линейно зависимый в линейно зависимый.*

*Доказательство.*  
1. Пусть $\bar{a}_1, \dots, \bar{a}_k$ — линейно независимый набор.  
Пусть к первой строке прибавили вторую: $\bar{a}_1 + \bar{a}_2, \bar{a}_2, \dots, \bar{a}_k$. Тогда линейная комбинация: $\alpha_1(\bar{a}_1 + \bar{a}_2) + \alpha_2\bar{a}_2 + \dots + \alpha_k\bar{a}_k$ или $\alpha_1\bar{a}_1 + (\alpha_1 + \alpha_2)\bar{a}_2 + \dots + \alpha_k\bar{a}_k \Longrightarrow$ т.к. набор линейно независимый, линейная комбинация равна нулю, когда $\alpha_1 = 0, \alpha_1 + \alpha_2 = 0 \Longrightarrow \alpha_2 = 0, \dots, \alpha_k = 0$. Получили тривиальный набор коэффициентов, значит новый набор тоже линейно независимый.
2. Предположим противное и получим противоречие, т.к. элементарные операции обратимы.  
$\square$

**Теорема 1.4.** *При элементарных преобразованиях со столбцами (строками) линейная зависимость строк (столбцов) сохраняется.*

*Доказательство.*  
Пусть $A$ — матрица с линейно зависимыми строками. Тогда $\exists \bar{c} \neq \bar{0} : \bar{c}A = \bar{0}$. Пусть мы провели некоторые элементарные операции с $A$, т.е. получили элементарную матрицу $S$. Тогда:
$$\bar{c}(AS) = (\bar{c}A)S = \bar{0}$$
$\square$

**Теорема 1.5.** *$\det A = 0 \Longleftrightarrow$ строки (столбцы) $A$ линейно зависимы.*

*Доказательство.*  
$\Longleftarrow$  
Какая-то строка — линейная комбинация остальных $\Longrightarrow$ элементарные преобразования $\Longrightarrow \det A = 0$

$\Longrightarrow$  
Пусть $\det A \neq 0$, тогда через элементарные преобразования строк $A$ можно привести к единичной матрице, но ее строки линейно независимые, пришли к противоречию.  
$\square$

**Теорема 1.6.** *Пусть квадратная матрица $A$ не вырождена. Тогда для любого столбца (строки) $\bar{d}$ высоты $n$ существует и единственнен набор коэффициентов $\bar{x} = x_1, \dots, x_n : A\bar{x} = \bar{d}$*

Далее будем говорить о матрицах произвольного размера, если не обговорено обратное.

**Определение 1.2.** Пусть имеется матрица $A$, у которой существует линейно независимый набор из $r_1$ строк, а $\forall r > r_1$ набор линейно зависим или не существует. Тогда будем говорить, что $r_1 = r_1(A)$ — *строчный ранг матрицы $A$.*

*Строчный ранг матрицы $A$* — максимальное количество строк, принадлежащих матрице $A$ и образующих линейно независимый набор.

Любой линейно независимый набор из $r_1(A)$ строк матрицы $A$ будем называть *базисом*, а строки *базисными*.

**Теорема 1.7.** *Любая строка матрицы равна линейной комбинации базисных.*

*Доказательство.*  
1. Эта строка базисная, значит просто обнуляем все строки в комбинации, кроме нее.
2. Эта строка, обозначим как $\tilde{a}$, не является базисной. Пусть $a_1, \dots, a_m$ — базисные, тогда $a_1, \dots, a_m, \tilde{a}$ — линейно зависимый набор, по определению строчного ранга $\Longrightarrow$ существует нетривиальная комбинация равная нулю, т.е.
$$\alpha_1 a_1 + \dots + \alpha_m a_m + \beta\tilde{a} = 0$$
При этом $\beta \neq 0$, т.к. в противном случае это был бы линейно независимый набор $\Longrightarrow \tilde{a}$ можно выразить через базисные строки.  
$\square$

**Определение 1.3.** Аналогично строчному рангу определяется *столбцовый ранг* $r_2(A)$ — максимальное количество столбцов, принадлежащих матрице $A$ и образующих линейно независимый набор.

**Замечание.** Базисные строки (столбцы) не всегда определяются единственным образом.

**Утверждение 1.1.** *Элементарные операции со строками и столбцами не меняют строчный и столбцовый ранги.*

**Утверждение 1.2.** *Пусть $A'$ — подматрица $A$. Тогда*
$$r_1(A') \leqslant r_1(A)$$
$$r_2(A') \leqslant r_2(A)$$

*Доказательство.*  
Если в $A$ некоторый набор строк (столбцов) линейно зависим, то в $A'$ набор строк (столбцов), которые являются частями изначальных также линейно зависим с теми же коэффициентами.  
$\square$

**Теорема 1.8.** *Пусть имеется набор, состоящий из $n + 1$ столбца (строки) высоты $n$. Тогда этот набор всегда будет линейно зависимым.*

*Доказательство.*  
Пусть заданы столбцы $a_1, \dots, a_{n+1}$.  
Для $a_1, \dots, a_n$ возможны 2 случая:
1. Этот поднабор линейно зависим, тогда весь набор линейно зависим.
2. Этот поднабор линейно независим, тогда $\exists x = x_1, \dots, x_n : Ax = a_{n+1} \ (\det A \neq 0) \Longrightarrow$ набор линейно зависим.  
$\square$

**Теорема 1.9.** *У любой матрицы $A$ выполняется $r_1(A) = r_2(A)$.*

*Доказательство.*  
Будем рассматривать ненулевую матрицу, т.к. для нулевой тривиально.

С помощью элементарных операций обнулим небазисные строки и получим матрицу $A'$ с $r_1(A)$ ненулевых строк. Линейная зависимость при элементарных операциях сохраняется, значит базисные столбцы остались теми же, что и в $A$.

Рассмотрим $D$ — матрицу, состоящую из пересечения базисных строк и столбцов $A' \Longrightarrow r_1(A) \geqslant r_2(A)$, т.к. $r_1$ — высота столбцов $D$, а $r_2$ — их количество.

Транспонируем матрицу $A$ и аналогичным способом получим $r_1(A) \leqslant r_2(A)$.  
Итого: $r_1(A) = r_2(A)$.  
$\square$

**Утверждение 1.3.** *Пусть $B_{k \times k}$ — невырожденная подматрица $A$. Тогда строки (столбцы) $A$, содержащие $B$ являются линейно независимыми.*

*Доказательство.*  
Если бы строки (столбцы) $A$ были линейно зависимыми, то и их части, являющиеся строками (столбцами) $B$, были бы линейно зависимыми. Но тогда $B$ — вырожденная матрица, получили противоречие.  
$\square$

**Следствие.** Не существует невырожденной подматрицы $A$ размера $k \times k$ такой, что $k > \operatorname{rg} A$.

Таким образом, доказана **теорема о ранге матрицы**:
$$\forall A \hookrightarrow r_1(A) = r_2(A) = r,$$
где $r$ - максимальный размер невырожденной подматрицы $A$. Эта подматрица является базисной.

**Определение 1.4.** *Минорный ранг* — максимальный размер невырожденной подматрицы.

**Теорема 1.10** (О базисном миноре). *Любая строка (столбец) матрицы может быть представлена как линейная комбинация строк (столбцов), содержащих базисный минор.*

*Доказательство.*  
Достаточно доказать, что строки (столбцы), содержащие базисный минор линейно независимые.

Предположим противное: если они линейно зависимы, то базисный минор — вырожденная матрица, что противоречит определению.

А т.к. строки (столбцы) линейно независимы и их количество равно рангу матрицы, они являются базисными.  
$\square$

**Теорема 1.11** (Кронекера-Капелли). *Пусть имеется некоторая матрица $A$. Приписывание к $A$ столбца (строки) $b$ не меняет ранга тогда и только тогда, когда $b$ является линейной комбинацией столбцов (строк) $A$.*

*Доказательство.*  
$\Longleftarrow$  
Припишем столбец $b$, являющийся линейной комбинацией столбцов $A$ к матрице: $A|b$. Тогда элементарными преобразованиями можно обнулить $b \Longrightarrow \operatorname{rg}(A|b) = \operatorname{rg}(A|0) = \operatorname{rg} A$.

$\Longrightarrow$  
Пусть $\operatorname{rg}(A|b) = \operatorname{rg} A$. Тогда в $A$ есть набор из $n$ базисных столбцов, но в $(A|b)$ те же столбцы образуют базис, значит $b$ можно выразить через них.  
$\square$

**Теорема 1.12** (О ранге произведения матриц). *Пусть имеются некоторые матрицы $A$ и $B$. Тогда:*
$$\operatorname{rg}(AB) \leqslant \operatorname{rg} A$$
$$\operatorname{rg}(AB) \leqslant \operatorname{rg} B$$

*Доказательство.*  
$\operatorname{rg}(AB) \leqslant \operatorname{rg}(A|AB)$, т.к. $AB$ — подматрица. Но $AB$ является линейной комбинацией столбцов $A$ с коэффициентами из $B \Longrightarrow$ элементарными преобразованиями можно обнулить $AB \Longrightarrow \operatorname{rg}(AB) \leqslant \operatorname{rg}(A|0) = \operatorname{rg} A$.

Аналогично для $\operatorname{rg}(AB) \leqslant \operatorname{rg} B$.  
$\square$

**Утверждение 1.4.** *Ранг может сохраниться при умножении, если $A$ невырожденная и определено $AB \Longrightarrow \operatorname{rg}(AB) = \operatorname{rg} B$. Аналогично, если $B$ невырожденная.*

*Доказательство.*  
По предыдущей теореме $\operatorname{rg}(AB) \leqslant \operatorname{rg} B$.  
Т.к. $A$ — невырожденная, существует ее обратная матрица $\Longrightarrow$
$$\operatorname{rg}(A^{-1}AB) = \operatorname{rg}(EB) = \operatorname{rg} B \leqslant \operatorname{rg}(AB) \leqslant \operatorname{rg} B$$
Значит $\operatorname{rg}(AB) = \operatorname{rg} B$.  
$\square$

**Определение 1.5.** Матрица $A_{m \times n}$ имеет *упрощенный вид*, если нижние $m - k$ строк нулевые, а некоторые $k$ столбцов являются столбцами единичной матрицы (достаточно треугольной матрицы).

Матрица $A_{m \times n}$ имеет *простейший вид*, если эти столбцы стоят слева на первых $k$ местах.

### Алгоритм нахождения ранга матрицы по методу Гаусса

Покажем, что методом Гаусса можно получить упрощенный вид некоторой матрицы $A$: если получаем нулевой столбец, пропускаем его, иначе, действуя методом Гаусса получаем $a_{11}, \dots, a_{kk}$ и нулевые строки снизу:
$$\begin{pmatrix}
a_{11} & \dots & \dots & \dots \\
0 & \dots & \dots & \dots \\
\dots & \dots & \dots & \dots \\
0 & \dots & a_{kk} & \dots \\
\dots & \dots & \dots & \dots \\
0 & \dots & \dots & 0
\end{pmatrix}$$

Далее элементарными преобразованиями со строками преобразуем $a_{11}, \dots, a_{kk}$ в единицы. Тогда $\operatorname{rg} A = k$.

**Теорема 1.13** (О ранге суммы матриц).
$$\operatorname{rg}(A+B) \leqslant \operatorname{rg} A + \operatorname{rg} B$$

*Доказательство.*  
К $(A+B)$ припишем матрицы $A$ и $B$. Тогда из теоремы о ранге подматрицы:
$$\operatorname{rg}(A+B) \leqslant \operatorname{rg}(A+B | A | B)$$

Пусть $\operatorname{rg} A = m$ (т.е. в $A$ существует $m$ линейно независимых столбцов) и $\operatorname{rg} B = k$ (т.е. в $B$ существует $k$ линейно независимых столбцов).

Из $i$-ого столбца $A+B$ будем вычитать сумму $i$-ых столбцов $A$ и $B$, и таким образом обнулим $A+B$. Далее обнулим небазисные столбцы $A$ и $B$, т.е. останется $m+k$ столбцов $\Longrightarrow$
$$\operatorname{rg}(A+B) \leqslant m + k = \operatorname{rg} A + \operatorname{rg} B$$
$\square$

## 2 Системы линейных уравнений

Системы линейных уравнений имеют вид:
$$\begin{cases}
a_{11}x_1 + \dots + a_{1n}x_n = b_1 \\
\dots \\
a_{m1}x_1 + \dots + a_{mn}x_n = b_m
\end{cases}$$
где $A$ — матрица системы, $A|b$ — расширенная матрица системы, $x$ — множество решений системы. Систему можно также записать в виде $Ax = b$.

Элементарным операциям со строками $(A|b)$ соответствуют операции с уравнениями системы, т.е. мы получим следствие системы.

Единственная допустимая операция со столбцами $A$ — перестановки.

Элементарные операции обратимы, значит они не меняют множество решений.

У системы есть решение, когда столбец $b$ — линейная комбинация столбцов $A$ (из теоремы Кронекера-Капелли).

**Теорема 2.1** (Критерий совместности системы).  
*Система $Ax = b$ совместна* $\Longleftrightarrow \operatorname{rg}(A|b) = \operatorname{rg} A$.

*Аналогичная формулировка: система несовместна* $\Longleftrightarrow$ при приведении $(A|b)$ к упрощенному виду есть строка вида $(0, \dots, 0, 1)$.

*Доказательство.*  
$\Longleftarrow$  
$$0 \cdot x_1 + \dots + 0 \cdot x_n = 1$$
Очевидно, что нет решений.

$\Longrightarrow$  
Пусть система несовместна, значит по теореме Кронекера-Капелли $\operatorname{rg}(A|b) > \operatorname{rg} A$. Приведем $(A|b)$ к упрощенному виду. Тогда будет $\operatorname{rg} A + 1$ столбец единичной матрицы, получим строку $(0, \dots, 0, 1)$ - противоречие.  
$\square$

**Теорема 2.2** (Критерий совместности системы — теорема Фредгольма). *Пусть дана система $Ax = b$. Рассмотрим сопряженную однородную систему $A^T y = 0$. Тогда $Ax = b$ совместна $\Longleftrightarrow \forall$ решения $y$ системы $A^T y = 0 \hookrightarrow y^T b = 0$.*

*Доказательство.*  
$\Longrightarrow$  
Пусть $Ax = b$ совместна $\Longrightarrow \exists x : Ax = b$. Домножим на $y^T \Longrightarrow y^T(Ax) = y^T b$, при этом, если $y$ — решение, то $y^T A = 0 \Longrightarrow \forall y \hookrightarrow y^T b = 0$.

$\Longleftarrow$  
Пусть $Ax = b$ несовместна $\Longrightarrow$ через элементарные преобразования можно привести матрицу $(A|b)$ к виду, в котором последняя строка $(0, \dots, 0, 1) \Longrightarrow \exists y : y^T(A|b) = (0, \dots, 0, 1) \Longrightarrow y^T A = 0$ и $y^T b = 1$, получили противоречие.  
$\square$

Рассмотрим алгоритм решения совместной системы $Ax = b$.

Пусть $h_1$ — решение $Ax = b$, $h_2$ — решение $Ax = 0 \Longrightarrow h_1 + h_2$ — решение $Ax = b$, т.к. $A(h_1 - h_2) = Ah_1 - Ah_2 = b - b = 0$.

Тогда решение $Ax = b \rightarrow \langle\text{частное решение } Ax = b\rangle + \langle\text{общее решение } Ax = 0\rangle$.

**Замечание.** Пусть $Ax = 0$, $h_1, h_2$ — ее решения. Тогда $\forall \alpha_1, \alpha_2 \hookrightarrow \alpha_1 h_1 + \alpha_2 h_2$ — тоже решение.

**Определение 2.1.** Пусть $Ax = 0$ — система с $n$ неизвестными. Матрицу $F$ из столбцов высоты $n$ назовем *фундаментальной матрицей системы* $Ax = 0$, если она удовлетворяет следующим условиям:
1. $AF = 0$, т.е. любой столбец $F$ является решением системы.
2. Столбцы $F$ линейно независимы.
3. $\operatorname{rg} F$ максимальный среди всех матриц, удовлетворяющих (1).

Столбцы $F$ — *фундаментальная система решений*. Любое решение выражается через них.

## 3 Решение неоднородной системы уравнений

**Теорема 3.1.** *Пусть $F$ - фундаментальная система решений для однородной системы уравнений $A$. Тогда:*
$$\operatorname{rg} F \leqslant n - \operatorname{rg} A$$

*Доказательство.*  
Сначала с помощью элементарных преобразований со строками приведем матрицу $A$ к простейшему виду, после чего преобразованием столбцов обнулим строки и столбцы матрицы, кроме единичной подматрицы:
$$A \longrightarrow \begin{pmatrix}
\dots & \dots & \dots & \dots & \dots & \dots \\
\dots & \dots & \dots & \dots & \dots & \dots \\
0 & 0 & 0 & \dots & 0 & 0
\end{pmatrix} \longrightarrow \begin{pmatrix}
E & \dots & \dots & \dots & \dots & \dots \\
0 & 0 & \dots & \dots & 0 & 0 \\
0 & 0 & 0 & \dots & 0 & 0
\end{pmatrix} \longrightarrow$$
$$\longrightarrow \begin{pmatrix}
E & 0 & 0 & \dots & 0 & 0 \\
0 & 0 & \dots & \dots & 0 & 0 \\
0 & 0 & 0 & \dots & 0 & 0
\end{pmatrix} = A'$$

$A' = PAQ$, т.к. элементарным операциям со строками и столбцами соответствует домножение исходной матрицы $A$ на элементарные матрицы слева и справа. $P$ и $Q$ - невырожденные, значит существуют матрицы, обратные им.

Рассмотрим $F' = Q^{-1}F$, при этом $\operatorname{rg} F' = \operatorname{rg} F$ по доказанной ранее теореме. Тогда в матрице $A'F'$ первые $\operatorname{rg} A$ строк совпадают с соответствующими строками $F'$. С другой стороны:
$$A'F' = PAQQ^{-1}F = PAF = 0,$$
т.к. по определению $AF = 0$. Тогда первые $\operatorname{rg} A$ строк $A'F'$ нулевые $\Longrightarrow \operatorname{rg} F' \leqslant n - \operatorname{rg} A$. Но $\operatorname{rg} F' = \operatorname{rg} F$, следовательно $\operatorname{rg} F \leqslant n - \operatorname{rg} A$.  
$\square$

**Теорема 3.2.** *Пусть $F$ - фундаментальная система решений для однородной системы уравнений $A$. Тогда:*
$$\operatorname{rg} F = n - \operatorname{rg} A$$

*Доказательство.*  
Рассмотрим систему $Ax = 0$. Тогда элементарными преобразованиями со строками $A$ приведем матрицу к упрощенному виду. Останется базисная подматрица, которая задает $\operatorname{rg} A$ базисных переменных и $n - \operatorname{rg} A$ небазисных.

Рассмотрим $n - \operatorname{rg} A$ столбцов, последние $n - \operatorname{rg} A$ строк которых задают единичную матрицу. Тогда оставшиеся $\operatorname{rg} A$ базисных переменных находятся однозначно, т.е. получена фундаментальная система решений и:
$$\operatorname{rg} F = n - \operatorname{rg} A$$
$\square$

Решение системы $Ax = 0$ в общем случае имеет вид:
$$\bar{x} = c_1\bar{h}_1 + \dots + c_{n-\operatorname{rg} A}\bar{h}_{n-\operatorname{rg} A}$$
или:
$$\bar{x} = F\bar{c}$$

Рассмотрим теперь решение неоднородной системы уравнений $Ax = b$. Его общий вид:
$$\bar{x} = \bar{h}_c + F\bar{c},$$
где $\bar{h}_c$ - частное решение системы, $F\bar{c}$ - решение соответствующей однородной системы.

### Алгоритм решения неоднородной системы уравнений

1. Припишем столбец $b$ к матрице $A$ справа.
2. С помощью элементарных преобразований *только* строк приведем к упрощенному виду. Если невозможно получить нулевую строку, то система не имеет решений.
3. Решаем систему $Ax = 0$.
4. Подбираем частное решение.

Если матрица системы квадратная, то:
* $\triangleright$ Если $\det A = 0$, то система имеет единственное решение.
* $\triangleright$ Если $\det A \neq 0$, то система или не имеет решений, или имеет бесконечное количество решений.

## 4 Линейные пространства

**Определение 4.1.** Множество $L$, элементы которого - векторы, называется *линейным пространством*, если на нем заданы:
1. операция сравнения;
2. операция сложения "+": $\forall x, y \in L \hookrightarrow x + y \in L$;
3. операция умножения на действительное или комплексное число "$\cdot$": $\forall \alpha \in \mathbb{R}(\mathbb{C}) \ \forall x \in L \hookrightarrow \alpha \cdot x \in L$.

Эти операции удовлетворяют следующим 8 аксиомам:
1. $\forall x, y \in L \hookrightarrow x + y = y + x$
2. $\forall x, y, z \in L \hookrightarrow (x + y) + z = x + (y + z)$
3. $\exists 0 \in L : \forall x \in L \hookrightarrow 0 + x = x$
4. $\forall x \in L \ \exists -x \in L : x + (-x) = 0$
5. $\forall \alpha, \beta \in \mathbb{R}(\mathbb{C}), \forall x \in L \hookrightarrow (\alpha\beta)x = \alpha(\beta x)$
6. $\forall x \in L \hookrightarrow 1 \cdot x = x$
7. $\forall \alpha, \beta \in \mathbb{R}(\mathbb{C}), \forall x \in L \hookrightarrow (\alpha + \beta)x = \alpha x + \beta x$
8. $\forall \alpha \in \mathbb{R}(\mathbb{C}), \forall x, y \in L \hookrightarrow \alpha(x + y) = \alpha x + \alpha y$

Любое линейное пространство непустое.

**Утверждение 4.1.** *0 в любом линейном пространстве единственнен.*

*Доказательство.*  
Предположим противное, т.е. $\exists 0', 0'' \Longrightarrow$
$$0' + 0'' = 0'' + 0' = 0'' = 0'$$
Получили противоречие.  
$\square$

### Примеры линейных пространств
1. Стандартные векторы со стандартными операциями.
2. Матрицы со стандартными операциями.
3. Многочлены степени не выше $n$.
4. Множество решений $Ax = 0$.

**Определение 4.2.** *Линейная комбинация* - элемент $x$ линейного пространства такой, что:
$$x = \alpha_1 a_1 + \dots + \alpha_n a_n,$$
где $\alpha_1, \dots, \alpha_n \in \mathbb{R}(\mathbb{C})$.

Все теоремы про линейные комбинации и их доказательства аналогичны тем, что были изучены в 1 семестре.

**Определение 4.3.** *Базис* в линейном пространстве $L$ - упорядоченный **конечный** набор векторов такой, что:
1. Это линейно независимый набор.
2. Любой вектор $L$ является линейной комбинацией этого набора.

Любой элемент раскладывается по базису однозначно, коэффициенты разложения — *координаты*.

**Теорема 4.1.**
$$\forall x, y \in L \hookrightarrow x+y = \begin{pmatrix} x_1 \\ \dots \\ x_n \end{pmatrix} + \begin{pmatrix} y_1 \\ \dots \\ y_n \end{pmatrix} = \begin{pmatrix} x_1 + y_1 \\ \dots \\ x_n + y_n \end{pmatrix}$$

**Теорема 4.2.**
$$\forall \alpha \in \mathbb{R}(\mathbb{C}), \forall x \in L \hookrightarrow \alpha x = \begin{pmatrix} \alpha x_1 \\ \dots \\ \alpha x_n \end{pmatrix}$$

**Теорема 4.3.** *Количество векторов во всех базисах одного пространства одинаково, т.е.*
$$L : (e_1, \dots, e_n), (f_1, \dots, f_m) \text{ — базисы } \Longrightarrow n = m$$

*Доказательство.*  
Пусть $m > n$. Каждый из векторов $f$ разложим по $e$. Тогда получим матрицу $m \times n \Longrightarrow \operatorname{rg} A \leqslant n$, но количество столбцов больше $\operatorname{rg} A$, значит координатные столбцы $f$ линейно зависимы $\Longrightarrow$ векторы тоже линейно зависимы, значит $f$ не является базисом. Получили противоречие.  
Аналогично для $n > m$.  
$\square$

**Определение 4.4.** Количество элементов в базисе — *размерность*.  
Обозначается как $\dim L = n$.  
Размерность главного (нулевого) пространства равна 0.

### Примеры
1. Пространство матриц $m \times n$: $\dim = m \cdot n$.
2. Пространство многочленов степени не выше $n$: $\dim = n + 1$.

**Следствие.** В пространстве функций, непрерывных на отрезке, нет конечного базиса.

**Теорема 4.4.** *Если $\dim L = n$, то любой упорядоченный линейно независимый набор из $n$ векторов - базис.*

*Доказательство.*  
Пусть $x_1, \dots, x_n$ - такой набор векторов. Докажем, что произвольный вектор $y$ раскладывается по нему.  
Заметим, что система $x_1, \dots, x_n, y$ линейно зависима, значит найдутся такие коэффициенты $\alpha_1, \dots, \alpha_n, \beta$, что:
$$\alpha_1 x_1 + \dots + \alpha_n x_n + \beta y = 0$$
Причем $\beta \neq 0$, т.к. иначе изначальная система была бы линейно зависима.  
$\square$

**Лемма 4.1.** *Пусть $\dim L = n$, $x_1, \dots, x_m$ — линейно независимый набор, $m < n$. Тогда этот набор можно дополнить до базиса.*

*Доказательство.*  
Будем добавлять вектора, которые не раскладываются по данному набору. Такие обязательно найдутся из определения базиса.  
$\square$

Далее работаем в конечном линейном пространстве, если не сказано обратное.

Пусть есть 2 базиса: $(e_1, \dots, e_n), (e'_1, \dots, e'_n)$. Тогда, чтобы перейти из одного базиса в другой, необходимо выразить базисные векторы $e'$ через базис $e$:
$$(e'_1, \dots, e'_n) = (e_1, \dots, e_n)S \Longleftrightarrow e' = e \cdot S$$
Отсюда $e' \cdot S^{-1} = e$.

Для координат:
$$x = e \cdot \xi = e' \cdot \xi' = e \cdot S \cdot \xi' \Longleftrightarrow \xi = S\xi'$$

## 5 Подпространства

**Определение 5.1.** Непустое подмножество $L'$ множества $L$ будем называть *подпространством*, если:
1. $\forall x, y \in L' \hookrightarrow x + y \in L'$
2. $\forall \alpha \in \mathbb{R}(\mathbb{C}), \forall x \in L' \hookrightarrow \alpha x \in L'$

В $L'$ всегда есть нейтральный элемент, оно замкнуто относительно сложения и умножения на число, все аксиомы выполнены. Значит $L'$ - линейное пространство.

У $L$ всегда есть нулевое подпространство. Само пространство также является своим подпространством.

**Определение 5.2.** Пусть $M \subset L$. Рассмотрим все возможные линейные комбинации, в которые входит конечное число элементов из $M$. Это множество $L$ называется *линейной оболочкой* $M$.

Линейная оболочка конечного числа элементов - это все возможные линейные комбинации этого набора.

Рассмотрим $p_1, \dots, p_m \subset L : p_1, \dots, p_m$ - линейно независимый набор. Тогда линейная оболочка $\langle p_1, \dots, p_m \rangle$ - $m$-мерное подпространство $L$.

**Утверждение 5.1.** *Размерность линейной оболочки $k$ векторов не более, чем $k$.*

**Утверждение 5.2.** *Если $L' \subset L$, то $\dim L' \leqslant \dim L$.*

**Утверждение 5.3.** *Если $L' \subset L$ и $\dim L' = \dim L$, то $L' = L$.*

**Теорема 5.1.**
$$x \in L' \Longleftrightarrow \text{все координаты, начиная с } k+1\text{-ой равны } 0$$

*Доказательство.*  
Введем базис $(e_1, \dots, e_n)$.  
Тогда действительно, если $x^{k+1} = \dots = x^n = 0$, то $x = x^1 e_1 + \dots + x^k e_k \Longrightarrow x \in L'$.

Докажем в обратную сторону. Вектор, принадлежащий $L'$ раскладывается в линейную комбинацию $x = x^1 e_1 + \dots + x^k e_k$, которая также является разложением $x$ по базису $e$ при $x^{k+1} = \dots = x^n = 0$.  
$\square$

Любое $k$-мерное подпространство может быть задано однородной системой ранга $n - k$.

## 6 Сумма и пересечение подпространств

**Определение 6.1.** Пусть $L', L'' \subset L$. Тогда *сумма $L' + L''$* — линейная оболочка $(L' \cup L'')$.  
Сумма задается также следующим образом:
$$x = x' + x'' \in L' + L'' : x' \in L', x'' \in L''$$

Сумма линейных пространств является линейным пространством.

Базисом суммы линейных пространств с базисами $(f_1, \dots, f_k)$ и $(h_1, \dots, h_m)$ является максимальный линейно независимый набор, выделенный из $(f_1, \dots, f_k, h_1, \dots, h_m)$.

Аналогичным способом вводится сумма большего числа подпространств.

**Определение 6.2.** Пусть $L', L'' \subset L$. Тогда *пересечение $L' \cap L''$* задается следующим образом:
$$x \in L' \cap L'' : x \in L', x \in L''$$

$L' \cap L''$ является подпространством, т.к. существует нейтральный элемент и $\forall x, y \in L' \cap L'' \hookrightarrow x + y \in L' \cap L''$ и $\forall x \in L' \cap L'' , \forall \alpha \in \mathbb{R}(\mathbb{C}) \hookrightarrow \alpha x \in L' \cap L''$.

Аналогичным способом вводится пересечение большего числа подпространств.

Заметим, что:
$$\dim(L^1 + L^2 + \dots + L^n) \leqslant \dim L^1 + \dim L^2 + \dots + \dim L^n$$

При этом, если достигается равенство, то сумма называется *прямой* и обозначается:
$$L^1 \oplus L^2 \oplus \dots \oplus L^n$$

**Теорема 6.1** (Формула Грассмана). *Предполагаем, что пространства ненулевые.*
$$\dim(L_1 + L_2) = \dim L_1 + \dim L_2 - \dim(L_1 \cap L_2)$$

*Доказательство.*  
Выберем в $L_1 \cap L_2$ базис $e_1, \dots, e_k$, он может состоять из 0 векторов. Его можно дополнить до базиса в $L_1$ векторами $g_1, \dots, g_s$ и до базиса в $L_2$ векторами $f_1, \dots, f_m$. Докажем, что $e, g, f$ — базис в $L_1 + L_2$.
$$\alpha_1 \cdot g_1 + \dots + \alpha_s \cdot g_s + \beta_1 \cdot e_1 + \dots + \beta_k \cdot e_k + \gamma_1 \cdot f_1 + \dots + \gamma_m \cdot f_m = 0$$

Пусть $\gamma_1 \cdot f_1 + \dots + \gamma_m \cdot f_m = x$, тогда $\alpha_1 \cdot g_1 + \dots + \alpha_s \cdot g_s + \beta_1 \cdot e_1 + \dots + \beta_k \cdot e_k = -x$. С одной стороны $x \in L_2$, но с другой $-x \in L_1$, значит и $x \in (L_1 \cap L_2) \Longrightarrow \forall i \ \alpha_i = 0$.

Аналогично обнулим $\gamma_i$. Но т.к. $e$ — базис, только тривиальная комбинация дает 0. Значит, $(g, e, f)$ — упорядоченный линейно независимый набор, т.е. базис в $(L_1 + L_2)$.  
$\square$

**Утверждение 6.1.** *Для того, чтобы сумма двух подпространств была прямой, необходимо и достаточно выполнение любого из следующих условий:*
1. $L_1 \cap L_2 = \{0\}$
2. базис в $L$ — объединение базисов $L_1$ и $L_2$
3. $\forall x \in L \ \exists! y \in L_1, z \in L_2 : x = y + z$
4. $\forall x \in L_1(x \neq 0), y \in L_2(y \neq 0) \hookrightarrow \{x, y\}$ — линейно независим.

Это утверждение обобщается на прямую сумму $k$ подмножеств:
1. Пересечение каждого из подпространств $L_i$ с суммой остальных — нулевое подпространство.
2. Аналогично случаю двух подмножеств.
3. Аналогично случаю двух подмножеств.
4.  Аналогично случаю двух подмножеств.

## 7 Линейные отображения и преобразования

**Определение 7.1.** *Отображение* из $L$ в $L'$ — правило, по которому каждому элементу из $L$ сопоставляется элемент из $L'$.  
Будем записывать $\varphi : L \longrightarrow L'$ или $\varphi(x) = y$, где $x \in L, y \in L'$.

**Определение 7.2.** Отображение называется *преобразованием*, если $L = L'$.

**Определение 7.3.** Отображение $\varphi$ *линейно*, если:
1. $\forall x, y \in L \hookrightarrow \varphi(x + y) = \varphi(x) + \varphi(y)$
2. $\forall \alpha \in \mathbb{R}, \forall x \in L \hookrightarrow \varphi(\alpha x) = \alpha\varphi(x)$

### Свойства линейных отображений

1. $\forall \alpha, \beta \in \mathbb{R}, \forall x, y \in L \hookrightarrow \varphi(\alpha x + \beta y) = \alpha\varphi(x) + \beta\varphi(y)$  
   *Доказательство.* Сразу следует из определения. $\square$
2. Линейно зависимый набор переходит в линейно зависимый набор.  
   *Доказательство.* Это следует из того, что $\varphi(0) = \varphi(0 \cdot x) = 0 \cdot \varphi(x) = 0$. $\square$
3. Линейно независимый набор переходит во что угодно.

Далее будем рассматривать только конечномерные пространства.

**Теорема 7.1.** *Пусть задано линейное отображение $\varphi : L \longrightarrow L', L_1 \subset L$. Тогда $\varphi(L_1) \subset L'$ и $\dim\varphi(L_1) \leqslant \dim L_1$.*

*Доказательство.*  
Если $L_1 = \{0\}$, очевидно. Далее считаем, что $L_1 \neq \{0\}$.  
В $L_1$ можно выбрать базис $f_1, \dots, f_k$. Рассмотрим вектор $x \in L_1$:
$$x = x_1 f_1 + \dots + x_k f_k$$
Тогда $\varphi(x) \in L'$:
$$\varphi(x) = x_1\varphi(f_1) + \dots + x_k\varphi(f_k)$$
$$\Longrightarrow \varphi(L_1) = \langle\varphi(f_1), \dots, \varphi(f_k)\rangle \text{ — подпространство } L' \text{ и } \dim\varphi(L_1) \leqslant k = \dim L_1.$$
$\square$

**Определение 7.4.** Рассмотрим в качестве $L_1$ все $L$. Тогда $\varphi(L) : y \in L' \longleftarrow \exists x : \varphi(x) = y$ — *образ отображения* $\varphi$, обозначается $\operatorname{Im}\varphi$.

**Определение 7.5.** *Ранг отображения* — размерность образа отображения, обозначается $\operatorname{rg}\varphi$.

**Определение 7.6.** Отображение называется *сюръективным*, если:
$$\forall y \in L' \ \exists x \in L : \varphi(x) = y$$

Достаточное условие сюръективности отображения:
$$\operatorname{rg}\varphi = \dim L' = \dim\operatorname{Im}\varphi$$

**Определение 7.7.** Множество векторов, отображающихся в нулевой при отображении $\varphi$, называются *ядром отображения* $\operatorname{Ker}\varphi$.

Ядро является линейным подпространством в $L$.

**Определение 7.8.** Отображение называется *инъективным*, если:
$$\forall x, y \in L(x \neq y) \hookrightarrow \varphi(x) \neq \varphi(y)$$

Достаточное условие инъективности отображения:
$$\operatorname{Ker}\varphi = \{0\}$$

**Определение 7.9.** Отображение *биективно*, если оно одновременно сюръективно и инъективно.

**Утверждение 7.1.** *Пусть линейное отображение инъективно, тогда линейно независимый набор векторов переходит в линейно независимый.*

*Доказательство.*  
Возьмем линейно независимый набор векторов $f_1, \dots, f_k \in L$. Предположим, что $\varphi(f_1), \dots, \varphi(f_k)$ — линейно зависим. Тогда существует нетривиальная линейная комбинация, равная 0, т.е.
$$\begin{cases}
\beta_1\varphi(f_1) + \dots + \beta_k\varphi(f_k) = 0 \\
\beta_1^2 + \dots + \beta_k^2 > 0
\end{cases}$$
Но тогда из линейности $\varphi(\beta_1 f_1 + \dots + \beta_k f_k) = 0$, т.е. $\beta_1 f_1 + \dots + \beta_k f_k = 0 \Longrightarrow \beta_i = 0$, получили противоречие.  
$\square$

Пусть $\varphi : L \longrightarrow L'$, в $L$ задан базис $e_1, \dots, e_n$, в $L'$ — $f_1, \dots, f_m$. Тогда:
$$x \in L \quad x = x_1 e_1 + \dots + x_n e_n \Longrightarrow$$
$$\varphi(x) = \varphi(x_1 e_1 + \dots + x_n e_n) = x_1\varphi(e_1) + \dots + x_n\varphi(e_n) =$$
$$= x_1 \begin{pmatrix} a_{11} \\ \dots \\ a_{m1} \end{pmatrix} + \dots + x_n \begin{pmatrix} a_{1n} \\ \dots \\ a_{mn} \end{pmatrix} = \begin{pmatrix} a_{11} & \dots & a_{1n} \\ \dots & \dots & \dots \\ a_{m1} & \dots & a_{mn} \end{pmatrix} \begin{pmatrix} x_1 \\ \dots \\ x_n \end{pmatrix} = Ax$$
где $A$ — матрица отображения.

**Утверждение 7.2.**
$$\dim\operatorname{Im}\varphi = \operatorname{rg}\varphi = \operatorname{rg} A$$

**Теорема 7.2.** *Размерность отображаемого пространства:*
$$\dim L = \dim\operatorname{Ker}\varphi + \dim\operatorname{Im}\varphi$$

*Доказательство.*  
Пусть $n$ — ранг отображаемого пространства, $\dim\operatorname{Im}\varphi = r$.  
$\operatorname{Ker}\varphi : Ax = 0$, тогда размерность множества решений $n - r \Longrightarrow r + n - r = n = \dim L$.  
$\square$

## 8 Изоморфизм линейных пространств

**Определение 8.1.** *Изоморфизм* линейных пространств — взаимнооднозначное линейное отображение из $L$ в $L'$. $L$ и $L'$ — *изоморфны*, если существует изоморфизм $\varphi : L \longrightarrow L'$.

**Теорема 8.1** (Об изоморфизме линейных пространств).  
*L и L' изоморфны* $\Longleftrightarrow \dim L = \dim L'$.

*Доказательство.*  
$\Longrightarrow$  
Отображение взаимнооднозначно $\Longrightarrow \dim L = \dim L'$.  
$\leftleftarrows$  
Пусть $\dim L = \dim L'$. Существует изоморфизм: пусть $\det A \neq 0$. Тогда $y = Ax$ — подходит, где $A$ - $(\dim L) \times (\dim L)$ матрица.  
$\square$

Альтернативное доказательство: координатный изоморфизм.

> ![Схема замены базиса при линейном отображении. Показаны пространства L и L' с базисами e, e' и f, f' соответственно, и матрицы переходов S и F.]

**Теорема 8.2.** *Пусть $\varphi : L \longrightarrow L'$, $e$ и $f$ — базисы в $L$ и $L'$ соответственно, $y = Ax$. Тогда $A' = F^{-1}AS$.*

*Доказательство.*
$$x = S x'; \quad y = F y'$$
$$\begin{cases}
y' = F^{-1} y = F^{-1}Ax = F^{-1}AS x' \\
y' = A' x'
\end{cases} \Longrightarrow A' = F^{-1}AS$$
$\square$

Если изначальные базисы $e$ и $f$ совпадают, то $L = L', S = F \Longrightarrow A' = S^{-1}AS$.

**Утверждение 8.1.** *Пусть $\varphi : L \longrightarrow L'$, $\operatorname{rg}\varphi = r$. Тогда существуют базисы $e, f$ :*
$$A_{\varphi} = \begin{pmatrix} E_r & | & 0 \\ \hline 0 & | & 0 \end{pmatrix}$$

*Доказательство.*  
$\dim\operatorname{Ker}\varphi = n - r$ — базис в ядре. Пусть он задан векторами $e_{r+1}, \dots, e_n$, тогда $\varphi(e_{r+1}) = \dots = \varphi(e_n) = 0$. Дополним $e_{r+1}, \dots, e_n$ до базиса в $L$ векторами $e_1, \dots, e_r$, тогда $\varphi(e_1), \dots, \varphi(e_r)$ — линейно независимый набор.  
Пусть $\varphi(e_1) = f_1, \varphi(e_2) = f_2, \dots, \varphi(e_r) = f_r$. Тогда $f_1, \dots, f_r$ — дополнение до базиса в $L'$ — независимо от дополнения, корректный базис, требуемый нам.  
$\square$

**Определение 8.2.** Пусть $\varphi : L \longrightarrow L'$ — линейное отображение. Рассмотрим все такие отображения. На этом множестве $M$ можно ввести следующие *операции*:
$$\forall A, B \in M, \forall \alpha \in \mathbb{R}$$
$$(A + B)(x) := A(x) + B(x)$$
$$(\alpha A)(x) := \alpha A(x)$$

**Определение 8.3.** *Композицией* линейных отображений $A$ и $B$ называется последовательное применение этих отображений, обозначается как $BA$.

По теореме о ранге, $\operatorname{rg} BA \leqslant \operatorname{rg} A$ и $\operatorname{rg} BA \leqslant \operatorname{rg} B$.

Ранг отображения не превосходит размерности отображаемого пространства.

## 9 Линейные преобразования ($\varphi : L \longrightarrow L$)

По умолчанию $\dim L = n$, базис в $L$ — $e$; $y = Ax$, $e \longrightarrow e' \Longrightarrow A' = S^{-1}AS$.

**Определение 9.1.** Матрицы $A$ и $B$ называются *перестановочными*, если $AB = BA$.
* $\triangleright$ $AE = EA$
* $\triangleright$ $AA^{-1} = A^{-1}A$
* $\triangleright$ $AA = AA$

**Определение 9.2.** Пусть задана матрица $A$. Тогда ее *степенью* называется $k \in \mathbb{N} : A^k = A \cdot A \cdot \dots \cdot A$ — $k$ раз, $A^0 = E$.

**Определение 9.3.** Пусть $A$ — матрица. Тогда
$$P(A) = \alpha_0 E + \alpha_1 A + \alpha_2 A^2 + \dots + \alpha_k A^k$$
— *многочлен преобразования*.

Пусть $P_1(A)$ и $P_2(A)$ — многочлены преобразования. Тогда $P_1 P_2(A) = P_2 P_1(A)$.

**Определение 9.4.** Подпространство $L_1 \subset L$ называется *инвариантным* относительно $\varphi$, если
$$\forall x \in L_1 \quad \varphi(x) \in L_1, \text{ или, что то же, } \varphi(L_1) \subset L_1$$
* $\triangleright$ $L$ инвариантно относительно любого $\varphi$
* $\triangleright$ $\{0\}$ инвариантно относительно любого $\varphi$

Пусть задано $\varphi$. Тогда:
1. $L_1 = \operatorname{Ker}\varphi$ — инвариантно ($\forall x \in L_1 \hookrightarrow \varphi(x) = 0 \in L_1$)
2. $L_2 = \operatorname{Im}\varphi$ — инвариантно ($\forall x \in L_2 \hookrightarrow \varphi(x) \in L_2$)

Пусть $\varphi : L \longrightarrow L'$ — линейное преобразование. Пусть $e_1, \dots, e_k$ — базис в $L_1$, $e_{k+1}, \dots, e_n$ — дополнение до базиса $L$.
$$x \in L_1 \Longrightarrow x = \alpha_1 e_1 + \dots + \alpha_k e_k$$
$$\varphi(x) = \alpha_1\varphi(e_1) + \dots + \alpha_k\varphi(e_k) = \beta_1 e_1 + \dots + \beta_k e_k$$
$\Longrightarrow$ все координаты, начиная с $k + 1$, равны $0$ и у $x$, и у $\varphi(x)$.

$$A_{\varphi} = \left( \begin{array}{c|c} A_1 & A_2 \\ \hline 0 & A_3 \end{array} \right)$$
где блок $A_1$ имеет размер $k \times k$.

Если матрица имеет вид:
$$\begin{pmatrix} k \times k & | & A_2 \\ \hline 0 & | & A_3 \end{pmatrix}$$
— матрица преобразования $\varphi : L \longrightarrow L'$, то $L_1 = \langle e_1, \dots, e_k \rangle$ — инвариантно относительно $\varphi$.

Аналогично: пусть $e_{n-m+1}, \dots, e_n$ — базис в инвариантном подпространстве, тогда:
$$A_{\varphi} = \left( \begin{array}{c|c} A_1 & 0 \\ \hline A_2 & A_3 \end{array} \right) \begin{array}{l} \} n-m \\ \end{array}$$

Пусть $e_{k+1}, \dots, e_{k+s}$ — базис в инвариантном отношению $\varphi$ подпространстве, тогда:
$$A_{\varphi} = \begin{pmatrix} \dots \\ 0 \\ \dots \\ 0 \\ \dots \end{pmatrix}$$
где нули находятся в позициях над и под блоком размера $s \times s$.

**Определение 9.5.** Матрица преобразования, в которой от нуля отличаются только элементы квадратных клеток порядков $k_1, \dots, k_m$ на диагонали, называется *клеточно-диагональной*.
$$L = L_1 \oplus \dots \oplus L_m \Longleftrightarrow \begin{pmatrix} A_{k_1 \times k_1} & 0 & \dots & 0 \\ 0 & A_{k_2 \times k_2} & \dots & 0 \\ \dots & \dots & \dots & \dots \\ 0 & 0 & \dots & A_{k_m \times k_m} \end{pmatrix}$$

## 10 Задача о собственных векторах

**Определение 10.1.** Преобразование $A$, сопоставляющее каждому вектору из инвариантного подпространства $L'$ вектор из $L'$, называется *ограничением* $A$ на $L'$.

**Теорема 10.1.** *Пусть $A$ и $B$ — перестановочные линейные преобразования. Тогда ядро и образ одного из них инвариантно относительно другого.*

*Доказательство.*  
1. $x \in \operatorname{Ker} A \Longrightarrow A(x) = 0 \Longrightarrow B(A(x)) = 0$. Тогда $A(B(x)) = 0 \Longrightarrow B(x) \in \operatorname{Ker} A \ \forall x \in \operatorname{Ker} A$.
2. Если $x \in \operatorname{Im} A \Longrightarrow \exists \text{ вектор } z : x = A(z)$. Тогда $B(x) = B(A(z)) = A(B(z)) \Longrightarrow B(x) \in \operatorname{Im} A \ \forall x \in \operatorname{Im} A$.  
$\square$

$A$ и $A - \lambda E$ — перестановочные как многочлены от $A$.

$\operatorname{Ker}(A - \lambda E)$ — инвариантно относительно $A$.

**Определение 10.2.** Пусть для некоторого $\lambda$ и для $A$, $\operatorname{Ker}(A - \lambda E)$ — ненулевое. Тогда $\lambda$ — *собственное значение* $A$, $\operatorname{Ker}(A - \lambda E)$ — *собственное подпространство* $A$, соответствующее собственному значению $\lambda$.

Рассмотрим частный случай. Пусть $\operatorname{Ker}(A - \lambda E) \neq \{0\}$. Тогда $\exists x \neq 0 : (A - \lambda E)x = 0 \Longrightarrow Ax = \lambda Ex = \lambda x$. В таком случае если $\lambda = 0$, то $x \in \operatorname{Ker} A$.  
При $\lambda \neq 0 : Ax = \lambda x$ — гомотетия.

**Определение 10.3.** Пусть $x$ — ненулевой вектор такой, что $Ax = \lambda x$. Тогда $x$ называется *собственным вектором*, соответствующим собственному значению $\lambda$.

Пусть $x \neq 0, Ax = \lambda x$. Тогда $A(kx) = kAx = k\lambda x = \lambda(kx)$.

**Теорема 10.2.** *Собственные вектора и только они являются базисами одномерных инвариантных подпространств.*

*Доказательство.*  
1. Пусть вектор $x$ — собственный а $y$ принадлежит одномерному подпространству $L'$ с базисом $x$. Тогда $y = \alpha x$ и $Ay = \alpha Ax = \alpha\lambda x \Longrightarrow Ay \in L'$.
2. Пусть $x$ — базис инвариантного подпространства $L'$. Тогда $Ax$ лежит в $L'$ и раскладывается по базису: $Ax = \lambda x$. Т.к. $x \neq 0$, он собственный.  
$\square$

*Как искать?*
$$\operatorname{Ker}(A - \lambda E) \neq \{0\} \Longleftrightarrow \det(A - \lambda E) = 0 \longrightarrow$$
$$\longrightarrow \text{решаем } (A - \lambda E)x = 0 \longrightarrow \text{получаем собственные вектора.}$$

### 10.1 Характеристическое уравнение

Пусть $\varphi : L \longrightarrow L$, $e$ — базис, $A_{\varphi} = A$.

**Определение 10.4.** $\det(A - \lambda E) = 0$ — *характеристическое уравнение*, а его корни — *характеристические числа*.
$$\begin{pmatrix}
a_{11} - \lambda & a_{12} & a_{13} & \dots & a_{1n} \\
a_{21} & a_{22} - \lambda & a_{23} & \dots & \dots \\
\dots & \dots & \dots & \dots & \dots \\
a_{n1} & \dots & \dots & \dots & a_{nn} - \lambda
\end{pmatrix}$$

$p(\lambda) = (-\lambda)^n + \dots$ — характеристический многочлен $A$. $p(0) = \det A$ — свободный член.

### 10.2 Изменение характеристического уравнения при смене базиса

Пусть $e$ — старый базис, $e'$ — новый, $S$ — матрица перехода.
$$p'(\lambda) = \det(A' - \lambda E) = 0$$
$$\det(A' - \lambda E) = \det(S^{-1}AS - \lambda S^{-1}ES) = \det(S^{-1}(A - \lambda E)S) =$$
$$= \det(S^{-1}) \cdot \det(A - \lambda E) \cdot \det(S) = \det(A - \lambda E)$$
— инвариант, т.е. не зависит от базиса $\Longrightarrow p(\lambda)$ — характеристический многочлен преобразования. Тогда для линейного преобразования $\varphi : L \longrightarrow L$ при переходе от $e$ к $e'$, где $A$ и $A'$ — соответствующие матрицы преобразования в этих базисах:
$$\det A = \det A'$$
$$\operatorname{tr} A = \operatorname{tr} A'$$

Пусть $A$ имеет вид:
$$A = \begin{pmatrix}
\dots \\
0 \\
0 \\
a_{ii} \\
0 \\
0 \\
0 \\
\dots
\end{pmatrix} \leftarrow i\text{-я строка}$$

Тогда $\varphi(e_i) = a_{ii}e_i$.

Если существует базис из собственных векторов, то $A = \operatorname{diag}(\lambda_1, \dots, \lambda_n)$, где $\lambda_1, \dots, \lambda_n$ — собственные значения.

**Теорема 10.3.** *Пусть есть собственные инвариантные подпространства с различными $\lambda$, тогда их сумма — прямая $\Longleftrightarrow$ собственные векторы, отвечающие разным собственным значениям, линейно независимы.*

*Доказательство.*  
Пусть:
* $x_1, \dots, x_k$ — собственные векторы
* $\lambda_1, \dots, \lambda_k$ — отвечающие им собственные значения (по определению они попарно различны)

Введем операторы $B_i = A - \lambda_i E$. Имеем:
$$B_i(x_j) = Ax_j - \lambda_i Ex_j = (\lambda_j - \lambda_i)x_j = \begin{cases} 0, & i = j \\ \neq 0, & i \neq j \end{cases}$$

Пусть векторы линейно зависимы, тогда:
$$\begin{cases}
\exists x_k = \alpha_1 x_1 + \dots + \alpha_{k-1} x_{k-1} \\
B_1 B_2 \dots B_{k-1}(x_k) = (\lambda_k - \lambda_{k-1})\dots(\lambda_k - \lambda_1)x_k, \text{ где } (\lambda_k - \lambda_{k-1})\dots(\lambda_k - \lambda_1) \neq 0 \\
B_1 B_2 \dots B_{k-1}(x_i) = 0, \quad i \in \{1, \dots, k - 1\}
\end{cases} \Longrightarrow$$
$$x_k = 0\text{, что противоречит определению собственного вектора.}$$
$\square$

**Теорема 10.4.** *Пусть $\varphi : L \longrightarrow L$, $p(\lambda)$ — характеристический многочлен. Пусть $p(\lambda) = (\lambda - \lambda_0)^S Q(\lambda)$, $Q(\lambda_0) \neq 0$, $L_1$ — собственное подпространство, соответствующее $\lambda_0$. Тогда:*
$$1 \leqslant \dim L_1 \leqslant S$$

*Доказательство.*  
Пусть $L_1$ — собственное подпространство размерности $k$. Выберем базис: $e_1, \dots, e_k$ — базис в $L_1$. Произвольно дополним его до базиса во всем $L$.
$$A = \begin{pmatrix}
\lambda_0 & \dots & 0 & | & \\
\dots & \lambda_0 & \dots & | & \\
0 & \dots & \lambda_0 & | & A_1 \\
\hline
0 & \dots & 0 & | & \\
\dots & \dots & \dots & | &
\end{pmatrix} \quad \det(A - \lambda E) = (\lambda - \lambda_0)^k R(\lambda) = (\lambda - \lambda_0)^S Q(\lambda)$$

$$\varphi(e_1) = \lambda_0 e_1$$
$$\dots$$
$$\varphi(e_k) = \lambda_0 e_k$$
где $k$ — геометрическая кратность, $S$ — алгебраическая кратность.  
$\square$

*Пример:* $k < S$
$$A = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix} \quad \det(A - \lambda E) = \det \begin{pmatrix} 1 - \lambda & 1 \\ 0 & 1 - \lambda \end{pmatrix} = (1 - \lambda)^2 = 0$$
$$\Longrightarrow \lambda_{1,2} = 1 \text{ — корень кратности 2}$$
$$(A - \lambda E)x = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}x = 0 \Longrightarrow x = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$$

В случае $\lambda \in \mathbb{C}$, если $\lambda$ — корень кратности $t$, то $\bar{\lambda}$ — тоже корень кратности $t$.

**Теорема 10.5.** *Пусть $\lambda, \bar{\lambda}$ — комплексные корни характеристического многочлена вещественного пространства $A$. Тогда им соответствует ненулевое инвариантное подпространство $L_1$ такое, что через любой его ненулевой вектор проходит 2-мерное инвариантное подпространство и $L_1$ не содержит вещественных собственных векторов.*

*Доказательство.*  
Составим многочлен $\lambda, \bar{\lambda} \longrightarrow x^2 + px + q = 0$, где $\lambda, \bar{\lambda}$ — его корни.
$$B = A^2 + pA + qE$$
Докажем, что $L_1 = \operatorname{Ker} B$ — искомое.
$$B = (A - \lambda E)(A - \bar{\lambda} E)$$
$\det B = 0$, т.к. $\det(A - \lambda E) = 0$.

Почему в $L_1$ нет собственных векторов?  
Пусть $h$ — собственный вектор из $L_1 \Longrightarrow Bh = 0 \Longrightarrow (A^2 + pA + qE)h = (\lambda^2 + p\lambda + q)(h) = 0 \Longrightarrow h = 0$ — не является собственным вектором, противоречие.

Пусть $x \in L_1$, $\langle x, A(x) \rangle$ — л.н.з., т.к. иначе $x$ — собственный вектор, но, как было доказано выше, в $L_1$ нет собственных векторов.
$$B(x) = A^2(x) + p(A(x)) + qEx = 0$$
$$A^2(x) = -pA(x) - qEx \Longrightarrow A(\alpha x + \beta(A(x))) = \tilde{\alpha}x + \tilde{\beta} A(x)$$
$\square$

## 11 Диагонализируемость

**Утверждение 11.1.** *У линейного преобразования существует либо одномерное собственное подпространство, либо инвариантное двумерное.*

**Лемма 11.1.** *Если пространство имеет нечетную размерность, то у него существует одномерное собственное подпространство.*

*Доказательство.*  
$p(\lambda)$ — имеет нечетную степень, старший коэффициент равен $-1 \Longrightarrow$ существует вещественный корень, которому соответствует собственный вектор $\Longrightarrow$ ему можно сопоставить собственное подпространство.  
$\square$

Если имеется базис из собственных векторов преобразования $e_1, \dots, e_n$, то в этом базисе:
$$A_{\varphi} = \operatorname{diag}(\lambda_1, \dots, \lambda_n), \quad \varphi(e_i) = \lambda_i e_i$$

**Определение 11.1.** Если существует базис, в котором матрица преобразования диагональная, то оно называется *диагонализируемым*.

**Теорема 11.1** (Критерий диагонализируемости). *Пусть заданы $L, \varphi$.*  
*$\varphi$ — диагонализируемо $\Longleftrightarrow L = L_1 \oplus \dots \oplus L_k$, где $L_i$ — собственные подпространства.*

*Доказательство.*  
$\Longrightarrow$  
$\varphi$ диагонализируемо, значит существует базис, в котором $A_{\varphi} = \operatorname{diag}(\lambda_1, \dots, \lambda_k)$. Упорядочим $\lambda_1 \leqslant \dots \leqslant \lambda_k$, т.е. каждому $\lambda_i$ соответствует собственное подпространство $L_i$, т.к. объединение — прямая сумма.  
$\Longleftarrow$  
В каждом подпространстве выбираем базис, объединение базисов — базис пространства $\Longrightarrow A_{\varphi}$ имеет нужный вид.  
$\square$

**Теорема 11.2** (Необходимое условие диагонализируемости). *Пусть заданы $L, \varphi$. Все корни характеристического многочлена $p(\lambda)$ — вещественные.*

*Доказательство.*  
Пусть задан корень характеристического многочлена $\lambda_i \in \mathbb{R}$, кратности $\alpha_i$. Тогда ему соответствует собственное подпространство такое, что $1 \leqslant \dim L_i \leqslant \alpha_i$.  
Если существует корень характеристического многочлена $\lambda_i \in \mathbb{C}$ (не вещественный) $\Longrightarrow$ сумма кратностей вещественных корней меньше размерности пространства $\Longrightarrow$ не диагонализируемо.  
$\square$

**Теорема 11.3** (Достаточное условие диагонализируемости). *Пусть заданы $L, \varphi$. Все корни характеристического многочлена $p(\lambda)$ — вещественные и различные.*

Собственные векторы, отвечающие различным собственным значениям линейно независимы, значит можно выбрать в качестве базиса собственные векторы диагонализируемого пространства.

*Матричная формулировка:* Пусть у квадратной матрицы $A$ все собственные значения вещественны и различны $\Longrightarrow$
$$\exists S : S^{-1}AS = \operatorname{diag}$$

*Доказательство.*  
Т.к. матрица квадратная, ей можно поставить в соответствие преобразование $\varphi$. Из необходимого условия, все корни характеристического многочлена преобразования вещественные. Если они все различные, то матрицу преобразования можно представить в диагональном виде $\Longrightarrow \varphi$ диагонализируемо.  
$\square$

Критерий диагонализируемости $\Longleftrightarrow$ все $\lambda_1, \dots, \lambda_n$ — вещественные и для каждого $\lambda_i$ кратности $\alpha_i$ размерность собственного подпространства равна $\alpha_i$.

**Теорема 11.4** (Гамильтона-Кэли). *Рассмотрим характеристический многочлен $p(\lambda) = \det(A - \lambda E)$. Тогда:*
$$p(A) = 0$$

*Доказательство.*  
Матрица $A - \lambda E$ почти всегда невырождена. Считаем, что $\lambda$ не является корнем характеристического уравнения $\Longrightarrow$ существует обратная матрица:
$$(A - \lambda E)^{-1} = \frac{1}{\det(A - \lambda E)} B(\lambda) \tag{*}$$
где элементы $B(\lambda)$ — многочлены степени $\leqslant n - 1$:
$$B(\lambda) = B_0 + B_1\lambda + \dots + B_{n-1}\lambda^{n-1}$$

Умножим (*) на $(A - \lambda E)\det(A - \lambda E)$:
$$E p(\lambda) = B(\lambda)(A - \lambda E)$$
$$p(\lambda)E = (\alpha_0 E + \alpha_1 A + \dots + \alpha_n A^n)E = (B_0 + B_1\lambda + \dots + B_{n-1}\lambda^{n-1})(A - \lambda E)$$

Приравняем коэффициенты при одинаковых степенях $\lambda$:
$$\begin{aligned}
\lambda^0 &: \quad \alpha_0 E = B_0 A \\
\lambda^1 &: \quad \alpha_1 E = B_1 A - B_0 E \quad &| \cdot A \\
\lambda^2 &: \quad \alpha_2 E = B_2 A - B_1 E \quad &| \cdot A^2 \\
&\dots \\
\lambda^n &: \quad \alpha_n E = -B_{n-1} E \quad &| \cdot A^n
\end{aligned}$$

Сложим и получим:
$$0 = \alpha_0 E + \alpha_1 A + \alpha_2 A^2 + \dots + \alpha_n A^n = p(A)$$
$\square$

## 12 Линейные функции

**Определение 12.1.** $f$ — *линейная функция*, если:
1. $\forall x, y \in L \hookrightarrow f(x + y) = f(x) + f(y)$
2. $\forall x \in L \ \forall \alpha \in \mathbb{R} \hookrightarrow f(\alpha x) = \alpha f(x)$

**Определение 12.2.** Пусть задан базис $e$. Тогда:
$$f(x) = f(x_1 e_1 + \dots + x_n e_n) = x_1 f(e_1) + \dots + x_n f(e_n) = (f(e_1), \dots, f(e_n))\begin{pmatrix} x_1 \\ \dots \\ x_n \end{pmatrix} = \varphi_e x$$
где $(f(e_1), \dots, f(e_n)) = \varphi_e$ — *компоненты функции*, или *координатная строка*. Функция однозначно задается $\varphi_e$ в базисе $e$.

Для перехода между базисами верно, что:
$$A' = P^{-1}AS \Longrightarrow \varphi'_e = P^{-1}\varphi_e S$$

Для действительных чисел:
$$A' = AS \Longrightarrow \varphi'_e = \varphi_e S$$

Пусть $M$ — множество всех линейных функций, заданных на $L$. Введем сумму линейных функций:
$$\forall f_1, f_2 \in M \ \forall x \in L \hookrightarrow (f_1 + f_2)(x) = f_1(x) + f_2(x)$$
и умножение линейной функции на число:
$$\forall f \in M \ \forall \alpha \in \mathbb{R} \ \forall x \in L \hookrightarrow (\alpha f)(x) = \alpha f(x)$$

Таким образом, $M$ — линейное пространство, причем если в $L$ выбран базис $e$, то любая функция из $M$ задается координатной строкой в этом базисе.

В сумме функций координатные строки складываются. Если умножить функцию на число, координатная строка умножится на это число $\Longrightarrow M$ изоморфно пространству строк длины $n$ со стандартными операциями, т.е. пространство $L$ размерности $n$ переходит в $n$-мерное пространство линейных функций на $L$. Это пространство называют *сопряженным* к $L$ — $L^*$.

Если $L$ имеет базис $e$, а $L^*$ базис $f$, то $f$ взаимен (биортогонален) с $e$, если:
$$f_i(e_j) = \begin{cases} 0, & i \neq j \\ 1, & i = j \end{cases}$$

Существует $L^* \longrightarrow L^{**}$.

**Теорема 12.1.** $L^{**}$ *может быть естественно сопоставлено с $L$.*

**Определение 12.3.** Пусть дано $L$, $B(x, y)$ — функция такая, что $\forall x, y \in L \hookrightarrow B(x, y) \in \mathbb{R}$. $B(x, y)$ называется *билинейной*, если выполнена линейность по двум аргументам, т.е.:
$$\forall x, y, z \in L \hookrightarrow B(x+y, z) = B(x, z) + B(y, z)$$
$$\forall x, y \in L \ \forall \alpha \in \mathbb{R} \hookrightarrow B(\alpha x, y) = \alpha B(x, y)$$
и
$$\forall x, y, z \in L \hookrightarrow B(x, y+z) = B(x, y) + B(x, z)$$
$$\forall x, y \in L \ \forall \alpha \in \mathbb{R} \hookrightarrow B(x, \alpha y) = \alpha B(x, y)$$

Пусть задано пространство $L_n$, $e$ — базис, $x = \begin{pmatrix} x_1 \\ \dots \\ x_n \end{pmatrix}, y = \begin{pmatrix} y_1 \\ \dots \\ y_n \end{pmatrix}$. Тогда:
$$B(x, y) = B(x_1 e_1 + \dots + x_n e_n, y_1 e_1 + \dots + y_n e_n) = \sum_{i,j=1}^n x_i y_j B(e_i, e_j) =$$
$$= \begin{pmatrix} x_1 & \dots & x_n \end{pmatrix} \begin{pmatrix} B(e_1, e_1) & \dots & B(e_1, e_n) \\ \dots & \dots & \dots \\ B(e_n, e_1) & \dots & B(e_n, e_n) \end{pmatrix} \begin{pmatrix} y_1 \\ \dots \\ y_n \end{pmatrix} = x^T B y$$

Билинейная функция $\Longleftrightarrow$ билинейная форма.

## 13 Билинейные и квадратичные формы

Для $B' = S^T B S$ верно, что:
1. $\det B' = (\det S)^2 \det B \Longrightarrow$ знак определителя матрицы формы не меняется при вещественной замене базиса.
2. $B, S$ — невырожденные $\Longrightarrow \operatorname{rg} B' = \operatorname{rg} B$.

**Определение 13.1.** Пусть $b(x, y)$ — симметричная билинейная форма в базисе $e$. Тогда *квадратичной формой* называется:
$$k(x) = b(x, x) = x^T B x,$$
где $B$ — матрица квадратичной формы в базисе $e$.

Для $k(x + y)$ верно, что:
$$k(x + y) = b(x + y, x + y) = b(x, x) + b(x, y) + b(y, x) + b(y, y)$$

При этом существует и единственна симметричная билинейная форма, порождающая данную квадратичную.

**Определение 13.2.** Квадратичная форма $k(x)$ в базисе $e$ имеет *диагональный* вид, если:
$$k(x) = \sum_{i=1}^n \alpha_i x_i^2$$
В этом базисе матрица квадратичной формы диагональна.

**Определение 13.3.** Квадратичная форма $k(x)$ в базисе $e$ имеет *канонический* вид, если:
$$k(x) = \sum_{i=1}^n \alpha_i x_i^2, \quad \alpha_i = \pm 1, 0$$

**Теорема 13.1.** *Для любой квадратичной формы существует базис, в котором она имеет диагональный (и канонический) вид.*

*Доказательство.*  
Пусть имеется квадратичная форма $k(x)$ и ее матрица в некотором базисе $B$. Применим к $B$ последовательность элементарных преобразований. На первом этапе возможны случаи:
1. $b_{11} \neq 0 \Longrightarrow$ вычитаем первую строку, умноженную на $\frac{b_{i1}}{b_{11}}$ для $i$-ой строки, из всех нижележащих строк. Аналогично вычитаем первый столбец. Матрица $B$ переходит в:
$$B_1 = \begin{pmatrix} b'_{11} & 0 & \dots & 0 \\ 0 & & & \\ \dots & & C_1 & \\ 0 & & & \end{pmatrix}$$
где $C_1$ — симметрическая квадратная матрица порядка $n - 1$.
2. $b_{11} = 0$ — особый случай.
   * $\triangleright$ $b_{1i} = 0 \ \forall i = 2, \dots, n \Longrightarrow$ матрица уже имеет вид $B_1$.
   * $\triangleright$ $\exists i : b_{1i} \neq 0$. Приведем матрицу к виду $B_1$ с помощью вспомогательного преобразования: если $b_{ii} \neq 0$, то $i$-ая строка переставляется с первой и $i$-ый столбец переставляется с первым; если $b_{ii} = 0$, то $i$-ая строка прибавляется к первой и $i$-ый столбец прибавляется к первому.

Пусть после $k$ шагов матрица имеет вид:
$$B_k = \begin{pmatrix} b'_{11} & & & | & \\ \dots & \dots & \dots & | & 0 \\ & & b'_{kk} & | & \\ \hline - & - & - & | & - \\ & 0 & & | & C_k \end{pmatrix}$$
где $C_k$ — симметрическая матрица порядка $n - k$. После $(n - 1)$ шага матрица $C_{n-1}$ имеет порядок 1, а $B$ имеет вид:
$$B' = \begin{pmatrix} b'_{11} & \dots & \dots \\ \dots & \dots & \dots \\ \dots & \dots & b'_{nn} \end{pmatrix}$$
Итого $B' = S^T B S$, где $S = S_1 \dots S_n \Longrightarrow$ существует базис, в котором форма имеет канонический вид.  
$\square$

**Определение 13.4.** *Ранг квадратичной формы* — количество ненулевых элементов ($\pm 1$) на диагонали матрицы в каноническом виде, равен рангу матрицы $B$.

**Определение 13.5.** Квадратичная форма $k(x)$ *положительно определена* на $L_1$, если:
$$\forall x \in L_1, \ x \neq 0 \hookrightarrow k(x) > 0$$
Квадратичная форма $k(x)$ *положительно полуопределена* на $L_1$, если:
$$\forall x \in L_1, \ x \neq 0 \hookrightarrow k(x) \geqslant 0$$
Аналогично для *отрицательной* определенности и полуопределенности.

Если форма положительно/отрицательно полуопределена/определена, то она называется *знакоопределенной*, иначе знаконеопределенной.

| Знакоопределенность | Канонический вид |
| :--- | :--- |
| $k(x) > 0$ | $\operatorname{diag}(1, \dots, 1)$ |
| $k(x) \geqslant 0$ | $\operatorname{diag}(1 \text{ и } 0)$ |
| $k(x) < 0$ | $\operatorname{diag}(-1, \dots, -1)$ |
| $k(x) \leqslant 0$ | $\operatorname{diag}(-1 \text{ и } 0)$ |
| знаконеопределенность | $\operatorname{diag}(1 \text{ и } -1)$ |

**Теорема 13.2** (Закон инерции квадратичных форм). *В любом базисе, в котором форма имеет канонический вид, количество "+1" и "-1" на диагонали постоянно.*

**Определение 13.6.** Пусть $L^{(-)} \subset L$ — пространство максимальной размерности, где $k(x) < 0$. Эту размерность назовем *отрицательным индексом инерции*. Аналогично определяется *положительный индекс инерции*.

**Лемма 13.1.** *Отрицательный индекс инерции равен количеству "-1" в каноническом виде, аналогично положительный индекс равен количеству "+1".*

*Доказательство.*  
Пусть $r = \operatorname{rg} k(x)$.  
Пусть $k(x) = -x_1^2 - x_2^2 - \dots - x_k^2 + x_{k+1}^2 + \dots + x_r^2 + 0 \cdot x_{r+1}^2 + \dots + 0 \cdot x_n^2$.  
Определим подпространства:
$$L_1 = \langle e_1, \dots, e_k \rangle \quad L_2 = \langle e_{k+1}, \dots, e_n \rangle \quad L_1 \oplus L_2 = L$$
$k(x)$ — отрицательно определена на $L_1 \Longrightarrow \dim L^{(-)} \geqslant k$.  
$k(x)$ — положительно полуопределена на $L_2$.

Пусть $L^{(-)}$ таково, что $\dim L^{(-)} > k$. Тогда для $L^{(-)} \cap L_2$ верно:
1. $\dim L^{(-)} + \dim L_2 > n$.
2. Отсюда пересечение нетривиально, т.е. $\exists \tilde{x} \neq 0 \in L^{(-)} \cap L_2 \longrightarrow \begin{cases} k(\tilde{x}) \geqslant 0 & \text{(т.к. } \tilde{x} \in L_2) \\ k(\tilde{x}) < 0 & \text{(т.к. } \tilde{x} \in L^{(-)}) \end{cases}$ — противоречие.

Пусть $p$ — положительный индекс, $q$ — отрицательный индекс $\Longrightarrow \operatorname{rg} = p + q$.  
$\square$

**Определение 13.7.** *Сигнатура* — разность положительного и отрицательного индексов инерции.

**Лемма 13.2.** *Если $k(x) > 0 \Longrightarrow$ в любом базисе диагональные элементы матрицы $b_{ii} > 0$.*
$$b_{ii} = b(e_i, e_i) = k(e_i) > 0$$

**Замечание.** Обратное неверно, контрпример:
$$\begin{pmatrix} 1 & 2 \\ 2 & 1 \end{pmatrix} \text{ — не является положительно определенной, хотя } b_{11}, b_{22} > 0$$

**Теорема 13.3** (Критерий Сильвестра).
1. *Квадратичная форма $k(x)$ положительно определена ($k(x) > 0$) тогда и только тогда, когда в любом базисе все главные угловые миноры матрицы больше 0.*
2. *Квадратичная форма $k(x)$ отрицательно определена ($k(x) < 0$) тогда и только тогда, когда главные угловые миноры матрицы меняют знак: нечетные — меньше 0, четные — больше 0.*

*Доказательство.*  
1. $\Longrightarrow$  
Пусть $k(x) > 0$. Начнем приведение матрицы к диагональному виду:
$$\begin{pmatrix} b_{11} & \dots \\ \dots & \dots \end{pmatrix} \quad \text{по лемме нет особого случая } \Longrightarrow$$
вычитаем только верхние строки из нижних (и аналогичные столбцы), переходя к:
$$\begin{pmatrix} b_{11} & \dots & 0 \\ \dots & b'_{22} & \dots \\ \dots & \dots & \dots \end{pmatrix}, \text{ где } b'_{ii} > 0$$
В каждом главном миноре выполняем только такие элементарные операции, которые не меняют его определитель. Все $\Delta_i > 0$ и они не менялись.

$\Longleftarrow$  
Пусть все главные угловые миноры больше 0. Приведем матрицу к диагональному виду:
$$\begin{pmatrix} b_{11} & 0 & \dots & 0 \\ 0 & b'_{22} & \dots & 0 \\ \dots & \dots & \dots & \dots \\ 0 & \dots & \dots & b'_{nn} \end{pmatrix}$$
$$b_{11} = \Delta_1 > 0$$
$$\Delta_2 = b'_{22}b_{11} > 0 \Longrightarrow b'_{22} > 0$$
$$\dots \text{ аналогично остальные } b'_{ii} > 0.$$

2. $k(x)$ отрицательно определена $\Longleftrightarrow (-1)k(x)$ — положительно определена.
$$\begin{pmatrix} -1 & 0 & \dots & 0 \\ 0 & -1 & \dots & 0 \\ \dots & \dots & \dots & \dots \\ 0 & \dots & 0 & -1 \end{pmatrix} \quad \det(-E_k) = (-1)^k$$
$\square$

**Утверждение 13.1** (Достаточное условие знаконеопределенности $k(x)$). *Если второй угловой минор $\Delta_2 < 0$, то $k(x)$ — знаконеопределена.*

*Доказательство.*  
Пусть задана матрица $B$ квадратичной формы $k(x)$, $\Delta_2 < 0$. Рассмотрим ограничение формы на подпространство $L_1 = \langle e_1, e_2 \rangle$.  
На $L_1$ у $k(x)$ матрица $\tilde{B}$ имеет определитель $|\tilde{B}| = \Delta_2 < 0$.  
Перейдем в канонический вид:
$$\begin{cases} \tilde{B}' = S^T \tilde{B} S \\ |\tilde{B}'| < 0 \end{cases} \quad \det\tilde{B}' = (\det S)^2 \det\tilde{B},$$
где $(\det S)^2 > 0$ и $\det\tilde{B} = \Delta_2 < 0$.  
Получаем:
$$\det\tilde{B}' = \tilde{b}'_{11}\tilde{b}'_{22} < 0$$
Следовательно, в каноническом виде на диагонали содержатся коэффициенты разных знаков (и 1, и -1).  
$\square$

## 14 Евклидовы пространства

**Определение 14.1.** *Евклидово пространство* $\mathcal{E}$ — линейное пространство, на котором введено скалярное произведение:
$$\forall x, y \in \mathcal{E} \hookrightarrow (x, y) \in \mathbb{R}$$
удовлетворяющее аксиомам:
1. $\forall x, y \in \mathcal{E} \hookrightarrow (x, y) = (y, x)$
2. $\forall x, y, z \in \mathcal{E} \hookrightarrow (x + y, z) = (x, z) + (y, z)$
3. $\forall x, y \in \mathcal{E} \ \forall \alpha \in \mathbb{R} \hookrightarrow (\alpha x, y) = \alpha(x, y)$
4. $\forall x \neq 0, \ x \in \mathcal{E} \hookrightarrow (x, x) > 0$

Свойства 1-3 скалярного произведения порождают симметричную билинейную форму, а аксиома 4 гарантирует, что порождаемая ею квадратичная форма положительно определена.

**Определение 14.2.** *Длина вектора* $x \in \mathcal{E}$:
|x| = \sqrt{(x, x)}

**Определение 14.3.** *Угол* между ненулевыми векторами $x, y \in \mathcal{E}$:
$$\cos(x^{\wedge}y) = \frac{(x, y)}{|x||y|}$$

**Теорема 14.1** (Неравенство Коши-Буняковского (КБШ)). *Пусть задано евклидово пространство $\mathcal{E}$, тогда $\forall x, y \in \mathcal{E}$ справедливо неравенство:*
$$(x, y)^2 \leqslant (x, x)(y, y)$$

**Теорема 14.2** (Неравенство треугольника). *Пусть задано евклидово пространство $\mathcal{E}$, тогда $\forall x, y \in \mathcal{E}$ справедливо неравенство:*
$$|x + y| \leqslant |x| + |y|$$

*Доказательство.*  
$$(x+y, x+y) = |x|^2 + 2(x, y) + |y|^2 \leqslant \{ \text{т.к. из КБШ } |(x, y)| \leqslant |x||y| \} \leqslant |x|^2 + 2|x||y| + |y|^2 = (|x| + |y|)^2$$
$$\Longrightarrow |x + y| \leqslant |x| + |y|$$
$\square$

## 15 Ортогонализация

**Определение 15.1.** Будем говорить, что векторы $x$ и $y$ *ортогональны*, если $(x, y) = 0$.

**Следствие.** Если вектор $x$ ортогонален к любому вектору пространства, то $x = 0$.

**Определение 15.2.** Пусть дано $\mathcal{E}$ — евклидово пространство, в нем задан базис $e_1, \dots, e_n$. Тогда $(x, y) = x^T \Gamma y$, где:
$$\Gamma = \begin{pmatrix} (e_1, e_1) & \dots & (e_1, e_n) \\ \dots & \dots & \dots \\ (e_n, e_1) & \dots & (e_n, e_n) \end{pmatrix} \text{ — } \textit{матрица Грама}$$

**Теорема 15.1.** *Пусть задан некоторый набор векторов $f_1, \dots, f_n$. Тогда $\det \Gamma_f \geqslant 0$, причем:*
1. $\det \Gamma_f > 0$, если набор линейно независим.
2. $\det \Gamma_f = 0$, если набор линейно зависим.

*Доказательство.*  
Пусть $f_1, \dots, f_k$ — линейно зависимы $\Longrightarrow$ существует нетривиальная линейная комбинация, равная нулю:
$$\alpha_1 f_1 + \dots + \alpha_k f_k = 0, \quad \alpha_1^2 + \dots + \alpha_k^2 > 0$$
Домножим это равенство скалярно поочередно на $f_1, \dots, f_k$:
$$\begin{cases}
\alpha_1(f_1, f_1) + \alpha_2(f_1, f_2) + \dots + \alpha_k(f_1, f_k) = 0 \\
\alpha_1(f_2, f_1) + \alpha_2(f_2, f_2) + \dots + \alpha_k(f_2, f_k) = 0 \\
\dots \\
\alpha_1(f_k, f_1) + \dots + \alpha_k(f_k, f_k) = 0
\end{cases}$$
Полученная система линейно однородна относительно $\alpha_i$ и имеет нетривиальное решение $\Longrightarrow \det \Gamma_f = 0$.

Докажем неравенство КБШ через определитель Грама двух векторов:
$$\det \begin{pmatrix} (x, x) & (x, y) \\ (y, x) & (y, y) \end{pmatrix} \geqslant 0 \Longrightarrow (x, x)(y, y) - (x, y)^2 \geqslant 0$$
Отсюда получаем искомое, причем равенство достигается $\Longleftrightarrow x$ и $y$ — линейно зависимые.  
$\square$

При переходе к другому базису матрица Грама преобразуется как $\Gamma' = S^T \Gamma S$.  
В ОНБ $\Gamma = E$.

Если базис $f_1, \dots, f_n$ — ортогональный, то разложение вектора $x$ имеет вид:
$$x = \sum_{i=1}^n \frac{(x, f_i)}{|f_i|^2} \cdot f_i$$

**Теорема 15.2** (Равенство Парсеваля).
$$(x, x) = |x|^2 = \sum_{i=1}^n \frac{(x, f_i)^2}{|f_i|^2}$$

**Определение 15.3.** Матрица $S$ — *ортогональная*, если $S^T S = E$. Для такой матрицы верно, что:
1. $S^T = S^{-1} \Longrightarrow S S^T = E$
2. $(\det S)^2 = 1 \Longrightarrow \det S = \pm 1$

**Определение 15.4.** Пусть $E_1 \subset E$ — подпространство евклидова пространства $E$. Тогда $E_1^{\perp}$ — *ортогональное дополнение* к $E_1$, определяемое как:
$$E_1^{\perp} = \{ x \in E \mid \forall y \in E_1 : (x, y) = 0 \}$$

$E_1^{\perp}$ является подпространством в $E$.  
Действительно, если $\tilde{x}, \tilde{\tilde{x}} \in E_1^{\perp}$, то $(\alpha\tilde{x} + \beta\tilde{\tilde{x}}, y) = \alpha(\tilde{x}, y) + \beta(\tilde{\tilde{x}}, y) = 0$, т.к. $(\tilde{x}, y) = 0$ и $(\tilde{\tilde{x}}, y) = 0$. Также $0 \in E_1^{\perp}$.

Кроме того, если $\dim E = n, \dim E_1 = k$, то $\dim E_1^{\perp} = n - k$.

**Определение 15.5.** Пусть $E_1, E_2$ — подпространства $E$. Они называются *ортогональными*, если:
$$\forall x \in E_1, \ \forall y \in E_2 \hookrightarrow (x, y) = 0$$

Пусть $E_1 \subset E$, и в $E_1$ выбран ортогональный базис $f_1, \dots, f_k$. Найдем ортогональную проекцию произвольного вектора $x$ на подпространство $E_1$. Дополним базис $E_1$ до ортогонального базиса всего пространства $E$, добавив ортогональный базис $E_1^{\perp}$ из векторов $f_{k+1}, \dots, f_n$. Тогда:
$$x = \alpha_1 f_1 + \dots + \alpha_k f_k + \dots + \alpha_n f_n,$$
где вектор $x_{E_1} = \alpha_1 f_1 + \dots + \alpha_k f_k$ — проекция на $E_1$. Отсюда $\alpha_i = \frac{(x, f_i)}{(f_i, f_i)}$, и окончательно:
$$x_{E_1} = \sum_{i=1}^k \frac{(x, f_i)}{(f_i, f_i)} \cdot f_i$$

### Процесс ортогонализации Грама-Шмидта

Пусть задан базис $f_1, \dots, f_n$ в $E_1$.  
Положим $g_1 = f_1$. Далее из $f_2$ вычтем его ортогональную проекцию на линейную оболочку $g_1$:
$$g_2 = f_2 - \frac{(f_2, g_1)}{|g_1|^2} g_1$$
Отметим, что $g_2$ выражается через $f_1$ и $f_2$, причем $g_2 \neq 0$, так как иначе $f_1$ и $f_2$ были бы линейно зависимы.

Продолжим по индукции. Пусть построены $k$ попарно ортогональных векторов $g_1, \dots, g_k$, тогда следующий вектор строится по правилу:
$$g_{k+1} = f_{k+1} - \sum_{i=1}^k \frac{(f_{k+1}, g_i)}{|g_i|^2} g_i$$

Таким образом, мы построим систему из $n$ попарно ортогональных ненулевых векторов, каждый из которых представляет собой разность исходного вектора и его проекции на предыдущие. Поскольку все $g_i \neq 0$, полученная система векторов образует ортогональный базис.

## 16 Сопряженные преобразования

**Определение 16.1.** Преобразование $A^*$ называется *сопряженным* к преобразованию $A$, если:
$$\forall x, y \in E \quad (A(x), y) = (x, A^*(y))$$

**Теорема 16.1.** *Для любого линейного преобразования $A$ в евклидовом пространстве существует и единственно сопряженное преобразование $A^*$.*

*Доказательство.*  
В матричном представлении:
$$x^T A^T \Gamma y = (Ax)^T \Gamma y = x^T \Gamma A^* y \Longrightarrow A^T \Gamma = \Gamma A^*$$
Отсюда матрица преобразования $A^*$ имеет вид $A^* = \Gamma^{-1} A^T \Gamma$, что доказывает существование и единственность.  
$\square$

**Теорема 16.2.**
$$\operatorname{Im} A = (\operatorname{Ker} A^*)^{\perp}$$

*Доказательство.*  
Для любого $x \in E$ и $y \in \operatorname{Ker} A^*$:
$$(A(x), y) = (x, A^*(y)) = (x, 0) = 0 \Longrightarrow \operatorname{Im} A \subset (\operatorname{Ker} A^*)^{\perp}$$
Обозначим $\dim E = n, \operatorname{rg} A = r$.  
Тогда $\dim(\operatorname{Ker} A^*) = n - r$ (размерность пространства решений однородной системы $A^T y = 0$).  
Отсюда:
$$\dim(\operatorname{Ker} A^*)^{\perp} = n - (n - r) = r = \dim\operatorname{Im} A$$
Следовательно, подпространства совпадают.  
$\square$

## 17 Самосопряженные преобразования

**Определение 17.1.** Преобразование $A$ называется *самосопряженным*, если $A = A^*$, т.е.:
$$(A(x), y) = (x, A(y))$$

Из определения следует, что преобразование самосопряжено тогда и только тогда, когда его матрица в любом ортонормированном базисе (ОНБ) симметрична.

**Утверждение 17.1.** *Все корни характеристического многочлена самосопряженного преобразования вещественны.*

*Доказательство.*  
Докажем от противного. Пусть у матрицы $A$ существует комплексный корень характеристического многочлена $\lambda \in \mathbb{C} \setminus \mathbb{R} \Longrightarrow$ на вещественном пространстве существует инвариантное двумерное подпространство $E_1$, не содержащее вещественных собственных векторов. Рассмотрим ограничение оператора $A$ на $E_1$:
$$x \in E_1 \longrightarrow A(x) \in E_1$$
Выберем в $E_1$ ОНБ, тогда матрица ограничения примет вид:
$$\tilde{A} = \begin{pmatrix} \alpha & \beta \\ \beta & \gamma \end{pmatrix}$$
Ее характеристический многочлен:
$$\det(\tilde{A} - \lambda E) = (\alpha - \lambda)(\gamma - \lambda) - \beta^2 = \lambda^2 - (\alpha + \gamma)\lambda + \alpha\gamma - \beta^2 = 0$$
Дискриминант этого уравнения:
$$D = (\alpha + \gamma)^2 - 4(\alpha\gamma - \beta^2) = (\alpha - \gamma)^2 + 4\beta^2 \geq 0$$
Корни обязаны быть вещественными, что противоречит предположению о чисто комплексных сопряженных корнях без вещественного представителя.  
$\square$

**Следствие.** Все собственные значения симметрической вещественной матрицы вещественны.

**Теорема 17.1.** *Собственные векторы самосопряженного преобразования, отвечающие различным собственным значениям, попарно ортогональны.*

*Доказательство.*  
Пусть $A$ — самосопряженное преобразование, $h_1, h_2$ — собственные векторы, отвечающие различным собственным значениям $\lambda_1 \neq \lambda_2$ соответственно.  
С одной стороны:
$$(Ah_1, h_2) = (\lambda_1 h_1, h_2) = \lambda_1(h_1, h_2)$$
С другой стороны:
$$(h_1, Ah_2) = (h_1, \lambda_2 h_2) = \lambda_2(h_1, h_2)$$
Поскольку $(Ah_1, h_2) = (h_1, Ah_2)$, получаем:
$$(\lambda_1 - \lambda_2)(h_1, h_2) = 0 \Longrightarrow (h_1, h_2) = 0$$
$\square$

**Теорема 17.2.** *Пусть задано самосопряженное преобразование $A$ пространства $E$, и подпространство $E_1 \subset E$ инвариантно относительно $A$. Тогда его ортогональное дополнение $E_1^{\perp}$ также инвариантно относительно $A$.*

*Доказательство.*  
По условию для любого $x \in E_1 \hookrightarrow A(x) \in E_1$.  
Возьмем произвольный $y \in E_1^{\perp}$. Тогда для любого $x \in E_1$:
$$(A(y), x) = (y, A(x)) = 0 \Longrightarrow A(y) \in E_1^{\perp}$$
$\square$

**Теорема 17.3.** *Для любого самосопряженного преобразования $A$ пространства $E$ существует ортонормированный базис из собственных векторов этого преобразования.*

*Доказательство.*  
Пусть $L = L_1 \oplus \dots \oplus L_k$ — прямая сумма всех собственных подпространств преобразования. Покажем, что $E = L$.  
Предположим противное: $L \neq E \Longrightarrow \dim L < \dim E$.  
Поскольку подпространство $L$ инвариантно относительно $A$, то по теореме 17.2 подпространство $L^{\perp}$ также инвариантно относительно $A$, причем $\dim L^{\perp} \geqslant 1$.  
Рассмотрим ограничение $A$ на $L^{\perp}$. Оно также является самосопряженным, следовательно, имеет по крайней мере один вещественный корень характеристического многочлена и соответствующий ему собственный вектор в $L^{\perp}$. Но этот вектор должен лежать в $L$, что невозможно, поскольку $L \cap L^{\perp} = \{0\}$. Противоречие.  
$\square$

**Теорема 17.4** (Обратная теорема). *Если в евклидовом пространстве $E$ существует ортонормированный базис из собственных векторов преобразования $A$, то $A$ — самосопряженное преобразование.*

*Доказательство.*  
В этом базисе матрица преобразования диагональна, а следовательно, симметрична.  
$\square$

**Определение 17.2.** Преобразование $A$ пространства $E$ называется *ортогональным*, если:
$$\forall x, y \in E \hookrightarrow (A(x), A(y)) = (x, y)$$

**Теорема 17.5.** *Преобразование $A$ ортогонально тогда и только тогда, когда $A^* = A^{-1}$.*

*Доказательство.*  
$$(x, A^*(A(y))) = (A(x), A(y)) = (x, y) \Longrightarrow (x, (A^*A - E)y) = 0 \Longrightarrow A^*A = E \Longrightarrow A^{-1} = A^*$$
$\square$

В ОНБ это свойство равносильно условию $A^T = A^{-1}$.

**Теорема 17.6.** *Если $e$ и $f$ — два произвольных ОНБ пространства, то существует единственное ортогональное преобразование $S$ такое, что $S(e_i) = f_i \ \forall i$.*

**Теорема 17.7.** *Пусть $A$ — ортогональное преобразование пространства $E$, и подпространство $E_1 \subset E$ инвариантно относительно $A$. Тогда $E_1^{\perp}$ также инвариантно относительно $A$.*

*Доказательство.*  
Так как $A$ — ортогонально и сохраняет размерность, из инвариантности $E_1$ следует $A(E_1) = E_1$.  
Пусть $x \in E_1, y \in E_1^{\perp}$. Тогда:
$$0 = (x, y) = (A(x), A(y))$$
Поскольку $A(E_1) = E_1$, любой вектор из $E_1$ представим в виде $A(x) \Longrightarrow A(y) \in E_1^{\perp}$.  
$\square$

**Теорема 17.8** (О полярном разложении). *Любое невырожденное преобразование $A$ может быть представлено в виде полярного разложения:*
$$A = QS,$$
где $Q$ — ортогональное преобразование, $S$ — самосопряженное преобразование с положительными собственными значениями.

*Доказательство.*  
Оператор $A^*A$ всегда является самосопряженным, поскольку $(A^*A)^* = A^*(A^*)^* = A^*A$.  
Пусть $h$ — собственный вектор оператора $A^*A$, отвечающий собственному значению $\lambda$:
$$(A^*Ah, h) = \lambda(h, h) \Longrightarrow (Ah, Ah) = \lambda(h, h)$$
Поскольку $(Ah, Ah) \geqslant 0$ и $(h, h) > 0$, получаем $\lambda \geqslant 0$.

Пусть $e_1, \dots, e_n$ — ОНБ из собственных векторов оператора $A^*A$. Упорядочим их по убыванию собственных значений: $\lambda_1 \geqslant \dots \geqslant \lambda_n \geqslant 0$.  
Рассмотрим скалярные произведения образов:
$$(A(e_i), A(e_j)) = (A^*A(e_i), e_j) = \lambda_i(e_i, e_j) = \begin{cases} 0, & i \neq j \\ \lambda_i, & i = j \end{cases}$$
Векторы $A(e_i)$ попарно ортогональны, а их длины $|A(e_i)| = \sqrt{\lambda_i}$ называются *сингулярными числами* оператора $A$. Обозначим $\alpha_i = \sqrt{\lambda_i} \geqslant 0$.  

Для всех $\lambda_i \neq 0$ положим $f_i = \frac{A(e_i)}{\alpha_i}$. Полученную систему векторов дополним до ОНБ $f_1, \dots, f_n$ всего пространства.  
Зададим ортогональный оператор $Q$ условием $Q(e_i) = f_i$.  
Тогда оператор $S = Q^{-1}A$ в базисе из векторов $e_i$ является диагональным с положительными числами $\alpha_i$ на диагонали:
$$S(e_i) = Q^{-1}A(e_i) = Q^{-1}(\alpha_i f_i) = \alpha_i e_i$$
Это означает, что $S$ — самосопряженный оператор с положительными собственными значениями.  
$\square$

Из диагонализуемости $S$ следует представление через матрицу перехода:
$$S = PDP^{-1}, \quad D = P^{-1}SP$$
Отсюда получаем сингулярное разложение оператора (матрицы):
$$A = Q P D P^{-1} = Q_1 D Q_2$$

**Определение 17.3.** Линейное преобразование $A$ называется *присоединенным* к билинейной форме $b(x, y)$, если:
$$\forall x, y \in E \hookrightarrow b(x, y) = (x, Ay)$$

*Доказательство корректности формулировки:*  
В матричной форме равенство записывается как:
$$x^T B y = x^T \Gamma A y \Longrightarrow B = \Gamma A \Longrightarrow A = \Gamma^{-1} B$$
Поскольку решение единственно, присоединенное преобразование всегда существует и единственно.  
$\square$

**Замечание.** В ортонормированном базисе матрицы $A$ и $B$ совпадают.

**Теорема 17.9.** *Билинейная форма $b(x, y)$ симметрична тогда и только тогда, когда присоединенное к ней преобразование самосопряжено.*

**Следствие.** Если билинейная форма $b(x, y)$ симметрична, то существует ОНБ из собственных векторов присоединенного преобразования. В этом базисе матрица формы имеет диагональный вид.

*Вопрос:*  
Пусть заданы две квадратичные формы $k_1(x), k_2(x)$. Существует ли базис, в котором обе формы одновременно принимают диагональный вид?

*Ответ:*  
Не всегда. Например, формы $k_1(x) = x_1^2 - x_2^2$ и $k_2(x) = x_1 x_2$ одновременно диагонализировать нельзя.

**Теорема 17.10.** *Пусть заданы квадратичные формы $k(x)$ и $h(x)$, причем $h(x)$ — положительно определенная форма. Тогда существует базис, в котором обе формы одновременно имеют диагональный вид.*

## 18 Объем n-мерного параллелепипеда

Чтобы задать $n$-мерный параллелепипед, натянутый на векторы $f_1, \dots, f_n$, рассмотрим множество их линейных комбинаций с коэффициентами $\alpha_i \in [0, 1]$:
$$\{\alpha_1 f_1 + \dots + \alpha_n f_n \mid \alpha_i \in [0, 1]\}$$

### Индукционное задание объема

Будем определять объем $(n + 1)$-мерного параллелепипеда через объем $n$-мерного параллелепипеда и длину перпендикуляра (высоты), опущенного на него:
$$V_{f_1, \dots, f_k} = V_{f_1, \dots, f_{k-1}} \cdot |h_k|,$$
где $h_k$ — ортогональная проекция вектора $f_k$ на ортогональное дополнение к подпространству $\langle f_1, \dots, f_{k-1} \rangle$.  
База индукции:
$$V_{f_1} = |f_1|$$

Применяя к системе $f_1, \dots, f_k$ процесс ортогонализации Грама-Шмидта, получаем попарно ортогональные векторы $h_1, \dots, h_k$.  
Тогда объем выражается как произведение длин этих векторов:
$$V_{f_1, \dots, f_k} = |h_1| \cdot |h_2| \cdot \dots \cdot |h_k| = \sqrt{\det \Gamma_h},$$
где $\Gamma_h = \operatorname{diag}(|h_1|^2, \dots, |h_k|^2)$ — матрица Грама ортогональной системы векторов.

Поскольку $\Gamma_h = S^T \Gamma_f S$, где $S$ — треугольная матрица перехода с единицами на диагонали ($\det S = 1$), то:
$$V_{f_1, \dots, f_k} = \sqrt{\det \Gamma_f}$$

Заметим также, что если $F$ — матрица, столбцами которой являются координаты векторов $f_1, \dots, f_k$ в некотором ОНБ $e$, то:
$$\Gamma_f = F^T \Gamma_e F \Longrightarrow \det \Gamma_f = (\det F)^2 \cdot \det \Gamma_e$$
Отсюда:
$$V_{f_1, \dots, f_k} = |\det F| \cdot V_{e_1, \dots, e_k}$$

Если исходный базис $e$ является ортонормированным ($V_{e_1, \dots, e_k} = 1$), то:
$$V_{f_1, \dots, f_k} = |\det F|$$

## 19 Унитарные пространства

Определение евклидова пространства со стандартными аксиомами скалярного произведения:
1. $(x, y) = (y, x)$
2. $(\alpha x, y) = \alpha(x, y)$
3. $(x + y, z) = (x, z) + (y, z)$
4. $(x, x) > 0 \quad \forall x \neq 0$

не может быть напрямую перенесено на случай комплексных чисел.  
Контрпример: для вектора $ix$ получили бы $(ix, ix) = i^2 (x, x) = -(x, x) < 0$, что противоречит аксиоме положительной определенности (4).

**Определение 19.1.** Скалярное произведение на комплексном линейном пространстве называется *унитарным*, если выполнены аксиомы:
1. $(x, y) = \overline{(y, x)}$
2. $(\alpha x, y) = \bar{\alpha}(x, y)$
3. $(x + y, z) = (x, z) + (y, z)$
4. $(x, x) \in \mathbb{R}$ и $(x, x) > 0 \quad \forall x \neq 0$

### Сравнительная таблица вещественных и комплексных пространств

| Вещественное евклидово пространство | Комплексное унитарное пространство |
| :--- | :--- |
| Ортогональное преобр.: $(Ax, Ay) = (x, y)$ | Унитарное преобр.: $(Ax, Ay) = (x, y)$ |
| Ортогональная матрица: $A^T A = E$ | Унитарная матрица: $A^T \bar{A} = E$ |
| Сопряженное преобр.: $(Ax, y) = (x, A^*y)$ | Сопряженное преобр.: $(Ax, y) = (x, A^*y)$ |
| Сопряженная матрица: $A^* = \Gamma^{-1}A^T\Gamma$ | Сопряженная матрица: $A^* = \Gamma^{-1}\bar{A}^T\Gamma$ |
| Самосопряженное преобр.: $(Ax, y) = (x, Ay)$ | Эрмитово преобр.: $(Ax, y) = (x, Ay)$ |
| Самосопряженная матрица: $A = A^T$ | Эрмитова матрица: $A^T = \bar{A}$ |

Для ортогонального оператора $A$ в вещественном пространстве собственные значения $\lambda \in \mathbb{R}$ удовлетворяют условию:
$$\lambda^2(h, h) = (Ah, Ah) = (h, h) \Longrightarrow |\lambda| = 1$$
Если собственные значения комплексные $\lambda \in \mathbb{C}$, то соответствующая пара сопряженных корней $\lambda, \bar{\lambda}$ удовлетворяет условию:
$$|\lambda \cdot \bar{\lambda}| = 1 \Longrightarrow |\lambda| = 1$$

### Линии второго порядка

Общее уравнение вещественной линии второго порядка на плоскости:
$$\begin{cases}
Ax^2 + 2Bxy + Cy^2 + 2Dx + 2Ey + F = 0 \\
A^2 + B^2 + C^2 > 0
\end{cases}$$

При замене декартовой системы координат старшая часть $Ax^2 + 2Bxy + Cy^2$ преобразуется как квадратичная форма, поэтому ее ранг и сигнатура являются инвариантами.  
Для исследования линии можно рассмотреть соответствующую квадратичную форму в трехмерном пространстве:
$$Ax^2 + 2Bxy + Cy^2 + 2Dxz + 2Eyz + Fz^2 = 0$$
Сечение этого конуса плоскостью $z = 1$ определяет исходную линию второго порядка.
