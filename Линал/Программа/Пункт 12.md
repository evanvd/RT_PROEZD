# Пункт 12. Билинейные и квадратичные функции. Координатное представление билинейных и квадратичных форм. Изменение матриц при изменении базиса

### Билинейные и квадратичные функции

> [!info] **Определение.** (Билинейная форма)
> **Билинейной функцией** (или билинейной формой) на линейном пространстве $L$ называется отображение $b : L \times L \rightarrow \mathbb{F}$, линейное по каждому аргументу ($x, x_1, x_2, y, y_1, y_2 \in L$; $\alpha, \beta \in \mathbb{F}$):
> 1. $b(x_1 + x_2, y) = b(x_1, y) + b(x_2, y);$
> 2. $b(x, y_1 + y_2) = b(x, y_1) + b(x, y_2);$
> 3. $b(\alpha x, y) = \alpha b(x, y);$
> 4. $b(x, \beta y) = \beta b(x, y).$
> ^def1

> [!info] **Определение.** (Симметричная БФ)
> Билинейная форма называется **симметричной**, если $b(x, y) = b(y, x), \ \forall x, y \in L$.
> ^def2

*Примеры билинейных функций:*
1. $f(x, y) = x_1 y_1 + \dots + x_n y_n$ в $\mathbb{R}^n$.
2. скалярное произведение в $f(x, y) = (x, y) = |x||y|\cos(\alpha)$, $\alpha$ — угол между векторами $x$ и $y$.

> [!info] **Определение.** (Квадратичная форма)
> Пусть $b(\cdot, \cdot)$ — симметричная билинейная функция. Тогда **квадратичной функцией** (или квадратичной формой), порождённой $b(\cdot, \cdot)$, называется отображение $k : L \rightarrow \mathbb{F}$, которое определяется формулой: $k(x) = b(x, x)$.
> ^def3

*Примеры квадратичных функций:*
1. Скалярное произведение в $\mathbb{R}^n$ векторов $(x, y)$ — симметричная билинейная функция. Квадратичная форма $k(x) = b(x, x) = (x, x)$ сопоставляет вектору $x$ квадрат его длины.
2. Функции $x_1^2 - x_1 x_2 + x_3^2$, $\sqrt{2024} \cdot x_1 x_2$, $ix_1^2$ являются квадратичными формами.

Уже из приведённых выше примеров понятно, что квадратичная функция в общем случае линейной не является. В качестве примера рассмотрим образ суммы $x_1 + x_2$ под действием произвольной квадратичной функции $k(\cdot)$:
$$k(x_1 + x_2) = b(x_1 + x_2, x_1 + x_2) = b(x_1, x_1) + b(x_2, x_2) + b(x_1, x_2) + b(x_2, x_1) = k(x_1) + k(x_2) + 2b(x_1, x_2).$$

---

Из последней формулы можно заметить, что билинейную функцию на паре векторов $x_1, x_2$ можно выразить с помощью соответствующей ей квадратичной функции:
$$b(x_1, x_2) = \frac{k(x_1 + x_2) - k(x_1) - k(x_2)}{2}.$$

То есть, как квадратичная форма порождается симметричной билинейной формой, так и по квадратичной форме можно однозначно восстановить породившую её симметричную билинейную форму!

> [!info] **Определение.** (Знакоопределенность КФ)
> Квадратичная функция $k(\cdot)$ называется:
> * **положительно определённой**, если $k(x) > 0, \ x \neq \mathbf{0};$
> * **отрицательно определённой**, если $k(x) < 0, \ x \neq \mathbf{0};$
> * **положительно полуопределённой**, если $k(x) \geqslant 0, \ \forall x;$
> * **отрицательно полуопределённой**, если $k(x) \leqslant 0, \ \forall x.$
> ^def4

### Координатное представление билинейных и квадратичных форм
Рассмотрим базис $e = \{e_1, \dots, e_n\}$. Любой вектор этого пространства можно представить как линейную комбинация базисных векторов:
$$x = x_1 e_1 + \dots + x_n e_n, \quad y = y_1 e_1 + \dots + y_n e_n.$$

Подставим это в билинейную функцию:
$$b(x, y) = \sum_{i,j=1}^n x_i b(e_i, e_j) y_j. \quad (1)$$

Если ввести столбцы $x = (x_1, \dots, x_n)^T$ и $y = (y_1, \dots, y_n)^T$ и матрицу
$$B = \begin{pmatrix}
b(e_1, e_1) & \dots & b(e_1, e_n) \\
\vdots & \ddots & \vdots \\
b(e_n, e_1) & \dots & b(e_n, e_n)
\end{pmatrix} - \text{матрица билинейной функции,}$$
то выражение для (1) можно записать в следующем виде:
$$b(x, y) = x^T B y. \quad (2)$$

> [!error] **Утверждение.** (О симметричности матрицы БФ)
> Билинейная функция $b(\cdot, \cdot)$ симметрична тогда и только тогда, когда её матрица $B$ в некотором базисе симметрична.
> ^th1

**Доказательство:**
$\square$
**(Необходимость $\Rightarrow$):** Если $b(\cdot, \cdot)$ симметрична, то $b(e_i, e_j) = b(e_j, e_i) \ \forall i, j \Leftrightarrow B = B^T.$
**(Достаточность $\Leftarrow$):** Если матрица $B$ симметрична, то $b(e_i, e_j) = b(e_j, e_i) \ \forall i, j \Rightarrow b(x, y) = b(y, x).$ $\blacksquare$

Для квадратичной функции $k(\cdot)$, построенной по симметричной билинейной $b(\cdot, \cdot)$ с матрицей $B$ в базисе $e$ получаем:
$$k(x) = \sum_{i,j=1}^n x_i b(e_i, e_j) x_j = x^T B x.$$

### Изменение матриц при изменении базиса

> [!note] **Лемма.** (О единственности матрицы КФ)
> Пусть квадратные симметричные матрицы $A$ и $B$ порядка $n$ таковы, что для любого столбца $x$ размером $n \times 1$ выполняется $x^T A x = x^T B x$. Тогда $A = B$.

**Доказательство:**
$\square$ Рассмотрим матрицу $C = A - B = (c_{ij})$. Тогда $x^T C x = 0$ для любого столбца $x$. Сначала в качестве $x$ возьмём столбец $E_i$, у которого на $i$-ом месте стоит единица, а остальные нули. Тогда $E_i^T C E_i = c_{ii} = 0$. Затем в качестве $x$ возьмём столбец $E_{ij}$, у которого на местах с номерами $i$ и $j$ стоят единицы, а остальные нули. Тогда $E_{ij}^T C E_{ij} = c_{ii} + 2c_{ij} + c_{jj} = 0$. Отсюда получаем $c_{ij} = 0$, а значит и $A = B$. $\blacksquare$

При замене базиса матрица билинейной функции меняется. Пусть в базисе $e$ матрица $b(\cdot, \cdot)$ есть $B$. Пусть новый базис $e'$ связан со старым матрицей перехода $S$, то есть $e' = e S$. Пусть $x'$ и $y'$ — координатные столбцы векторов $x$ и $y$ в базисе $e'$. Тогда $x = S x'$ и $y = S y'$. По формуле (2) имеем
$$x'^T B' y' = b(x, y) = (S x')^T B (S y') = x'^T (S^T B S) y'.$$

И это верно для любой пары $x', y'$. Поэтому можно утверждать:
$$B' = S^T B S. \quad (3)$$
