# Пункт 16. Линейные преобразования евклидова пространства. Преобразование, сопряжённое данному. Матрица сопряжённого преобразования. Свойства сопряжённого преобразования.

### Линейные преобразования евклидова пространства

Линейное преобразование $\varphi: \mathbb{E} \to \mathbb{E}$ в евклидовом пространстве обладает специфическими свойствами, связанными со скалярным произведением.

### Преобразование, сопряженное данному

Рассмотрим некоторые примеры:
1) $\varphi = \operatorname{id}$ (тождественное преобразование). Тогда:
$$(\varphi(x), y) = (x, \varphi^*(y)) \Rightarrow (x, y - \varphi^*(y)) = 0 \quad \forall x, y \in \mathbb{E}.$$
Значит,
$$y = \varphi^*(y) \ \forall y \Rightarrow \varphi^* = \varphi = \operatorname{id}$$

2) $\varphi = \mathbf{0}$ (нулевое преобразование).
$$(\varphi(x), y) = (x, \varphi^*(y)) \Rightarrow 0 = (\mathbf{0}, y) = (x, \varphi^*(y)) \Rightarrow \varphi^*(y) = \mathbf{0} \quad \forall y$$

3) $\varphi(x) = [a, x]$, $a \neq \mathbf{0}$.
$$(\varphi(x), y) = ([a, x], y) = (a, x, y) = (x, [y, a])$$
Тогда
$$(x, [y, a] - \varphi^*(y)) = 0 \Rightarrow \varphi^*(y) = [y, a] = -[a, y] = -\varphi(y) \Rightarrow \varphi^* = -\varphi.$$

> [!info] **Определение 16.1.**
> Преобразование $A^*$ называется **сопряженным** к преобразованию $A$, если:
> $$\forall x, y \in E \quad (A(x), y) = (x, A^*(y))$$

> [!error] **Теорема 16.1.**
> Для любого линейного преобразования $A$ в евклидовом пространстве существует и единственно сопряженное преобразование $A^*$.

**Доказательство:**
$\square$ В матричном представлении:
$$x^T A^T \Gamma y = (Ax)^T \Gamma y = x^T \Gamma A^* y \Longrightarrow A^T \Gamma = \Gamma A^*$$
Отсюда матрица преобразования $A^*$ имеет вид $A^* = \Gamma^{-1} A^T \Gamma$, что доказывает существование и единственность. $\blacksquare$

### Матрица сопряженного преобразования

Как было показано в доказательстве выше, матрица $A^*$ сопряжённого преобразования $\varphi^*$ связана с матрицей $A$ исходного преобразования $\varphi$ соотношением:
$$A^* = \Gamma^{-1} A^T \Gamma,$$
где $\Gamma$ — матрица Грама базиса. В ортонормированном базисе (ОНБ) матрица Грама $\Gamma = E$, и формула упрощается до:
$$A^* = A^T.$$

### Свойства сопряженного преобразования

> [!error] **Теорема 16.2.**
> $$\operatorname{Im} A = (\operatorname{Ker} A^*)^{\perp}$$

**Доказательство:**
$\square$ Для любого $x \in E$ и $y \in \operatorname{Ker} A^*$:
$$(A(x), y) = (x, A^*(y)) = (x, 0) = 0 \Longrightarrow \operatorname{Im} A \subset (\operatorname{Ker} A^*)^{\perp}$$
Обозначим $\dim E = n, \operatorname{rg} A = r$.  
Тогда $\dim(\operatorname{Ker} A^*) = n - r$ (размерность пространства решений однородной системы $A^T y = 0$).  
Отсюда:
$$\dim(\operatorname{Ker} A^*)^{\perp} = n - (n - r) = r = \dim\operatorname{Im} A$$
Следовательно, подпространства совпадают. $\blacksquare$

**Другие свойства:**
1. $(\varphi^*)^* = \varphi$. (В ОНБ, как отмечалось выше, $A^* = A^T$, а у нас также $(A^T)^T = A$).
2. $(\varphi\psi)^* = \psi^*\varphi^*$. (Так как $(AB)^T = B^T A^T$).
3. Характеристические многочлены $\varphi$ и $\varphi^*$ совпадают. (Так как совпадают ХМ $A$ и $A^T$).
4. Все собственные значения и их кратности совпадают. (Так как $(A - \lambda E)^T = (A^T - \lambda E)$, и детерминант не изменяется при транспонировании).
