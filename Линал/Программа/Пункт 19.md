# Пункт 19. Сингулярное разложение. Полярное разложение линейных преобразований евклидова пространства и матриц.

> [!error] **Теорема 17.8 (О полярном разложении).**
> Любое невырожденное преобразование $A$ может быть представлено в виде полярного разложения:
> $$A = QS,$$
> где $Q$ — ортогональное преобразование, $S$ — самосопряженное преобразование с положительными собственными значениями.

**Доказательство:**
$\square$ Оператор $A^*A$ самосопряжен и положительно определен ($(A^*Ah, h) = |Ah|^2 > 0$). Пусть $e_1, \dots, e_n$ — ОНБ из СВ оператора $A^*A$. $(A(e_i), A(e_j)) = (A^*A(e_i), e_j) = \lambda_i \delta_{ij}$. Векторы $A(e_i)$ ортогональны, $|A(e_i)| = \sqrt{\lambda_i} = \sigma_i$ (сингулярные числа). Положим $f_i = A(e_i)/\sigma_i$. Зададим $Q(e_i) = f_i$ (ортогонально). Тогда $S = Q^{-1}A$ в базисе $e_i$ диагонален: $S(e_i) = Q^{-1}(\sigma_i f_i) = \sigma_i e_i$. $\blacksquare$

#### Сингулярное разложение
Из доказательства теоремы 17.8 следует сингулярное разложение:
$$A = Q S = U \Sigma V^T$$
где $\Sigma = \operatorname{diag}(\sigma_1, \dots, \sigma_n)$, а $U, V$ — ортогональные матрицы.
Сингулярные числа $\sigma_i$ — это длины образов векторов ортонормированного базиса.
