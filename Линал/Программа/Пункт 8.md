# Пункт 8. Собственные векторы и собственные значения. Линейная независимость собственных векторов, принадлежащих различным собственным значениям. Инвариантные подпространства линейных преобразований. Ограничение преобразования на инвариантное подпространство. Собственные подпространства.

### Собственные векторы и собственные значения

> [!info] **Определение 10.3.**
> Пусть $x$ — ненулевой вектор такой, что $Ax = \lambda x$. Тогда $x$ называется *собственным вектором*, соответствующим собственному значению $\lambda$.

Пусть $x \neq 0, Ax = \lambda x$. Тогда $A(kx) = kAx = k\lambda x = \lambda(kx)$.

### Линейная независимость собственных векторов

> [!note] **Лемма.**
> Собственные векторы $x_1, \dots, x_m$, соответствующие различным собственным значениям $\lambda_1, \dots, \lambda_m$, линейно независимы.

**Доказательство:**
$\square$ Используем метод математической индукции по числу собственных значений $m$:
1. База индукции: $m = 1$. Так как собственный вектор $x \neq \mathbf{0}$, то система $\{x\}$ линейно независима.
2. Предположение индукции. Пусть для $(m - 1)$ векторов утверждение верно.
3. Шаг индукции. Рассмотрим для $m$:
$$\alpha_1 x_1 + \dots + \alpha_{m-1} x_{m-1} + \alpha_m x_m = \mathbf{0} \mid \cdot \lambda_m$$
$$\alpha_1 \lambda_m x_1 + \dots + \alpha_{m-1} \lambda_m x_{m-1} + \alpha_m \lambda_m x_m = \mathbf{0}$$
Теперь подействуем $\varphi$ на первое уравнение:
$$\alpha_1 \varphi(x_1) + \dots + \alpha_{m-1} \varphi(x_{m-1}) + \alpha_m \varphi(x_m) = \alpha_1 \lambda_1 x_1 + \dots + \alpha_{m-1} \lambda_{m-1} x_{m-1} + \alpha_m \lambda_m x_m = \mathbf{0}$$
Вычтем из второго уравнения третье:
$$\alpha_1(\lambda_m - \lambda_1) x_1 + \dots + \alpha_{m-1}(\lambda_m - \lambda_{m-1}) x_{m-1} = \mathbf{0}$$
В силу того, что $\lambda_m - \lambda_i \neq 0$, $i = 1, \dots, m - 1$, получаем $\alpha_1 = \dots = \alpha_{m-1} = 0$. Тогда $\alpha_m x_m = \mathbf{0} \Rightarrow \alpha_m = 0$, так как $x_m \neq \mathbf{0}$. $\blacksquare$

### Инвариантные подпространства

> [!info] **Определение 9.4.**
> Подпространство $L_1 \subset L$ называется *инвариантным* относительно $\varphi$, если
> $$\forall x \in L_1 \quad \varphi(x) \in L_1, \text{ или, что то же, } \varphi(L_1) \subset L_1$$

> [!error] **Теорема 10.1.**
> Пусть $A$ и $B$ — перестановочные линейные преобразования. Тогда ядро и образ одного из них инвариантно относительно другого.

**Доказательство:**
$\square$ 1. $x \in \operatorname{Ker} A \Longrightarrow A(x) = 0 \Longrightarrow B(A(x)) = 0$. Тогда $A(B(x)) = 0 \Longrightarrow B(x) \in \operatorname{Ker} A \ \forall x \in \operatorname{Ker} A$.
2. Если $x \in \operatorname{Im} A \Longrightarrow \exists \text{ вектор } z : x = A(z)$. Тогда $B(x) = B(A(z)) = A(B(z)) \Longrightarrow B(x) \in \operatorname{Im} A \ \forall x \in \operatorname{Im} A$. $\blacksquare$

$A$ и $A - \lambda E$ — перестановочные как многочлены от $A$. $\operatorname{Ker}(A - \lambda E)$ — инвариантно относительно $A$.

### Ограничение преобразования на инвариантное подпространство

> [!info] **Определение 10.1.**
> Преобразование $A$, сопоставляющее каждому вектору из инвариантного подпространства $L'$ вектор из $L'$, называется *ограничением* $A$ на $L'$.

> [!error] **Теорема 10.2.**
> Собственные вектора и только они являются базисами одномерных инвариантных подпространств.

**Доказательство:**
$\square$ 1. Пусть вектор $x$ — собственный а $y$ принадлежит одномерному подпространству $L'$ с базисом $x$. Тогда $y = \alpha x$ и $Ay = \alpha Ax = \alpha\lambda x \Longrightarrow Ay \in L'$.
2. Пусть $x$ — базис инвариантного подпространства $L'$. Тогда $Ax$ лежит в $L'$ и раскладывается по базису: $Ax = \lambda x$. Т.к. $x \neq 0$, он собственный. $\blacksquare$

### Собственные подпространства

> [!info] **Определение 10.2.**
> Если для числа $\lambda$ подпространство $\operatorname{Ker}(A - \lambda E)$ — ненулевое, то $\lambda$ называется собственным значением преобразования, а подпространство — собственным подпространством, соответствующим (или принадлежащим) собственному значению $\lambda$.
