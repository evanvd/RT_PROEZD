# Билет 12. Критерий интегрируемости Дарбу

### Верхние и нижние суммы Дарбу, их свойства

Пусть $[a; b]$ — отрезок в $\mathbb{R}^1$; $a < b$. Рассмотрим разбиение отрезка $R$: 
$$x_0 = a < x_1 < x_2 < \dots < x_N = b.$$
 Длины отрезков разбиения 
$$\Delta x_i = x_i - x_{i-1}, i = 1, \dots, N.$$
 Мелкость разбиения 
$$|R| = \max_{i=1, \dots, N} \Delta x_i.$$


Пусть $f$ — ограниченная функция на $[a; b]$;

$$M_i = \sup_{[x_{i-1};x_i]} f(x); \quad m_i = \inf_{[x_{i-1};x_i]} f(x), \quad \omega_i = M_i - m_i, \quad i = 1, \dots, N.$$


> [!info] Определение (Суммы Дарбу).
> Для ограниченной функции $f$ на отрезке $[a; b]$ и разбиения $R$ определим:
> $$S^*_R = \sum_{i=1}^N M_i \Delta x_i \quad \text{(верхняя сумма Дарбу);}$$
> $$S_{*R} = \sum_{i=1}^N m_i \Delta x_i \quad \text{(нижняя сумма Дарбу);}$$
> $$\omega_R = S^*_R - S_{*R} = \sum_{i=1}^N \omega_i \Delta x_i \quad \text{(колебание функции на разбиении)}.$$
> ^def12dSum

> [!note] Лемма 12.1 (Свойства сумм Дарбу при измельчении разбиения).
> Для двух разбиений $R_1$ и $R_2$ отрезка $[a; b]$ таких, что $R_2 > R_1$, и для функции $f$, ограниченной на $[a; b]$, выполняются неравенства
> $$S^*_{R_2} \le S^*_{R_1}; \quad S_{*R_2} \ge S_{*R_1}; \quad \omega_{R_2} \le \omega_{R_1}.$$
> ^l12d1

**Доказательство:**
$\square$ Докажем первое из нужных неравенств. Достаточно рассмотреть случай, когда $R_2$ получается из $R_1$ добавлением одной точки $x' \in (x_{i-1}; x_i)$. Тогда все слагаемые суммы $S^*_{R_1}$, кроме $i$-го, не изменятся; $i$-е же слагаемое, которое было равно $M_i(x_i - x_{i-1})$, превратится в сумму $M'_i(x_i - x') + M''_i(x' - x_{i-1})$, где 
$$M'_i = \sup_{[x';x_i]} f(x), M''_i = \sup_{[x_{i-1};x']} f(x).$$
 Ясно, что 
$$M'_i \le M_i, M''_i \le M_i,$$
 поэтому последняя сумма не превосходит $M_i(x_i - x' + x' - x_{i-1}) = M_i(x_i - x_{i-1})$. Значит, $S^*_{R_2} \le S^*_{R_1}$. Аналогично $S_{*R_2} \ge S_{*R_1}$.
Наконец, 
$$\omega_{R_2} = S^*_{R_2} - S_{*R_2} \le S^*_{R_1} - S_{*R_1} = \omega_{R_1}. \blacksquare$$


> [!note] Лемма 12.2 (О сравнении верхней и нижней сумм Дарбу).
> Для любых двух разбиений $R_1$ и $R_2$ отрезка $[a; b]$ и для любой функции $f$, ограниченной на $[a; b]$, выполняется неравенство $S^*_{R_1} \ge S_{*R_2}$.
> ^l12d2

**Доказательство:**
$\square$ Рассмотрим разбиение $R = \max(R_1, R_2)$. Тогда 
$$R > R_1, R > R_2,$$
 и по [[#^l12d1|Лемме 12.1 (Свойства сумм Дарбу при измельчении разбиения)]] 
$$S^*_{R_1} \ge S^*_R \ge S_{*R} \ge S_{*R_2}. \blacksquare$$


### Критерий интегрируемости Дарбу.

> [!error] Теорема 12.1 (Критерий интегрируемости Дарбу).
> Для ограниченной функции $f$ на отрезке $[a; b]$ равносильны следующие три условия:
> 1° Функция $f$ интегрируема по Риману на $[a; b]$.
> 2° Для любого $\varepsilon > 0$ найдётся разбиение $R$ отрезка $[a; b]$ такое, что $\omega_R < \varepsilon$.
> 3° Для любого $\varepsilon > 0$ найдётся $\delta > 0$ такое, что для любого разбиения $R$ отрезка $[a; b]$, мелкость которого меньше $\delta$, выполняется неравенство $\omega_R < \varepsilon$.
> ^th12d1

**Доказательство:**
$\square$ 3° $\implies$ 2° — очевидно.
2° $\implies$ 1°. Пусть для любого $\varepsilon > 0$ найдётся разбиение $R$ такое, что 
$$\omega_R = S^*_R - S_{*R} < \varepsilon.$$
 Так как 
$$S^*_R \ge I^* \ge I_* \ge S_{*R},$$
 то 
$$0 \le I^* - I_* \le S^*_R - S_{*R} < \varepsilon.$$
 Но $\varepsilon > 0$ — произвольно, поэтому $I^* - I_* = 0$, и функция интегрируема по Риману на $[a; b]$.
1° $\implies$ 2°. Так как 
$$I^* = I_* = I = \sup_R S_{*R},$$
 то 
$$\forall \varepsilon > 0 \to \exists R_1 : S_{*R_1} > I - \frac{\varepsilon}{2}.$$
 Также $I = \inf_R S^*_R$, поэтому 
$$\forall \varepsilon > 0 \to \exists R_2 : S^*_{R_2} < I + \frac{\varepsilon}{2}.$$
 Рассмотрим разбиение $R = \max(R_1, R_2)$. Из [[#^l12d1|Леммы 12.1 (Свойства сумм Дарбу при измельчении разбиения)]] следует, что
$$S_{*R} \ge S_{*R_1} > I - \frac{\varepsilon}{2}; \quad S^*_R \le S^*_{R_2} < I + \frac{\varepsilon}{2}.$$
Значит, 
$$\omega_R = S^*_R - S_{*R} < \varepsilon.$$

2° $\implies$ 3° (самая сложная часть доказательства). Пусть для любого $\varepsilon > 0$ найдётся разбиение $R_0$ такое, что 
$$\omega_{R_0} = \sum_{i=1}^{N_0} \omega_i^0 \Delta x_i^0 < \frac{\varepsilon}{2}.$$
 Если функция $f$ тождественно равна нулю на $[a; b]$, то все $\omega_R = 0$ и доказывать нечего, если нет, то 
$$M = \sup_{[a;b]} |f(x)| > 0.$$

Рассмотрим 
$$\delta = \min \left( \delta_0, \frac{\varepsilon}{4MN_0} \right),$$
 где $\delta_0 = |R_0|$. Пусть $R$ — любое разбиение отрезка $[a; b]$ такое, что $|R| < \delta$. Тогда 
$$\omega_R = \sum_{i=1}^N \omega_i \Delta x_i = \Sigma' + \Sigma'',$$
 где $\Sigma'$ берётся по отрезкам, целиком принадлежащим отрезкам разбиения $R_0$, а $\Sigma''$ — по отрезкам, каждый из которых принадлежит объединению двух соседних отрезков $R_0$ (на рис. 12.2 отрезки из $\Sigma''$ отмечены дужками; их не более чем $N_0 - 1$). Рассмотрим разбиение $R_1 = \max(R_0, R)$ — это разбиение, состоящее из всех отрезочков на рис. 12.2.
![[Pasted image 20260519222900.png]]
Так как $R_1 > R_0$, то 
$$\omega_{R_1} \le \omega_{R_0} < \frac{\varepsilon}{2}$$
 (в силу [[#^l12d1|Леммы 12.1]]); значит, 
$$\Sigma' \le \omega_{R_1} < \frac{\varepsilon}{2}.$$

Далее, для всех $i = 1, \dots, N$ выполняется неравенство 
$$\omega_i = M_i - m_i \le 2M,$$
 поэтому 
$$\Sigma'' = \sum \omega_i \Delta x_i \le 2M(N_0 - 1)\delta.$$
 Значит, 
$$\Sigma'' < 2MN_0 \delta \le 2MN_0 \cdot \frac{\varepsilon}{4MN_0} = \frac{\varepsilon}{2}.$$
 Итак, 
$$\forall \varepsilon > 0 \to \exists \delta > 0 : \forall R, |R| < \delta \to \omega_R < \varepsilon. \blacksquare$$
