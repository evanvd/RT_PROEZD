# ГЛАВА IX. ФУНКЦИИ МНОГИХ ПЕРЕМЕННЫХ

## § 1. Точечное пространство $\mathbb{R}^n$. Некоторые классы точечных множеств

**Определение 9.1.** Пространством $\mathbb{R}^n$ (точечным $n$-мерным евклидовым пространством), $n \in \mathbb{N}$, называется множество всевозможных упорядоченных наборов из $n$ действительных чисел $x = (\alpha_1, \dots, \alpha_n)$. Два таких набора $x = (\alpha_1, \dots, \alpha_n)$ и $y = (\beta_1, \dots, \beta_n)$ называются равными, если $\alpha_1 = \beta_1, \dots, \alpha_n = \beta_n$. Расстоянием между $x$ и $y$ называется число $\rho(x, y) = \sqrt{(\alpha_1 - \beta_1)^2 + \dots + (\alpha_n - \beta_n)^2}$.

Перечислим основные свойства расстояния. Для любых $x, y, z \in \mathbb{R}^n$:
1. $\rho(x, y) \ge 0; \rho(x, y) = 0 \iff x = y$.
2. $\rho(x, y) = \rho(y, x)$.
3. $\rho(x, z) \le \rho(x, y) + \rho(y, z)$.

Множество произвольной природы, для любых двух элементов которого определено расстояние, удовлетворяющее условиям 1°, 2°, 3°, называется метрическим пространством. Более подробно об этом будет говориться в последней части курса, а пока убедимся в том, что для элементов $\mathbb{R}^n$ выполняется свойство 3° — так называемое неравенство треугольника (свойства 1° и 2° очевидны).

$\square$ Пусть $x = (\alpha_1, \dots, \alpha_n)$, $y = (\beta_1, \dots, \beta_n)$, $z = (\gamma_1, \dots, \gamma_n)$. Докажем, что
$$\sqrt{(\alpha_1 - \gamma_1)^2 + \dots + (\alpha_n - \gamma_n)^2} \le$$
$$\le \sqrt{(\alpha_1 - \beta_1)^2 + \dots + (\alpha_n - \beta_n)^2} +$$
$$+ \sqrt{(\beta_1 - \gamma_1)^2 + \dots + (\beta_n - \gamma_n)^2}.$$
Обозначим $\alpha_i - \beta_i = p_i$, $\beta_i - \gamma_i = q_i$; тогда $\alpha_i - \gamma_i = p_i + q_i$ ($i = 1, 2, \dots, n$). Нужное неравенство примет вид
$$\sqrt{(p_1 + q_1)^2 + \dots + (p_n + q_n)^2} \le \sqrt{p_1^2 + \dots + p_n^2} + \sqrt{q_1^2 + \dots + q_n^2}.$$

---

Возводя в квадрат обе (неотрицательные) части последнего неравенства, получим равносильное неравенство:
$$p_1^2 + 2p_1q_1 + q_1^2 + \dots + p_n^2 + 2p_nq_n + q_n^2 \le$$
$$\le p_1^2 + \dots + p_n^2 + q_1^2 + \dots + q_n^2 + 2 \sqrt{p_1^2 + \dots + p_n^2} \cdot \sqrt{q_1^2 + \dots + q_n^2};$$
достаточно доказать неравенство
$$(p_1^2 + \dots + p_n^2)(q_1^2 + \dots + q_n^2) \ge (p_1q_1 + \dots + p_nq_n)^2. \tag{9.1}$$
Последнее неравенство называется неравенством Коши–Буняковского.

Рассмотрим функцию одной переменной:
$$f(t) = (p_1t + q_1)^2 + \dots + (p_nt + q_n)^2 =$$
$$= (p_1^2 + \dots + p_n^2)t^2 + 2(p_1q_1 + \dots + p_nq_n)t + (q_1^2 + \dots + q_n^2).$$
Так как квадратный трёхчлен $f(t)$ при всех $t$ неотрицателен, то его дискриминант $(p_1q_1 + \dots + p_nq_n)^2 - (p_1^2 + \dots + p_n^2)(q_1^2 + \dots + q_n^2) \le 0$, что равносильно (9.1). $\blacksquare$

При $n = 1, 2, 3$ пространство $\mathbb{R}^n$ допускает естественную геометрическую интерпретацию (числовая прямая $\mathbb{R}^1$ (т.е. $\mathbb{R}$), плоскость $\mathbb{R}^2$, трёхмерное евклидово пространство $\mathbb{R}^3$).

**Замечание.** Точечное $n$-мерное евклидово пространство не следует путать с $n$-мерным векторным евклидовым пространством, изучаемым в линейной алгебре (там векторы, здесь точки; векторы можно складывать, точки — нет и т.д.).

**Определение 9.2.** $\varepsilon$-окрестностью точки $x_0 \in \mathbb{R}^n$, где $\varepsilon > 0$, называется множество
$$U_\varepsilon(x_0) = \{x \in \mathbb{R}^n : \rho(x, x_0) < \varepsilon\}.$$
Проколотой $\varepsilon$-окрестностью точки $x_0 \in \mathbb{R}^n$, где $\varepsilon > 0$, называется множество
$$\mathring{U}_\varepsilon(x_0) = U_\varepsilon(x_0) \setminus \{x_0\} = \{x \in \mathbb{R}^n : 0 < \rho(x, x_0) < \varepsilon\}.$$

При $n = 1$ для точек $x, y \in \mathbb{R}$ расстояние $\rho(x, y) = |x - y|$; понятие $\varepsilon$-окрестности совпадает с введённым ранее (определение 2.4). При $n = 2$ ($n = 3$) $\varepsilon$-окрестность точки $x_0$ иначе называют открытым кругом (шаром) радиуса $\varepsilon$ с центром в

---

точке $x_0$, поэтому в общем случае $U_\varepsilon(x_0)$ называют открытым $n$-мерным шаром радиуса $\varepsilon$ с центром в точке $x_0$.

**Определение 9.3.** Точка $x_0$ называется **внутренней точкой** множества $X \subset \mathbb{R}^n$, если она принадлежит $X$ вместе с некоторой окрестностью (см. рис. ).

**Определение 9.4.** Множество $G \subset \mathbb{R}^n$ называется **открытым**, если все его точки внутренние, т.е. $\forall x_0 \in G \to \exists \varepsilon > 0 : U_\varepsilon(x_0) \subset G$.

**Лемма 9.1.** $U_\varepsilon(x_0)$ — открытое множество ($x_0 \in \mathbb{R}^n, \varepsilon > 0$).
$\square$ Пусть $x_1 \in U_\varepsilon(x_0)$, т.е. $\rho(x_1, x_0) < \varepsilon$. Обозначим $\delta = \varepsilon - \rho(x_1, x_0) > 0$. Тогда $\forall x \in U_\delta(x_1)$ (см. рис. 9.2):
$$\rho(x, x_0) \le \rho(x, x_1) + \rho(x_1, x_0) < \delta + \varepsilon - \delta = \varepsilon,$$
значит, $x \in U_\varepsilon(x_0)$. Поэтому $U_\delta(x_1) \subset U_\varepsilon(x_0)$, так как $x_1$ — произвольная точка $U_\varepsilon(x_0)$, то $U_\varepsilon(x_0)$ — открытое множество. $\blacksquare$

**Пример 9.1.** Интервал $(a; b)$ ($a < b$; возможно, $a = -\infty$ или $b = +\infty$) является открытым множеством в $\mathbb{R}^1$, так как любая точка $(a; b)$ войдёт в него с некоторой окрестностью. Этот же интервал, рассматриваемый как множество в $\mathbb{R}^2$, не будет открытым множеством, так как не может содержать никакой двумерной окрестности. Отрезок $[a; b]$ не является открытым множеством в $\mathbb{R}^1$, так как никакая окрестность точки $a$ (или точки $b$) не может принадлежать целиком отрезку $[a; b]$.

---

Последовательностью точек в $\mathbb{R}^n$ будем называть функцию, определённую на множестве натуральных чисел, множество значений которой принадлежит $\mathbb{R}^n$. Индекс члена последовательности можно обозначать любой буквой, соответствующей произвольному натуральному числу. Мы не будем обозначать его буквой $n$ потому, что $n$ — это размерность пространства.

**Определение 9.5.** Говорят, что последовательность точек $x_k$ из $\mathbb{R}^n$ сходится к точке $x_0 \in \mathbb{R}^n$ (обозначение $\lim_{k \to \infty} x_k = x_0$), если $\lim_{k \to \infty} \rho(x_k, x_0) = 0$, т.е.
$$\forall \varepsilon > 0 \to \exists k_0 : \forall k \ge k_0 \to \rho(x_k, x_0) < \varepsilon.$$

**Лемма 9.2.** Последовательность точек $x_k = (\alpha_1^{(k)}, \dots, \alpha_n^{(k)}) \in \mathbb{R}^n, k = 1, 2, \dots,$ сходится к точке $x_0 = (\alpha_1^{(0)}, \dots, \alpha_n^{(0)}) \iff$ при всех $i = 1, 2, \dots, n$ выполняется равенство $\lim_{k \to \infty} \alpha_i^{(k)} = \alpha_i^{(0)}$ (сходимость в $\mathbb{R}^n$ равносильна покоординатной сходимости).

$\square$ ($\implies$) Так как $\rho(x_k, x_0) = \sqrt{(\alpha_1^{(k)} - \alpha_1^{(0)})^2 + \dots + (\alpha_n^{(k)} - \alpha_n^{(0)})^2}$, то $|\alpha_i^{(k)} - \alpha_i^{(0)}| \le \rho(x_k, x_0)$ при всех $i = 1, 2, \dots, n$. Поэтому если $\forall \varepsilon > 0 \to \exists k_0 : \forall k \ge k_0 \to \rho(x_k, x_0) < \varepsilon$, то и подавно $|\alpha_i^{(k)} - \alpha_i^{(0)}| < \varepsilon$ при всех $i = 1, 2, \dots, n$. Значит, $\lim_{k \to \infty} \alpha_i^{(k)} = \alpha_i^{(0)}$ при всех $i = 1, 2, \dots, n$.

($\impliedby$) Так как $\lim_{k \to \infty} \alpha_i^{(k)} = \alpha_i^{(0)}$ при всех $i = 1, 2, \dots, n$, то
$$\forall \varepsilon > 0 \to \exists k_i : \forall k \ge k_i \to |\alpha_i^{(k)} - \alpha_i^{(0)}| < \frac{\varepsilon}{\sqrt{n}}.$$
Тогда при $k \ge k_0 = \max(k_1, \dots, k_n)$ выполняется неравенство
$$\rho(x_k, x_0) < \sqrt{n \cdot \left(\frac{\varepsilon}{\sqrt{n}}\right)^2} = \varepsilon, \text{ значит, } \lim_{k \to \infty} x_k = x_0. \blacksquare$$

**Определение 9.6.** Множество $X \subset \mathbb{R}^n$ называется **ограниченным**, если оно целиком принадлежит некоторой окрестности точки $O = (0, 0, \dots, 0)$, т.е. $\exists C > 0 : X \subset U_C(0)$. Последовательность $x_k$ называется ограниченной, если множество её значений ограничено.

---

**Теорема 9.1 (теорема Больцано–Вейерштрасса в $\mathbb{R}^n$).** Если последовательность точек $x_k$ из $\mathbb{R}^n$ ограничена, то существует сходящаяся подпоследовательность $x_{k_m}$ ($k_1 < k_2 < \dots < k_m < \dots; k_m \in \mathbb{N}$).

$\square$ Доказательство проведём для случая $\mathbb{R}^2$ (в случае произвольного $n$ изменения очевидны, только доказательство становится более громоздким).
Пусть $x_k = (\alpha_1^{(k)}, \alpha_2^{(k)})$; так как при всех $k = 1, 2, \dots$ выполняется неравенство $\sqrt{(\alpha_1^{(k)})^2 + (\alpha_2^{(k)})^2} \le C$, то $|\alpha_1^{(k)}| \le C$, $|\alpha_2^{(k)}| \le C$. Последовательность $\alpha_1^{(k)}$ ограничена, поэтому по теореме Больцано–Вейерштрасса 2.8 существует сходящаяся подпоследовательность $\alpha_1^{(k_l)}$, $\lim_{l \to \infty} \alpha_1^{(k_l)} = \alpha_1^{(0)}$. Далее, так как последовательность $\alpha_2^{(k_l)}$ ограничена, то по теореме 2.8 существует сходящаяся подпоследовательность $\alpha_2^{(k_{l_m})}$; $\lim_{m \to \infty} \alpha_2^{(k_{l_m})} = \alpha_2^{(0)}$. Но $\lim_{m \to \infty} \alpha_1^{(k_{l_m})} = \alpha_1^{(0)}$ (предел подпоследовательности сходящейся последовательности), поэтому по лемме 9.2 $\lim_{m \to \infty} x_{k_{l_m}} = (\alpha_1^{(0)}, \alpha_2^{(0)})$; $x_{k_{l_m}}$ — подпоследовательность последовательности точек $x_k$. $\blacksquare$

**Теорема 9.2 (критерий Коши сходимости последовательности точек $\mathbb{R}^n$).** Последовательность $x_k$ точек $\mathbb{R}^n$ сходится $\iff \forall \varepsilon > 0 \to \exists k_0 \forall k, m \ge k_0 \to \rho(x_k, x_m) < \varepsilon$.

$\square$ ($\implies$) Пусть $\lim_{k \to \infty} x_k = x_0$. Тогда
$$\forall \varepsilon > 0 \to \exists k_0 : \left( \forall k \ge k_0 \to \rho(x_k, x_0) < \frac{\varepsilon}{2} \right) \land \left( \forall m \ge k_0 \to \rho(x_m, x_0) < \frac{\varepsilon}{2} \right).$$
Тогда для любых $k \ge k_0$ и $m \ge k_0$ выполняется неравенство
$$\rho(x_k, x_m) \le \rho(x_k, x_0) + \rho(x_0, x_m) < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon.$$

($\impliedby$) Пусть $x_k = (\alpha_1^{(k)}, \dots, \alpha_n^{(k)})$. Тогда $\forall k, m \ge k_0$ выполняется неравенство
$$\rho(x_k, x_m) = \sqrt{(\alpha_1^{(k)} - \alpha_1^{(m)})^2 + \dots + (\alpha_n^{(k)} - \alpha_n^{(m)})^2} < \varepsilon.$$

---

Поэтому при фиксированном $i = 1, 2, \dots, n$ и подавно $|\alpha_i^{(k)} - \alpha_i^{(m)}| \le \rho(x_k, x_m) < \varepsilon$. Значит, при фиксированном $i$ последовательность $\alpha_i^{(k)}$ фундаментальна, и по критерию Коши 2.13 она сходится к некоторому числу $\alpha_i^{(0)}$. По лемме 9.2 последовательность $x_k$ сойдется к $(\alpha_1^{(0)}, \dots, \alpha_n^{(0)})$. $\blacksquare$

**Определение 9.7.** Точка $x_0 \in \mathbb{R}^n$ называется **предельной точкой** множества $X \subset \mathbb{R}^n$, если существует последовательность точек $x_k$ из $X$ такая, что $x_k \ne x_0$ и $\lim_{k \to \infty} x_k = x_0$.

**Определение 9.8.** Множество $F \subset \mathbb{R}^n$ называется **замкнутым**, если оно содержит все свои предельные точки.

**Определение 9.9.** Замкнутое ограниченное множество в $\mathbb{R}^n$ называется **компактом**.

**Пример 9.2.** Интервал $(a; b)$ не является замкнутым множеством в $\mathbb{R}^1$, так как концы $a$ и $b$ являются его предельными точками ($\lim_{k \to \infty} (a + \frac{1}{k}) = a, a + \frac{1}{k} \in (a; b)$ при $k \ge k_0, a + \frac{1}{k} \ne a$; $\lim_{k \to \infty} (b - \frac{1}{k}) = b, b - \frac{1}{k} \in (a; b)$ при $k \ge k_0, b - \frac{1}{k} \ne b$). Отрезок $[a; b]$ является замкнутым множеством в $\mathbb{R}^1$ (кстати, и в $\mathbb{R}^n$ при $n \ge 1$), так как пределы последовательностей точек из $[a; b]$ лежат на $[a; b]$. Но отрезок — ограниченное множество, значит, это — компакт. Примерами замкнутых неограниченных множеств в $\mathbb{R}^1$ (и в $\mathbb{R}^n$ при $n \ge 1$) являются замкнутые лучи $(-\infty; b]$ и $[a; +\infty)$, а также вся прямая $\mathbb{R}^1$.

**Определение 9.10.** Точка $x_0$ множества $X \subset \mathbb{R}^n$ называется **изолированной**, если найдётся её проколотая окрестность, не имеющая общих точек с $X$ (т.е. $\exists \varepsilon > 0 : \mathring{U}_\varepsilon(x_0) \cap X = \varnothing$).

**Определение 9.11.** **Точками прикосновения** множества $X \subset \mathbb{R}^n$ называются все его предельные и изолированные точки.

**Лемма 9.3.** Если точка $x_0$ множества $X \subset \mathbb{R}^n$ не является изолированной, то она является предельной точкой $X$.
$\square$ Если точка $x_0$ не является изолированной точкой $X$, то
$$\forall \varepsilon > 0 \to \exists x \in \mathring{U}_\varepsilon(x_0) \cap X.$$

---

Рассмотрим $\varepsilon = \frac{1}{k}$, $k = 1, 2, \dots$ Тогда найдётся последовательность точек $x_k \in X$ такая, что $0 < \rho(x_k, x_0) < \frac{1}{k}$. Это значит, что $\lim_{k \to \infty} \rho(x_k, x_0) = 0$, т.е. $\lim_{k \to \infty} x_k = x_0$, и при этом $x_k \ne x_0$. Поэтому $x_0$ — предельная точка $X$. $\blacksquare$

Таким образом, все точки множества либо предельные, либо изолированные, т.е. любое множество в $\mathbb{R}^n$ является подмножеством своих точек прикосновения. При этом предельные точки множества могут ему не принадлежать (изолированные точки принадлежат множеству).

**Определение 9.12.** Пусть $\alpha_1(t), \alpha_2(t), \dots, \alpha_n(t)$ — непрерывные функции одной переменной $t \in I$ ($I$ — промежуток числовой прямой). Тогда множество точек из $\mathbb{R}^n$
$$\Gamma = \{ (\alpha_1, \dots, \alpha_n) : \alpha_1 = \alpha_1(t), \dots, \alpha_n = \alpha_n(t), t \in I \}$$
называется **непрерывной кривой**. Если $I = [a; b]$ — отрезок ($a < b$), то говорят, что кривая $\Gamma$ соединяет точки
$x_1 = (\alpha_1(a), \dots, \alpha_n(a))$ и $x_2 = (\alpha_1(b), \dots, \alpha_n(b))$.

Это определение обобщает определения непрерывной кривой в $\mathbb{R}^2$ и $\mathbb{R}^3$ (определения 4.3 и 6.8).

**Определение 9.13.** Непустое множество точек $X \subset \mathbb{R}^n$ называется **связным**, если для любых различных точек $x_1, x_2 \in X$ найдётся непрерывная кривая $\Gamma$, соединяющая точки $x_1$ и $x_2$, все точки которой принадлежат $X$. Множество, состоящее из одной точки, по определению является связным.

**Замечание.** В курсах математического анализа с более выраженным «топологическим уклоном» такие множества называются *линейно связными*. Термин «связное множество» применяется к более широкому классу множеств.

**Определение 9.14.** Открытое связное множество $G \subset \mathbb{R}^n$ называется **областью**.

**Пример 9.3.** Непрерывная кривая в $\mathbb{R}^1$ — это множество точек $\Gamma = \{x : x = x(t), t \in I \}$, где $x(t)$ — функция, непрерывная на промежутке $I$. Это множество значений функции,

---

непрерывной на промежутке $I$, т.е. промежуток или единственная точка (лемма 3.11).

**Пример 9.4.** Докажем, что множество точек $X \subset \mathbb{R}^1$, состоящее более чем из одной точки, связно $\iff X$ является промежутком.
$\square$ ($\impliedby$) Для любых двух точек $x_1, x_2 \in X$ отрезок с концами в $x_1$ и $x_2$ целиком принадлежит $X$, а отрезок — это непрерывная кривая в $\mathbb{R}^1$.
($\implies$) Любые две различные точки $x_1, x_2 \in X$ можно соединить непрерывной кривой $\Gamma = \{ x : x = x(t), t \in [a; b] \}$; при этом $x(a) = x_1, x(b) = x_2$. Кривая $\Gamma$ является множеством значений функции $x$ на отрезке $[a; b]$, которое содержит отрезок $[x_1; x_2]$ (теорема 3.14), поэтому $[x_1; x_2] \subset X$. Так как $x_1$ и $x_2$ — любые точки $X$, то $X$ — промежуток. $\blacksquare$

**Следствие.** Областями в $\mathbb{R}^1$ являются открытые промежутки и только они.

**Определение 9.15.** Отрезком прямой, соединяющим различные точки $x_1 = (\alpha_1^{(1)}, \dots, \alpha_n^{(1)})$ и $x_2 = (\alpha_1^{(2)}, \dots, \alpha_n^{(2)})$ в $\mathbb{R}^n$, называется непрерывная кривая
$$\Gamma = \{ (\alpha_1, \dots, \alpha_n) : \alpha_i = \alpha_i^{(1)} + (\alpha_i^{(2)} - \alpha_i^{(1)})t, t \in [0; 1], i = 1, 2, \dots, n \}.$$
Ясно, что при $t = 0$ получим точку $x_1$, а при $t = 1$ — точку $x_2$.

**Определение 9.16.** Непустое множество $X \subset \mathbb{R}^n$ называется **выпуклым**, если для любых различных точек $x_1, x_2 \in X$ найдётся отрезок прямой, соединяющий $x_1$ и $x_2$, все точки которого принадлежат $X$.
Так как отрезок является непрерывной кривой, то выпуклое множество всегда связно.

**Лемма 9.4.** $U_\varepsilon(x_0)$ — выпуклое множество ($x_0 \in \mathbb{R}^n, \varepsilon > 0$).
$\square$ Пусть $x_1, x_2 \in U_\varepsilon(x_0)$, т.е. $\rho(x_0, x_1) < \varepsilon, \rho(x_0, x_2) < \varepsilon$. Рассмотрим произвольную точку $x$, принадлежащую отрезку, со-

---

единяющему точки $x_1$ и $x_2$ (см. рис. 9.3). Имеем:
$$x_1 = (\alpha_1^{(1)}, \dots, \alpha_n^{(1)}), \quad x_2 = (\alpha_1^{(2)}, \dots, \alpha_n^{(2)});$$
$$x = (\alpha_1, \dots, \alpha_n), \text{ где } \alpha_i = \alpha_i^{(1)} + (\alpha_i^{(2)} - \alpha_i^{(1)})t, \quad t \in [0; 1], i = 1, 2, \dots, n; \quad x_0 = (\alpha_0^{(1)}, \dots, \alpha_0^{(n)}).$$

Запишем уравнение отрезка прямой, соединяющего $x_0$ и $x_2$:
$$\beta_i = \alpha_i^{(0)} + (\alpha_i^{(2)} - \alpha_i^{(0)})u, \quad u \in [0; 1], i = 1, 2, \dots, n.$$

> ![Изображение 9.3 (шар с центром x0, точками x1, x2, x и y)](Рисунок 9.3, иллюстрирующий доказательство выпуклости шара)

Возьмём на этом отрезке точку
$$y = (\beta_1, \dots, \beta_n),$$
соответствующую значению $u = t$, где $t$ — значение параметра, соответствующее точке $x$ на отрезке, соединяющем $x_1$ и $x_2$ (см. рис. 9.3). Тогда
$$\rho(x_0, x) \le \rho(x_0, y) + \rho(y, x). \tag{9.2}$$
Но $\rho(x_0, y) = \sqrt{\sum_{i=1}^n ((\alpha_i^{(2)} - \alpha_i^{(0)})t)^2} = t \cdot \rho(x_0, x_2)$;
$$\rho(y, x) = \sqrt{\sum_{i=1}^n ((\alpha_i - \alpha_i^{(0)}) - (\alpha_i^{(2)} - \alpha_i^{(0)})t)^2} =$$
$$= \sqrt{\sum_{i=1}^n ((\alpha_i^{(1)} - \alpha_i^{(0)}) + (\alpha_i^{(2)} - \alpha_i^{(1)})t - (\alpha_i^{(2)} - \alpha_i^{(0)})t)^2} =$$
$$= \sqrt{\sum_{i=1}^n ((\alpha_i^{(1)} - \alpha_i^{(0)})(1 - t))^2} = (1 - t) \cdot \rho(x_0, x_1).$$
Из (9.2) теперь легко заметить, что
$$\rho(x_0, x) \le t \cdot \varepsilon + (1 - t) \cdot \varepsilon = \varepsilon,$$
поэтому $x \in U_\varepsilon(x_0)$. Так как $x$ — произвольная точка отрезка, то $U_\varepsilon(x_0)$ — выпуклое множество. $\blacksquare$

**Замечание.** Доказательство достаточно громоздко, так как мы старались не прибегать к геометрической наглядности; двумерный рис. 9.3 служит лишь иллюстрацией.

---

На практике в таких рассуждениях значительно более свободно применяют геометрические соображения и пользуются известными утверждениями геометрии и векторной алгебры.
Например, приведём более простое доказательство леммы 9.4.
$\square$ Точка $x$ делит отрезок, соединяющий $x_1$ и $x_2$, в отношении $t : (1 - t)$, поэтому $\vec{x_0x} = (1 - t)\vec{x_0x_1} + t\vec{x_0x_2}$ и $\rho(x_0, x) \le (1 - t)\rho(x_0, x_1) + t\rho(x_0, x_2)$. Так как $\rho(x_0, x_1) < \varepsilon$ и $\rho(x_0, x_2) < \varepsilon$, то $\rho(x_0, x) \le (1 - t)\varepsilon + t \cdot \varepsilon = \varepsilon$. $\blacksquare$

В дальнейшем мы будем иногда применять подобные геометрические соображения, стараясь, впрочем, не злоупотреблять геометрической наглядностью.

## § 2. Предел функции многих переменных

Функцией нескольких переменных называется функция, область определения которой $D(f) \subset \mathbb{R}^n$, а множество значений $E(f) \subset \mathbb{R}$. Аргумент такой функции — точка $n$-мерного евклидова пространства $\bar{x} = (x_1, \dots, x_n)$, т.е. $f(\bar{x}) = f(x_1, \dots, x_n)$. Во избежание путаницы в §§ 2, 3, 4 этой главы если точка $\mathbb{R}^n$ обозначается малой буквой, то мы будем ставить над этой буквой черту; если черты нет — это действительное число, т.е. точка числовой прямой $\mathbb{R}^1$. Определения предела функции многих переменных при $\bar{x} \to \bar{x}_0$ совершенно аналогичны соответствующим определениям предела функции одной переменной (здесь $\beta$ — один из 6 СПС, см. § 6 главы II).

**Определение 9.17 (по Гейне).** Пусть функция $f$ определена в некоторой проколотой окрестности точки $\bar{a} \in \mathbb{R}^n$. Тогда говорят, что $\lim_{\bar{x} \to \bar{a}} f(\bar{x}) = \beta$, если для любой последовательности $\bar{x}_k$ такой, что $\lim_{k \to \infty} \bar{x}_k = \bar{a}$ и $\bar{x}_k \ne \bar{a}$, выполняется равенство $\lim_{k \to \infty} f(\bar{x}_k) = \beta$.

**Определение 9.18 (по Коши).** Говорят, что $\lim_{\bar{x} \to \bar{a}} f(\bar{x}) = \beta$, если для любого $\varepsilon > 0$ найдётся $\delta > 0$ такое, что для всех $\bar{x} \in \mathring{U}_\delta(\bar{a})$ соответствующие значения $f(\bar{x})$ принадлежат $U_\varepsilon(\beta)$.

---

Сохраняются вместе с доказательствами следующие утверждения главы III: эквивалентность определений предела функции по Гейне и по Коши, лемма о сохранении знака, теорема об арифметических действиях с пределами, теорема о предельном переходе в неравенстве, «теорема о двух милиционерах», теорема о замене переменной под знаком предела ($\bar{a} \in \mathbb{R}^n$, каждый из символов $\beta$ и $\gamma$ — один из 6 СПС), критерий Коши для существования конечного предела функции и т.д. Функция $f$ называется бесконечно малой при $\bar{x} \to \bar{a}$, если $\lim_{\bar{x} \to \bar{a}} f(\bar{x}) = 0$. Если функция $f$ бесконечно малая при $\bar{x} \to \bar{a}$, а функция $g$ ограничена в некоторой проколотой окрестности $\bar{a}$, то функция $fg$ является бесконечно малой при $\bar{x} \to \bar{a}$.

Говорят, что $f(\bar{x}) = o(g(\bar{x}))$ при $\bar{x} \to \bar{a}$, если $f(\bar{x}) = \alpha(\bar{x})g(\bar{x})$, где $\alpha(\bar{x})$ — бесконечно малая при $\bar{x} \to \bar{a}$. Если $g(\bar{x}) \ne 0$ в некоторой проколотой окрестности $\bar{a}$, то это определение равносильно равенству $\lim_{\bar{x} \to \bar{a}} \frac{f(\bar{x})}{g(\bar{x})} = 0$.

Несмотря на общность определения, исследование существования предела для функций многих переменных принципиально сложнее, чем в одномерном случае. Например, если функция одной переменной по-разному определяется при $x > a$ и $x < a$, то достаточно исследовать существование односторонних пределов справа и слева. На числовой прямой к точке можно «подобраться» двумя способами, а вот уже на плоскости таких способов бесконечно много. При этом в случае $n \ge 3$ картина не усложняется качественно. Принципиальным является именно переход от $n = 1$ к $n = 2$. В этом параграфе мы в дальнейшем будем считать, что $n = 2$, т.е. $\bar{x} = (x, y), f(\bar{x}) = f(x, y)$.

Предел функции двух переменных называется двойным пределом и обозначается
$$\lim_{(x,y) \to (x_0,y_0)} f(x, y) \text{ или } \lim_{\substack{x \to x_0 \\ y \to y_0}} f(x, y).$$

**Определение по Гейне:** $\lim_{\substack{x \to x_0 \\ y \to y_0}} f(x, y) = \beta$, если функция $f$ определена в некоторой проколотой окрестности точки $(x_0, y_0)$

---

и для любой последовательности точек $(x_k, y_k)$ такой, что $\lim_{k \to \infty} (x_k, y_k) = (x_0, y_0)$, но $(x_k, y_k) \ne (x_0, y_0)$, выполняется равенство $\lim_{k \to \infty} f(x_k, y_k) = \beta$.

Условие $(x_k, y_k) \ne (x_0, y_0)$ означает несовпадение точек; при этом возможно, например, что $x_k = x_0$ (но тогда обязательно $y_k \ne y_0$).

**Определение по Коши:** $\lim_{\substack{x \to x_0 \\ y \to y_0}} f(x, y) = \beta$, если
$$\forall \varepsilon > 0 \to \exists \delta > 0 :$$
$$\forall (x, y), \quad 0 < \sqrt{(x - x_0)^2 + (y - y_0)^2} < \delta \to f(x, y) \in U_\varepsilon(\beta).$$

Естественно возникает желание свести понятие двойного предела к уже изученным пределам функции одной переменной. Рассмотрим повторный предел
$$\lim_{x \to x_0} \left( \lim_{y \to y_0} f(x, y) \right). \tag{9.3}$$
Существование такого предела означает, что при всех $x$, принадлежащих некоторой проколотой окрестности точки $x_0$, существует конечное значение $\varphi(x) = \lim_{y \to y_0} f(x, y)$; затем оказывается, что существует $\lim_{x \to x_0} \varphi(x) = \beta$. Этот СПС $\beta$ и называется повторным пределом (9.3). Аналогично можно определить повторный предел
$$\lim_{y \to y_0} \left( \lim_{x \to x_0} f(x, y) \right). \tag{9.4}$$

Нетрудно привести пример, когда оба повторных предела существуют и совпадают, а двойной предел не существует.

**Пример 9.5.** Рассмотрим функцию $f(x, y) = \frac{(x + y)^2}{x^2 + y^2}$, $x^2 + y^2 > 0$. Для любого $x \ne 0$ существует $\varphi(x) = \lim_{y \to 0} \frac{(x + y)^2}{x^2 + y^2} = \frac{x^2}{x^2} = 1$, поэтому $\lim_{x \to 0} \left( \lim_{y \to 0} f(x, y) \right) = 1$. Аналогично для любого $y \ne 0$ существует $\psi(y) = \lim_{x \to 0} \frac{(x + y)^2}{x^2 + y^2} = \frac{y^2}{y^2} = 1$ и

---

$\lim_{y \to 0} \left( \lim_{x \to 0} f(x, y) \right) = 1$. Тем не менее двойной предел не существует. В самом деле, $f(x, x) = 2, f(x, -x) = 0$. Если бы двойной предел существовал и равнялся $b$, то для любой последовательности $(x_k, y_k) \ne (0, 0)$, такой, что $\lim_{k \to \infty} (x_k, y_k) = (0, 0)$, выполнялось бы равенство $\lim_{k \to \infty} f(x_k, y_k) = b$. Но если $x_k \ne 0$ и $\lim_{k \to \infty} x_k = 0$, то $f(x_k, x_k) = 2$, а $f(x_k, -x_k) = 0$, т.е. $b = 2 = 0$. Полученное противоречие доказывает отсутствие двойного предела при $x \to 0, y \to 0$.

Ещё одна попытка свести двойной предел к пределам функций одной переменной — это пределы по направлениям.

**Определение 9.19.** Пусть $\vec{e} = (\cos \varphi, \sin \varphi)$ — единичный вектор. Если функция двух переменных $f$ определена в некоторой проколотой окрестности точки $(x_0, y_0)$, то её пределом при $x \to x_0, y \to y_0$ по направлению вектора $\vec{e}$ (или по направлению, определяемому углом $\varphi$) называется предел функции одной переменной $\rho$:
$$\lim_{\rho \to +0} f(x_0 + \rho \cos \varphi, y_0 + \rho \sin \varphi).$$

**Замечание.** В этом случае фактически вводятся полярные координаты на плоскости с центром в точке $(x_0, y_0)$. Фиксированное значение $\varphi$ говорит о том, что функция рассматривается лишь на луче, выходящем из точки $(x_0, y_0)$ под углом $\varphi$ к положительному лучу прямой, параллельной оси $Ox$. При $\varphi = 0$ получаем предел справа функции одной переменной $f(x, y_0)$ в точке $x_0$:
$$\lim_{x \to x_0 + 0} f(x, y_0),$$
при $\varphi = \frac{\pi}{2}$ получаем $\lim_{y \to y_0 + 0} f(x_0, y)$ и т.д.

**Лемма 9.5.** Если $\lim_{\substack{x \to x_0 \\ y \to y_0}} f(x, y) = \beta$, то предел по любому направлению при $x \to x_0, y \to y_0$ равен $\beta$.
$\square$ Рассмотрим произвольную последовательность $\rho_k$ положительных чисел такую, что $\lim_{k \to \infty} \rho_k = 0$. Тогда последовательность точек $(x_k, y_k) = (x_0 + \rho_k \cos \varphi, y_0 + \rho_k \sin \varphi)$ стремится к

---

точке $(x_0, y_0)$ (это следует из того, что $\rho((x_k, y_k), (x_0, y_0)) = \rho_k$), причём $(x_k, y_k) \ne (x_0, y_0)$. Значит, $\lim_{k \to \infty} f(x_0 + \rho_k \cos \varphi, y_0 + \rho_k \sin \varphi) = \beta$, а отсюда следует, что $\lim_{\rho \to +0} f(x_0 + \rho \cos \varphi, y_0 + \rho \sin \varphi) = \beta$ (здесь использованы определения предела по Гейне функции двух переменных $x, y$ и функции одной переменной $\rho$). $\blacksquare$

Поэтому если пределы функции по двум разным направлениям в точке различны, то двойной предел не существует. Так, в примере 9.5
$$f(\rho \cos \varphi, \rho \sin \varphi) = \frac{(\rho \cos \varphi + \rho \sin \varphi)^2}{\rho^2 \cos^2 \varphi + \rho^2 \sin^2 \varphi} = (\cos \varphi + \sin \varphi)^2,$$
при $\varphi = \frac{\pi}{4}$ имеем 2, при $\varphi = -\frac{\pi}{4}$ имеем 0. По различным направлениям разные пределы, значит, двойной предел при $x \to 0, y \to 0$ не существует.

Оказывается, что если по любому направлению функция имеет в точке один и тот же предел $\beta$, то это ещё не гарантирует наличия двойного предела.

**Пример 9.6.** Рассмотрим функцию $f(x, y) = \frac{x^2 y}{x^4 + y^2}, x^2 + y^2 > 0$. Переходим к полярным координатам с центром в точке $(0, 0)$:
$$f(\rho \cos \varphi, \rho \sin \varphi) = \frac{\rho^3 \cos^2 \varphi \sin \varphi}{\rho^4 \cos^4 \varphi + \rho^2 \sin^2 \varphi} = \frac{\rho \cos^2 \varphi \sin \varphi}{\rho^2 \cos^4 \varphi + \sin^2 \varphi}.$$
Если $\sin \varphi = 0$, то последнее выражение равно 0 при любом $\rho > 0$. Если $\sin \varphi \ne 0$, то всё равно $\lim_{\rho \to +0} f(\rho \cos \varphi, \rho \sin \varphi) = 0$. Итак, по любому направлению предел функции при $x \to 0, y \to 0$ равен 0. Но легко видеть, что $f(x, x^2) = \frac{1}{2}$. Аналогично рассуждению в примере 9.5 если $x_k \ne 0$ и $\lim_{k \to \infty} x_k = 0$, то $f(x_k, 0) = 0, f(x_k, x_k^2) = \frac{1}{2}$. Двойной предел $\lim_{\substack{x \to 0 \\ y \to 0}} f(x, y)$ не существует.

Как же могло произойти так, что по каждому направлению предел равен 0 (вроде бы, как ни идти к точке, всё равно 0), а

---

по параболе получаем другой предел $\frac{1}{2}$? Возьмём, например, точку $A$ на рис. 9.4. Значение $f(A) = \frac{1}{2}$, но предел $f(x, y)$ по направлению вектора $\vec{OA}$ равен 0. То же самое можно сказать про точки $B, C, \dots$, лежащие на параболе. Во всех этих точках $f$ принимает значение $\frac{1}{2}$. Чем ближе такая точка к началу координат, тем короче отрезок, на котором функция должна «упасть» от значения $\frac{1}{2}$ до значения 0, но такой отрезок имеет положительную длину, и стремление функции к нулю вдоль этого отрезка ничему не противоречит («стремление к нулю по всем направлениям, но неравномерно по направлениям» — смысл этой фразы станет понятен позже).

> ![График парабол y = kx^2 и векторов, исходящих из начала координат](Рис. 9.4, поясняющий отсутствие предела в примере 9.6)

Таким образом, понятие двойного предела нельзя свести исключительно к пределам функции одной переменной. Нужны другие методы исследования.

**Лемма 9.6 (достаточное условие существования конечного двойного предела).** Пусть $\exists \rho_0 > 0 : \forall \rho \in (0; \rho_0) \to |f(x_0 + \rho \cos \varphi, y_0 + \rho \sin \varphi) - b| \le F(\rho)$, где $\lim_{\rho \to +0} F(\rho) = 0$. Тогда $\lim_{\substack{x \to x_0 \\ y \to y_0}} f(x, y) = b$.
В этом случае предел равен $b$ «равномерно по направлениям», функция $F$ не зависит от $\varphi$.

$\square$ Из условия следует, что
$$\forall \varepsilon > 0 \to \exists \delta > 0 : \forall \rho \in (0; \delta) \to F(\rho) < \varepsilon.$$
Для произвольной точки $(x, y)$ плоскости определим числа $\rho$ и $\varphi$ так, что $x = x_0 + \rho \cos \varphi, y = y_0 + \rho \sin \varphi$. Тогда $\rho = \sqrt{(x - x_0)^2 + (y - y_0)^2}$. Поэтому для всех точек $(x, y)$ таких, что $0 < \sqrt{(x - x_0)^2 + (y - y_0)^2} < \delta_1 \equiv \min(\rho_0, \delta)$, выполня-

---

ется неравенство $|f(x, y) - b| = |f(x_0 + \rho \cos \varphi, y_0 + \rho \sin \varphi) - b| \le F(\rho) < \varepsilon$. Это и означает, что $\lim_{\substack{x \to x_0 \\ y \to y_0}} f(x, y) = b$. $\blacksquare$

**Пример 9.7.** Рассмотрим функцию $f(x, y) = \frac{x^2 y}{x^2 + y^2}, x^2 + y^2 > 0$. Так как $f(\rho \cos \varphi, \rho \sin \varphi) = \frac{\rho^3 \cos^2 \varphi \sin \varphi}{\rho^2 (\cos^2 \varphi + \sin^2 \varphi)} = \rho \cos^2 \varphi \sin \varphi$, то $|f(\rho \cos \varphi, \rho \sin \varphi)| \le \rho \to 0$, и $\lim_{\substack{x \to 0 \\ y \to 0}} f(x, y) = 0$.

**Замечание.** Грубо ошибочным является рассуждение: $\lim_{\rho \to +0} \rho \cos^2 \varphi \sin \varphi = 0$, следовательно, $\lim_{\substack{x \to 0 \\ y \to 0}} f(x, y) = 0$. Доказано лишь то, что предел по каждому направлению равен 0. Этого ещё недостаточно для наличия двойного предела.

## § 3. Предел функции в точке по множеству

**Определение 9.20.** Пусть $\bar{a}$ — предельная точка множества $X \subset \mathbb{R}^n$, причём при некотором $\delta > 0$ функция $f$ определена в $\mathring{U}_\delta(\bar{a}) \cap X$. Тогда говорят, что $\lim_{\substack{\bar{x} \to \bar{a} \\ \bar{x} \in X}} f(\bar{x}) = \beta$, если для любой последовательности $\bar{x}_k$ такой, что $\bar{x}_k \in X, \bar{x}_k \ne \bar{a}, \lim_{k \to \infty} \bar{x}_k = \bar{a}$, выполняется равенство $\lim_{k \to \infty} f(\bar{x}_k) = \beta$ (определение по Гейне).

**Определение 9.21.** Пусть $\bar{a}$ — предельная точка множества $X \subset \mathbb{R}^n$. Тогда говорят, что $\lim_{\substack{\bar{x} \to \bar{a} \\ \bar{x} \in X}} f(\bar{x}) = \beta$, если для любого $\varepsilon > 0$ найдётся $\delta > 0$ такое, что для всех $\bar{x} \in \mathring{U}_\delta(\bar{a}) \cap X$ соответствующие значения $f(\bar{x})$ принадлежат $U_\varepsilon(\beta)$ (определение по Коши).

В этих определениях $\beta$ — один из 6 СПС. Напомним, что предельная точка может принадлежать множеству, а может и не принадлежать. Предел по множеству в изолированной точке не определяется.

Односторонние пределы функции одной переменной могут рассматриваться как пределы по множеству: $\lim_{x \to a+0} f(x) = \lim_{\substack{x \to a \\ x \in (a; b)}} f(x)$ — предел по интервалу с левым концом в точке

---

$a$; $\lim_{x \to b-0} f(x) = \lim_{\substack{x \to b \\ x \in (a; b)}} f(x)$ — предел по интервалу с правым концом в точке $b$.

Сохраняются вместе с доказательствами эквивалентность определений предела по Гейне и по Коши, а также другие утверждения, перечисленные после определения 9.18. Приведём, например, формулировку и доказательство теоремы о замене переменной под знаком предела.

**Теорема 9.3.** Пусть $\lim_{\substack{\bar{x} \to \bar{a} \\ \bar{x} \in X}} f(\bar{x}) = \beta$ и $f(\bar{x}) \ne \beta$ в некоторой $\mathring{U}_\delta(\bar{a})$. Пусть, далее, $\lim_{u \to \beta} g(u) = \gamma$ ($g$ — функция одной переменной; каждый из символов $\beta$ и $\gamma$ — один из 6 СПС). Тогда $\lim_{\substack{\bar{x} \to \bar{a} \\ \bar{x} \in X}} g(f(\bar{x})) = \gamma$.

$\square$ Пусть $\bar{x}_k$ — произвольная последовательность точек $X$ такая, что $\lim_{k \to \infty} \bar{x}_k = \bar{a}, \bar{x}_k \ne \bar{a}$. Тогда $\lim_{k \to \infty} f(\bar{x}_k) = \beta$. Рассмотрим последовательность $u_k = f(\bar{x}_k)$; $\lim_{k \to \infty} u_k = \beta$. Далее найдётся номер $k_0$ такой, что $\forall k \ge k_0 \to \bar{x}_k \in \mathring{U}_\delta(\bar{a})$, значит, $u_k = f(\bar{x}_k) \ne \beta$. Тогда $\lim_{k \to \infty} g(u_k) = \gamma$, т.е. $\lim_{k \to \infty} g(f(\bar{x}_k)) = \gamma$. Так как последовательность $\bar{x}_k$ — любая, то $\lim_{\substack{\bar{x} \to \bar{a} \\ \bar{x} \in X}} g(f(\bar{x})) = \gamma$. $\blacksquare$

**Пример 9.8.** Найти $\lim_{\substack{x \to 0 \\ y \to 0}} |xy|^{xy}$.
$\square$ Областью определения функции является множество $X$, полученное удалением из плоскости координатных осей. Поэтому в любой проколотой окрестности $(0, 0)$ найдутся точки, где функция не определена, и, строго говоря, нужно найти предел по множеству $X$. Тем не менее значок $\lim_{\substack{x \to 0 \\ y \to 0}}$ можно употреблять и в этом случае, имея в виду предел по области определения функции.
Применим теорему 9.3 при $f(x, y) = xy; g(u) = |u|^u$.
Имеем: $\lim_{\substack{x \to 0 \\ y \to 0}} f(x, y) = 0$ (после перехода к полярным координатам $|f(\rho \cos \varphi, \rho \sin \varphi)| = |\rho^2 \cos \varphi \sin \varphi| \le \rho^2 \to 0$), зна-

---

чит, $\lim_{\substack{(x,y) \to (0,0) \\ (x,y) \in X}} f(x, y) = 0; f(x, y) \ne 0$ при $(x, y) \in X$. Далее, $\lim_{u \to 0} g(u) = 1$. В самом деле, $\lim_{u \to +0} g(u) = \lim_{u \to +0} u^u = 1$ (пример 5.8); $\lim_{u \to -0} g(u) = \lim_{u \to -0} (-u)^u = \lim_{t \to +0} t^{-t} = \frac{1}{\lim_{t \to +0} t^t} = 1$; значит, и предел при $u \to 0$ равен 1. Тогда по теореме 9.3 $\lim_{\substack{(x,y) \to (0,0) \\ (x,y) \in X}} g(f(x, y)) = 1$. $\blacksquare$

## § 4. Непрерывность функций многих переменных. Непрерывные отображения

**Определение 9.22.** Функция $f$ называется непрерывной в точке $\bar{a} \in \mathbb{R}^n$, если она определена в некоторой окрестности точки $\bar{a}$ и $\lim_{\bar{x} \to \bar{a}} f(\bar{x}) = f(\bar{a})$.

**Определение 9.23.** Пусть $X \subset \mathbb{R}^n, \bar{a} \in X$ и функция $f$ определена в $U_\delta(\bar{a}) \cap X$ при некотором $\delta > 0$. Тогда:
1) если $\bar{a}$ — предельная точка множества $X$, то функция $f$ называется непрерывной в точке $\bar{a}$ по множеству $X$, если $\lim_{\substack{\bar{x} \to \bar{a} \\ \bar{x} \in X}} f(\bar{x}) = f(\bar{a})$;
2) если $\bar{a}$ — изолированная точка множества $X$, то функция $f$ по определению считается непрерывной в точке $\bar{a}$ по множеству $X$.

Определения 9.22 и 9.23 можно сформулировать на языке Гейне и на языке Коши (уже без оговорки $\bar{x}_k \ne \bar{a}$ и с применением $U_\delta(\bar{a})$ вместо $\mathring{U}_\delta(\bar{a})$).
Напомним, что предел по множеству в изолированной точке не определяется. Непрерывность же определяется естественно. В самом деле, если функция $f$ непрерывна в точке $\bar{a}$ по множеству $X$, то для любой последовательности $\bar{x}_k \in X$ такой, что $\lim_{k \to \infty} \bar{x}_k = \bar{a}$, выполняется равенство $\lim_{k \to \infty} f(\bar{x}_k) = f(\bar{a})$. Если $\bar{a}$ — изолированная точка $X$, то члены такой последовательности совпадают с $\bar{a}$ при $k \ge k_0$, и $f(\bar{x}_k) = f(\bar{a})$ при $k \ge k_0$, т.е. непрерывность имеет место автоматически.

---

**Определение 9.24.** Функция $f$ называется непрерывной на множестве $X \subset \mathbb{R}^n$, если она непрерывна в любой точке $\bar{a} \in X$ по множеству $X$.

**Замечание.** В $\mathbb{R}^1$ определение 3.10 непрерывности функции на промежутке $I$ совпадает с определением 9.24 для $X = I$.

Теорема о непрерывности суммы, произведения и частного непрерывных функций (в точке и по множеству) ничем не отличается от теоремы 3.7 для функций одной переменной.

**Теорема 9.4 (непрерывность суперпозиции непрерывных функций).** Пусть функция $f$ непрерывна по множеству $X \subset \mathbb{R}^n$ в точке $\bar{a}$, а функция одной переменной $g$ непрерывна в точке $b = f(\bar{a})$. Тогда сложная функция $g(f)$ непрерывна в точке $\bar{a}$ по множеству $X$.

$\square$ Если $\bar{a}$ — предельная точка множества $X$, то доказательство ничем не отличается от доказательства теоремы 9.3 и соответствующих одномерных утверждений — теорем 3.5 и 3.8. В изолированной точке множества любая функция, определённая в этой точке, непрерывна по этому множеству по определению. $\blacksquare$

Из непрерывности суперпозиции непрерывных функций и непрерывности суммы, произведения и частного непрерывных функций следует непрерывность всех функций, которые в открытых точках, в которых эти функции определены. Например, функция двух переменных $\varphi(x, y) = \text{tg}(\sin(x + y^2) - \ln \cos x)$ непрерывна во всех точках, где $\cos x > 0$ и $\sin(x + y^2) - \ln \cos x \ne \frac{\pi}{2} + \pi k, k \in \mathbb{Z}$. Функция трёх переменных $\varphi(x, y, z) = \sqrt{1 - x^2 - y^2 - z^2}$ непрерывна на шаре $\{x^2 + y^2 + z^2 \le 1\}$. В этом случае внешняя функция $g(u) = \sqrt{u}$ непрерывна справа в точке 0; применяются теорема 9.4 и её вариант для односторонней непрерывности функции $g$, что не усложняет доказательства.

**Определение 9.25.** Функция $\Phi$, область определения которой принадлежит $\mathbb{R}^n$, а множество значений принадлежит $\mathbb{R}^m$, называется отображением из $\mathbb{R}^n$ в $\mathbb{R}^m$. Если $\Phi(\bar{x}) = \bar{y}$,

---

где $\bar{x} = (x_1, \dots, x_n) \in D(\Phi) \subset \mathbb{R}^n, \bar{y} = (y_1, \dots, y_m) \in \mathbb{R}^m$, то $y_1 = \Phi_1(x_1, \dots, x_n), \dots, y_m = \Phi_m(x_1, \dots, x_n)$ — числовые функции $n$ переменных. Если все функции $\Phi_1, \dots, \Phi_m$ непрерывны в точке $\bar{x}^0$ или на множестве $X \subset D(\Phi)$, то таковым же считается и само отображение $\Phi$.

**Определение 9.26.** Образом множества $X \subset D(\Phi)$ при отображении $\Phi$ из $\mathbb{R}^n$ в $\mathbb{R}^m$ называется множество $\Phi(X) = \{\bar{y} \in \mathbb{R}^m : (\exists \bar{x} \in X : \bar{y} = \Phi(\bar{x}))\}$. Если $E(\Phi)$ — множество значений отображения $\Phi$, а $Y \subset E(\Phi)$, то полным прообразом множества $Y$ при отображении $\Phi$ называется множество $\Phi^{-1}(Y) = \{\bar{x} \in D(\Phi) : \Phi(\bar{x}) \in Y\}$.

**Лемма 9.7.** Если отображение $\Phi$ из $\mathbb{R}^n$ в $\mathbb{R}^m$ непрерывно на своей области определения $D(\Phi)$, которая является открытым множеством, то полный прообраз открытого множества $Y \subset E(\Phi)$ является открытым множеством в $\mathbb{R}^n$.

$\square$ Пусть отображение задаётся функциями $y_i = \Phi_i(\bar{x}), i = 1, \dots, m; X = \Phi^{-1}(Y), \bar{x}^0 \in X$. Так как отображение непрерывно в точке $\bar{x}^0$, то $\forall i = 1, \dots, m, \forall \varepsilon > 0 \to \exists \delta_i > 0 : \forall \bar{x}, \rho(\bar{x}, \bar{x}^0) < \delta_i \longrightarrow |\Phi_i(\bar{x}) - \Phi_i(\bar{x}^0)| < \frac{\varepsilon}{\sqrt{m}}$. Тогда при $\rho(\bar{x}, \bar{x}^0) < \delta = \min_{i=1, \dots, m} \delta_i$ выполняется неравенство $\rho(\bar{y}, \bar{y}^0) = \sqrt{\sum_{i=1}^m (\Phi_i(\bar{x}) - \Phi_i(\bar{x}^0))^2} < \sqrt{m \cdot \frac{\varepsilon^2}{m}} = \varepsilon$, где $\bar{y}^0 = (y_1^0, \dots, y_m^0)$, $y_i^0 = \Phi_i(\bar{x}^0), i = 1, \dots, m$. Поэтому образ $U_\delta(\bar{x}^0)$ при отображении $\Phi$ принадлежит $U_\varepsilon(\bar{y}^0)$. Так как множество $Y$ открыто и $\bar{y}^0 \in Y$, то $\varepsilon > 0$ можно выбрать настолько малым, что $U_\varepsilon(\bar{y}^0) \subset Y$, следовательно, $\Phi(U_\delta(\bar{x}^0)) \subset Y$. Поэтому $U_\delta(\bar{x}^0) \subset X$. Так как $\bar{x}^0$ — произвольная точка $X$, то $X$ — открытое множество. $\blacksquare$

Приведём теперь самый общий вариант теоремы о суперпозиции непрерывных функций.

**Теорема 9.5 (о непрерывности суперпозиции непрерывных отображений).** Пусть отображение $\Phi$ из $\mathbb{R}^n$ в $\mathbb{R}^m$ непрерывно на множестве $X \subset \mathbb{R}^n$, а отображение $F$ из $\mathbb{R}^m$ в $\mathbb{R}^p$ непрерывно на множестве $Y \subset \mathbb{R}^m$, причём $\Phi(X) \subset Y$.

---

Тогда сложное отображение $F(\Phi)$ из $\mathbb{R}^n$ в $\mathbb{R}^p$ непрерывно на множестве $X$.

$\square$ Пусть $\bar{x}_0 \in X$. Так как отображение $\Phi$ непрерывно в точке $\bar{x}_0$ по множеству $X$, то для любой последовательности точек $\bar{x}_k \in X$ такой, что $\lim_{k \to \infty} \bar{x}_k = \bar{x}_0$, выполняется равенство $\lim_{k \to \infty} \Phi(\bar{x}_k) = \Phi(\bar{x}_0)$ (это следует из непрерывности всех координатных функций отображения $\Phi$ в точке $\bar{x}_0$ по множеству $X$). Тогда если $\bar{y}_k = \Phi(\bar{x}_k) \in \Phi(X) \subset Y$, то $\lim_{k \to \infty} \bar{y}_k = \Phi(\bar{x}_0) = \bar{y}_0$. Так как отображение $F$ непрерывно в точке $\bar{y}_0$ по множеству $Y$, то $\lim_{k \to \infty} F(\bar{y}_k) = F(\bar{y}_0)$, т.е. $\lim_{k \to \infty} F(\Phi(\bar{x}_k)) = F(\Phi(\bar{x}_0))$. Но $\bar{x}_k$ — произвольная последовательность такая, что $\bar{x}_k \in X$ и $\lim_{k \to \infty} \bar{x}_k = \bar{x}_0$, поэтому отображение $F(\Phi)$ непрерывно в точке $\bar{x}_0$ по множеству $X$. $\blacksquare$

Мы будем часто применять частный случай этой теоремы, когда $p = 1$ (внешнее отображение — в числовую прямую), а непрерывность рассматривается в точках. Этот частный случай будем называть следствием из теоремы 9.5.

**Следствие.** Пусть функции $n$ переменных $x_1(t_1, \dots, t_n), \dots, x_m(t_1, \dots, t_n)$ непрерывны в точке $(t_1^0, \dots, t_n^0)$, а функция $m$ переменных $f(x_1, \dots, x_m)$ непрерывна в точке $(x_1^0, \dots, x_m^0)$, где $x_j^0 = x_j(t_1^0, \dots, t_n^0), j = 1, \dots, m$. Тогда сложная функция $f(x_1(t_1, \dots, t_n), \dots, x_m(t_1, \dots, t_n))$ непрерывна в точке $(t_1^0, \dots, t_n^0)$.

## § 5. Свойства функций, непрерывных на множествах

**Теорема 9.6 (о промежуточных значениях непрерывной функции).** Пусть функция $f$ непрерывна на связном множестве $X \subset \mathbb{R}^n$, причём $x_1, x_2 \in X$, $f(x_1) = y_1$, $f(x_2) = y_2$, то для любого значения $y_0 \in [y_1; y_2]$ найдётся точка $x_0 \in X$ такая, что $f(x_0) = y_0$.

$\square$ Случай $x_1 = x_2$ тривиален, поэтому считаем, что $x_1 \ne x_2$.
Рассмотрим непрерывную кривую
$\Gamma = \{ (\alpha_1, \dots, \alpha_n) : \alpha_1 = \alpha_1(t), \dots, \alpha_n = \alpha_n(t), t \in [a; b] \}$,

---

соединяющую точки $x_1$ и $x_2$ и лежащую во множестве $X$.
Функция одной переменной $\varphi(t) = f(\alpha_1(t), \dots, \alpha_n(t))$ непрерывна в каждой точке отрезка $[a; b]$ по теореме 9.5 ($\Phi : [a; b] \longrightarrow \Gamma \subset \mathbb{R}^n; F : \Gamma \longrightarrow \mathbb{R}; \varphi = F(\Phi) : [a; b] \longrightarrow \mathbb{R}$; сложное отображение является числовой функцией одной переменной $t$, которая непрерывна на отрезке $[a; b]$). При этом $\varphi(a) = y_1, \varphi(b) = y_2$. По теореме 3.14 о промежуточных значениях непрерывной функции одной переменной, $\exists t_0 \in [a; b] : \varphi(t_0) = y_0$, т.е.
$$\exists x_0 = (\alpha_1(t_0), \dots, \alpha_n(t_0)) \in X : f(x_0) = y_0. \blacksquare$$

Теорема 9.6 обобщает теорему 3.14 о промежуточных значениях функции, непрерывной на отрезке.

**Теорема 9.7 (обобщение теорем Вейерштрасса 3.11 и 3.12).** Если функция $f$ непрерывна на компакте $F \subset \mathbb{R}^n$, то:
1) $f$ ограничена на $F$;
2) $\exists x_1, x_2 \in F : f(x_1) = \sup_F f(x), f(x_2) = \inf_F f(x)$.

$\square$ Доказательство аналогично одномерному случаю. Приведём доказательство первой части; изменения в доказательстве второй части по сравнению с одномерным случаем те же самые.
Пусть функция $f$ неограничена на $F$. Тогда $\forall E > 0 \to \exists x \in F : |f(x)| > E$. Рассмотрим $E = k, k \in \mathbb{N}$. Тогда $\forall k \to \exists x_k \in F : |f(x_k)| > k$. Так как $F$ — ограниченное множество, то последовательность $x_k$ ограничена. По теореме 9.1 Больцано–Вейерштрасса в $\mathbb{R}^n$ существует сходящаяся подпоследовательность $x_{k_m}$; $\lim_{m \to \infty} x_{k_m} = x_0$. Если при $m \ge m_0$ все $x_{k_m} = x_0$, то $x_0 \in F$; если нет, то $x_0$ — предельная точка $F$, и, в силу замкнутости $F$, всё равно $x_0 \in F$. Так как функция $f$ непрерывна в точке $x_0$ по множеству $F$, то $\lim_{m \to \infty} f(x_{k_m}) = f(x_0)$. С другой стороны, так как при всех $m$ выполняется неравенство $|f(x_{k_m})| > k_m$, то $\lim_{m \to \infty} f(x_{k_m}) = \infty$ — противоречие. Значит, функция $f$ ограничена на $F$. $\blacksquare$

---

Докажем ещё одно свойство непрерывных функций, являющееся аналогом леммы о сохранении знака.

**Теорема 9.8.** Пусть функция $f$ непрерывна на открытом множестве $G \subset \mathbb{R}^n$. Тогда при всех $c \in \mathbb{R}$ множества $A_c = \{x \in G : f(x) > c\}$ и $B_c = \{x \in G : f(x) < c\}$ открыты.

$\square$ Доказательство следует из леммы 9.7 при $m = 1$, так как множества $A_c$ и $B_c$ являются полными прообразами открытых множеств $(c; +\infty)$ и $(-\infty; c)$ в $\mathbb{R}^1$. $\blacksquare$

**Замечание.** При $c = 0$ открытость множеств $\{x \in G : f(x) > 0\}$ и $\{x \in G : f(x) < 0\}$ является другой формулировкой леммы о сохранении знака для непрерывных функций.

## § 6. Равномерная непрерывность функции на множестве

**Определение 9.27.** Функция $f$ называется **равномерно непрерывной** на множестве $X \subset \mathbb{R}^n$, если для любого положительного числа $\varepsilon$ найдётся такое положительное число $\delta$, что для всех точек $x', x'' \in X$, таких, что $\rho(x', x'') < \delta$, выполняется неравенство $|f(x') - f(x'')| < \varepsilon$.

На языке кванторов:
$$\forall \varepsilon > 0 \to \exists \delta > 0 :$$
$$\forall x', x'' \in X, \quad \rho(x', x'') < \delta \to |f(x') - f(x'')| < \varepsilon. \tag{9.5}$$

Определение по Коши непрерывности функции $f$ на множестве $X$ записывается так:
$$\forall x' \in X \quad \forall \varepsilon > 0 \to \exists \delta > 0 :$$
$$\forall x'' \in X, \quad \rho(x', x'') < \delta \to |f(x') - f(x'')| < \varepsilon \tag{9.6}$$
(т.е. $\forall x' \in X \to \lim_{\substack{x'' \to x' \\ x'' \in X}} f(x'') = f(x')$).

В записи (9.6) $\delta = \delta(\varepsilon, x')$; в записи (9.5) $\delta = \delta(\varepsilon)$; эти две записи отличаются перестановкой фрагмента $«\forall x' \in X»$. Так как зависимость только от $\varepsilon$ является частным случаем зависимости от $\varepsilon$ и от $x'$, то (9.6) следует из (9.5), т.е. равномерно

**Теорема 9.9 (Кантора).** Если функция $f$ непрерывна на компакте $F \subset \mathbb{R}^n$, то она равномерно непрерывна на нём.

$\square$ Пусть функция $f$ не является равномерно непрерывной на $F$. Тогда
$$\exists \varepsilon > 0 : \forall \delta > 0 \to \exists x', x'' \in F, \rho(x', x'') < \delta :$$
$$|f(x') - f(x'')| \ge \varepsilon.$$
Возьмём $\delta = \frac{1}{k}, k = 1, 2, \dots$ Тогда можно построить две последовательности $x'_k$ и $x''_k$ точек из $F$ таких, что $\rho(x'_k, x''_k) < \frac{1}{k}$, но $|f(x'_k) - f(x''_k)| \ge \varepsilon$. Так как $F$ — ограниченное множество, то по теореме 9.1 из последовательности $x'_k$ можно выделить сходящуюся подпоследовательность $x'_{k_m}$; $\lim_{m \to \infty} x'_{k_m} = x_0$. Если при $m \ge m_0$ все $x'_{k_m} = x_0$, то $x_0 \in F$; если нет, то $x_0$ — предельная точка $F$ и в силу замкнутости $F$ всё равно $x_0 \in F$.

Далее $\rho(x''_{k_m}, x_0) \le \rho(x''_{k_m}, x'_{k_m}) + \rho(x'_{k_m}, x_0) < \frac{1}{k_m} + \rho(x'_{k_m}, x_0)$. Так как $k_1 < k_2 < \dots < k_m < \dots$, то $k_m \ge m$, и $\lim_{m \to \infty} \frac{1}{k_m} = 0$. Также $\lim_{m \to \infty} \rho(x'_{k_m}, x_0) = 0$. Поэтому
$$\lim_{m \to \infty} \rho(x''_{k_m}, x_0) = 0,$$
т.е. $\lim_{m \to \infty} x''_{k_m} = x_0$. Но функция $f$ непрерывна в точке $x_0$ по множеству $F$, поэтому
$$\lim_{m \to \infty} f(x'_{k_m}) = \lim_{m \to \infty} f(x''_{k_m}) = f(x_0),$$
и
$$\lim_{m \to \infty} |f(x'_{k_m}) - f(x''_{k_m})| = 0,$$
что противоречит неравенству $|f(x'_{k_m}) - f(x''_{k_m})| \ge \varepsilon$, выполненному для всех $m = 1, 2, \dots$ $\blacksquare$

**Следствие.** Функция одной переменной $f$, непрерывная на отрезке $[a; b]$, равномерно непрерывна на нём.

**Теорема 9.10.** Функция одной переменной $f$, непрерывная на конечном интервале $(a; b)$, равномерно непрерывна на нём $\iff$ существуют конечные $f(a+0)$ и $f(b-0)$.

---

$\square$ ($\impliedby$) Доопределим функцию $f$ в точках $a$ и $b$: $f(a) = f(a+0)$, $f(b) = f(b-0)$. Полученная функция непрерывна на отрезке $[a; b]$; по теореме Кантора она равномерно непрерывна на $[a; b]$, а значит, и на его подмножестве $(a; b)$.
($\implies$) $\forall \varepsilon > 0 \to \exists \delta > 0 : \forall x', x'' \in (a; b), |x' - x''| < \delta \to |f(x') - f(x'')| < \varepsilon$. Если $x', x'' \in (a; a + \delta)$, то $|x' - x''| < \delta$. Поэтому
$$\forall \varepsilon > 0 \to \exists \delta > 0 : \forall x', x'' \in (a; a + \delta) \to |f(x') - f(x'')| < \varepsilon.$$
Поэтому выполнено условие Коши (критерий Коши существования конечного предела $\lim_{x \to a+0} f(x)$), и по теореме 3.6 существует конечный предел $f(a+0)$. Аналогично существует конечный $f(b-0)$. $\blacksquare$

Естественно, необходимым и достаточным условием равномерной непрерывности на конечном полуинтервале $(a; b]$ или $[a; b)$ функции $f$, непрерывной на нём, является существование конечного предела $f$ в том конце полуинтервала, который ему не принадлежит. Для бесконечного промежутка нет работающих на практике одновременно необходимых и достаточных условий равномерной непрерывности; можно привести некоторые только необходимые или только достаточные условия (теоремы 9.11 и 9.12, упражнение 9.19).

**Теорема 9.11.** Пусть функция одной переменной $f$ непрерывна на промежутке $I$ и имеет ограниченную производную на промежутке $I_1$, который получается из $I$ удалением концов (если они принадлежат $I$). Тогда функция $f$ равномерно непрерывна на $I$.

$\square$ Пусть $\forall x \in I_1 \to |f'(x)| \le M$, где $M > 0$. Тогда по теореме Лагранжа
$$\forall x', x'' \in I \to f(x') - f(x'') = f'(\xi)(x' - x''),$$
где $\xi \in (x'; x'') \subset I_1$, поэтому $|f(x') - f(x'')| \le M|x' - x''|$.
Значит,
$$\forall \varepsilon > 0 \to \exists \delta = \frac{\varepsilon}{M} : \forall x', x'' \in I, |x' - x''| < \delta \implies$$
$$\implies |f(x') - f(x'')| < \varepsilon. \blacksquare$$

Функция $f$ равномерно непрерывна на $I$. $\blacksquare$

**Пример 9.9.** Является ли функция $f(x) = \frac{\sin x}{x}$ равномерно непрерывной: а) на $(0; 1)$; б) на $(1; +\infty)$?
$\square$ а) Существуют конечные $f(+0) = 1$ и $f(1-0) = \sin 1$, поэтому функция $f$ равномерно непрерывна на $(0; 1)$ по теореме 9.10.
б) Так как $f'(x) = \frac{x \cos x - \sin x}{x^2} = \frac{\cos x}{x} - \frac{\sin x}{x^2}$, то при всех $x > 1$ выполняется неравенство $|f'(x)| \le \frac{1}{x} + \frac{1}{x^2} < 2$. Производная $f'$ ограничена на $(1; +\infty)$, поэтому функция $f$ равномерно непрерывна на $(1; +\infty)$ по теореме 9.11. $\blacksquare$

**Пример 9.10.** Является ли функция $f(x) = \ln x$ равномерно непрерывной: а) на $(0; 1)$; б) на $(1; +\infty)$?
$\square$ а) Так как $\lim_{x \to +0} f(x) = -\infty$, то по теореме 9.10 функция $f$ не является равномерно непрерывной на $(0; 1)$.
б) Заметим, что $f'(x) = \frac{1}{x}$ и при $x > 1$ выполняется неравенство $|f'(x)| < 1$, значит, функция $f$ равномерно непрерывная на $(1; +\infty)$ по теореме 9.11. $\blacksquare$

**Пример 9.11.** Является ли функция $f(x) = x^2$ равномерно непрерывной: а) на $(0; 1)$; б) на $(1; +\infty)$?
$\square$ а) Так как существуют конечные $f(+0) = 0$ и $f(1-0) = 1$, то функция $f$ равномерно непрерывна на $(0; 1)$ по теореме 9.10.
б) Теорема 9.11 в этом случае неприменима, так как $f'(x) = 2x$ неограничена на $(1; +\infty)$. Докажем, что функция $f$ не является равномерно непрерывной на $(1; +\infty)$ непосредственно по определению, т.е. покажем, что
$$\exists \varepsilon > 0 : \forall \delta > 0 \to \exists x', x'' > 1, |x' - x''| < \delta : |x'^2 - x''^2| \ge \varepsilon. \tag{9.7}$$
Возьмём $x' = n + \frac{1}{n}$, $x'' = n$, $n = 2, 3, \dots$ Тогда $x' > x'' > 1$; $|x' - x''| = \frac{1}{n}$, и за счёт выбора $n$ это выражение может быть сделано меньше произвольного положительного числа $\delta$.
При этом $|x'^2 - x''^2| = (n + \frac{1}{n})^2 - n^2 = 2 + \frac{1}{n^2} > 2$. Поэтому условие (9.7) выполняется при $\varepsilon = 2$. $\blacksquare$

**Замечание.** Отсутствие равномерной непрерывности $f$ на $(1; +\infty)$ следует также из того, что $\lim_{x \to +\infty} f'(x) = +\infty$ (упражнение 9.19).

**Теорема 9.12.** Если функция одной переменной $f$, непрерывная на луче $[a; +\infty)$, имеет конечный предел $f(+\infty)$, то она равномерно непрерывна на $[a; +\infty)$.

$\square$ Запишем условие Коши существования конечного предела $\lim_{x \to +\infty} f(x)$ (теорема 3.6):
$$\forall \varepsilon > 0 \to \exists \Delta > 0 : \forall x', x'' \ge \Delta \to |f(x') - f(x'')| < \frac{\varepsilon}{2} \tag{9.8}$$
(вместо $\forall x', x'' > \Delta$ записано нестрогое неравенство — это не имеет значения). Не уменьшая общности, можно считать, что $\Delta > a$. Так как функция $f$ непрерывна на отрезке $[a; \Delta]$, то по теореме Кантора она равномерно непрерывна на нём. Значения $\varepsilon > 0$ и $\Delta = \Delta(\varepsilon)$ уже фиксированы:
$$\exists \delta > 0 : \forall x', x'' \in [a; \Delta], |x' - x''| < \delta \to |f(x') - f(x'')| < \frac{\varepsilon}{2}. \tag{9.9}$$
Полученное значение $\delta$ зависит от $\varepsilon$. Докажем, что
$$\forall x', x'' \in [a; +\infty), |x' - x''| < \delta \to |f(x') - f(x'')| < \varepsilon \tag{9.10}$$
(этим будет доказано, что функция $f$ равномерно непрерывна на $[a; +\infty)$).
