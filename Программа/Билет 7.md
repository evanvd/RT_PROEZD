# БИЛЕТ 7. ГЕОМЕТРИЧЕСКИЕ ПРИЛОЖЕНИЯ ОПРЕДЕЛЕННОГО ИНТЕГРАЛА

### Площадь криволинейной трапеции

>[!info] **Определение 12.3.** (Подграфик и криволинейная трапеция)
> Графиком функции $y = f(x)$, $a \le x \le b$, называется множество точек из $\mathbb{R}^2$:
> $$\Gamma_f = \{ (x, y) : y = f(x), a \le x \le b \}.$$
> Если функция $f$ неотрицательна на отрезке $[a; b]$, то её подграфиком (криволинейной трапецией) называется множество точек из $\mathbb{R}^2$:
> $$\Pi_f = \{ (x, y) : 0 \le y \le f(x), a \le x \le b \}.$$ ^def12d3

>[!error] **Теорема 12.25.** (Мера графика и подграфика)
> 1) Если функция $f$ интегрируема на отрезке $[a; b]$, то её график $\Gamma_f$ измерим в $\mathbb{R}^2$ и $\mu \Gamma_f = 0$.
> 2) Если неотрицательная функция $f$ интегрируема на $[a; b]$, то её подграфик $\Pi_f$ измерим в $\mathbb{R}^2$, и $\mu \Pi_f = \int_a^b f(x) \, dx$. ^th12d25

$\square$ 2) По пункту 2° критерия Дарбу для любого $\varepsilon > 0$ найдётся разбиение $R$ отрезка $[a; b]$ такое, что $\omega_R = S^*_R - S_{*R} < \varepsilon$. Но $S^*_R = \sum_{i=1}^N M_i \Delta x_i, S_{*R} = \sum_{i=1}^N m_i \Delta x_i$ — это площади ступенчатых фигур $\Pi^*$ и $\Pi_{*}$, являющихся объединениями нескольких прямоугольников. Так как $\Pi_{*} \subset \Pi_f \subset \Pi^*$, а множества $\Pi_{*}$ и $\Pi^*$ измеримы, то $\mu \Pi_{*} \le \mu_* \Pi_f \le \mu^* \Pi_f \le \mu \Pi^*$. Тогда $0 \le \mu^* \Pi_f - \mu_* \Pi_f \le S^*_R - S_{*R} < \varepsilon$, т.е. $\Pi_f$ измерим. Далее, т.к. $S_{*R} \le \mu \Pi_f \le S^*_R$ и $S_{*R} \le I \le S^*_R$, то $|\mu \Pi_f - I| \le \omega_R < \varepsilon$, откуда $\mu \Pi_f = I = \int_a^b f(x) \, dx$.
1) $\Gamma_f \subset \Gamma$, где $\Gamma$ — ступенчатая фигура из прямоугольников высоты $M_i - m_i$. Её мера $\mu \Gamma = \omega_R$. Т.к. $\omega_R$ может быть сколь угодно малым, то $\mu \Gamma_f = 0$. $\blacksquare$

>[!error] **Теорема 12.26.** (Площадь области между графиками)
> Пусть $f$ и $g$ — две интегрируемые на $[a; b]$ функции, причём $f(x) \ge g(x)$ на $[a; b]$. Тогда множество из $\mathbb{R}^2$
> $$\Pi = \{ (x, y) : g(x) \le y \le f(x), a \le x \le b \}$$
> измеримо в $\mathbb{R}^2$, и $\mu \Pi = \int_a^b (f(x) - g(x)) \, dx$. ^th12d26

$\square$ Т.к. $f$ и $g$ ограничены, то $\exists N : f(x) \ge g(x) \ge -N$. Рассмотрим $f_1 = f + N, g_1 = g + N \ge 0$. Тогда $\Pi = (\Pi_{f_1} \setminus \Pi_{g_1}) \cup \Gamma_{g_1}$. По [[#^th12d25|Теореме 12.25 (Мера графика и подграфика)]] эти множества измеримы. Т.к. $\Pi_{g_1} \subset \Pi_{f_1}$ и $\mu \Gamma_{g_1} = 0$, то $\mu \Pi = \mu \Pi_{f_1} - \mu \Pi_{g_1} = \int (f+N) - \int (g+N) = \int (f-g)$. $\blacksquare$

### Объем тела вращения

>[!error] **Теорема 12.27.** (Объём тела вращения)
> Пусть функция $f$ неотрицательна и интегрируема на отрезке $[a; b]$;
> $$V_f = \{ (x, y, z) : y^2 + z^2 \le (f(x))^2, a \le x \le b \}$$
> — множество точек $\mathbb{R}^3$, полученное в результате вращения подграфика $\Pi_f$ вокруг оси $Ox$. Тогда множество $V_f$ измеримо в $\mathbb{R}^3$ и
> $$\mu V_f = \pi \int_a^b (f(x))^2 \, dx. $$ ^th12d27

$\square$ Т.к. $f$ интегрируема, то и $\pi f^2$ интегрируема. По критерию Дарбу для любого $\varepsilon > 0$ $\exists R : S^*_R - S_{*R} < \varepsilon$ для функции $\pi f^2$. Построим ступенчатые тела $T_1$ и $T_2$, состоящие из цилиндров с радиусами $\sqrt{m_i/\pi}$ и $\sqrt{M_i/\pi}$ соответственно. Их объемы $\mu T_1 = S_{*R}, \mu T_2 = S^*_R$. Т.к. $T_1 \subset V_f \subset T_2$, то $0 \le \mu^* V_f - \mu_* V_f \le S^*_R - S_{*R} < \varepsilon$, откуда следует измеримость $V_f$. Т.к. $S_{*R} \le \mu V_f \le S^*_R$ и $S_{*R} \le I \le S^*_R$, то $\mu V_f = I = \pi \int f^2$. $\blacksquare$

### Длина кривой

>[!error] **Теорема 12.28.** (Длина дуги кривой)
> Если кривая $\Gamma = \{ \vec{r} : \vec{r} = \vec{r}(t), t \in [a; b] \}$ непрерывно дифференцируема, то её длина равна $l_\Gamma = \int_a^b |\vec{r}'(t)| \, dt$. ^th12d28

$\square$ Согласно свойствам длины дуги, переменная длина $s(t)$ — непрерывно дифференцируемая функция и $s'(t) = |\vec{r}'(t)|$. Тогда по [[#^th12d21|Формуле Ньютона–Лейбница]] (из Билета 6) $\int_a^b |\vec{r}'(t)| \, dt = \int_a^b s'(t) \, dt = s(b) - s(a) = l_\Gamma$. $\blacksquare$

### Вычисление площади поверхности вращения

>[!error] **Формула площади поверхности вращения** (без доказательства)
> Площадь поверхности, образованной вращением графика непрерывно дифференцируемой функции $y = f(x) \ge 0, x \in [a; b]$, вокруг оси $Ox$, равна
> $$P = 2\pi \int_a^b f(x) \sqrt{1 + (f'(x))^2} \, dx.$$ ^th12dSurf
